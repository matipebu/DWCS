Transliterator::listIDs

transliterator_list_ids

(PHP 5 >= 5.4.0, PHP 7, PHP 8, PECL intl >= 2.0.0)

Transliterator::listIDs -- transliterator_list_ids — Obtener los IDs de los transliterardores

Descripción

Estilo orientado a objetos

public static Transliterator::listIDs(): array

Estilo por procedimientos

transliterator_list_ids(): array

Devuelve un array con los IDs de los transliteradores registrados.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array con los IDs de los transliteradores registrados en caso de éxito. o false en caso de error.
