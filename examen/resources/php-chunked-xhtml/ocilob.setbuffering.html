OCILob::setBuffering

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::setBuffering — Cambia el estado actual del uso del búfer para objetos grandes

Descripción

public OCILob::setBuffering(bool $mode): bool

Establece el uso del búfer para objetos grandes, dependiendo del valor del parámetro on_off.

El uso de esta función podría proporcionar mejoras en el rendimiento mediante el almacenamiento en búfer de lecturas y escrituras pequeñas de LOBs, reduciendo del número de viajes de ida y vuelta de la red y las versiones del LOB. Debería usarse OCILob::flush() para vaciar los búferes cuando se haya finalizado de trabajar con el objeto grande.

Parámetros

on_off

true para activarlo y false para desactivarlo.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error. Repetidas llamadas a este método con la misma bandera devolverán true.
