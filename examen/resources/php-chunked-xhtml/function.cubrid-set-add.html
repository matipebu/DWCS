cubrid_set_add

(PECL CUBRID >= 8.3.0)

cubrid_set_add — Insertar un único elemento en la columna del tipo de conjunto usando OID

Descripción

cubrid_set_add(
    resource $conn_identifier,
    string $oid,
    string $attr_name,
    string $set_element
): bool

La función cubrid_set_add() se usa para insertar un único elemento en un atributo de tipo de conjunto (conjunto, multiconjunto, secuencia) que se solicitó.

Parámetros

conn_identifier

Identificador de conexión.

oid

OID de la instancia con la que se quiere trabajar.

attr_name

Nombre del atributo en donde se quiere insertar un elemento.

set_element

Contenido del elemento que se quiere insertar.

Valores devueltos

true, cuando el proceso tiene éxito.

false, cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_set_add()

<?php
$conn = cubrid_connect("localhost", 33000, "demodb", "dba");

@cubrid_execute($conn, "DROP TABLE foo");
cubrid_execute($conn, "CREATE TABLE foo(a int AUTO_INCREMENT, b set(int), c list(int), d char(10))");
cubrid_execute($conn, "INSERT INTO foo(a, b, c, d) VALUES(1, {1,2,3}, {11,22,33,333}, 'a')");

$req = cubrid_execute($conn, "SELECT * FROM foo", CUBRID_INCLUDE_OID);

cubrid_move_cursor($req, 1, CUBRID_CURSOR_FIRST);
$oid = cubrid_current_oid($req);

$attr = cubrid_col_get($conn, $oid, "b");
var_dump($attr);

cubrid_set_add($conn, $oid, "b", "4");

$attr = cubrid_col_get($conn, $oid, "b");
var_dump($attr);

cubrid_close_request($req);
cubrid_disconnect($conn);
?>

El resultado del ejemplo sería:

array(3) {
  [0]=>
  string(1) "1"
  [1]=>
  string(1) "2"
  [2]=>
  string(1) "3"
}
array(4) {
  [0]=>
  string(1) "1"
  [1]=>
  string(1) "2"
  [2]=>
  string(1) "3"
  [3]=>
  string(1) "4"
}
