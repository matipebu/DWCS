pspell_check

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

pspell_check — Verificar una palabra

Descripción

pspell_check(int $dictionary_link, string $word): bool

pspell_check() verifica la ortografía de una palabra.

Parámetros

dictionary_link

word

La palabra comprobada.

Valores devueltos

Devuelve true si la ortografía es correcta, false si no.

Ejemplos

Ejemplo #1 Ejemplo de pspell_check()

<?php
$pspell_link = pspell_new("en");

if (pspell_check($pspell_link, "testt")) {
echo "Ésta es una ortografía correcta";
} else {
echo "Lo siento, ortografía errónea";
}
?>
