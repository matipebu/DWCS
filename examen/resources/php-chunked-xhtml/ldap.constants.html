Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

LDAP_DEREF_NEVER (integer)
LDAP_DEREF_SEARCHING (integer)
LDAP_DEREF_FINDING (integer)
LDAP_DEREF_ALWAYS (integer)
LDAP_OPT_DEREF (integer)
Especifica reglas alternativas para el seguimiento de aliases en el servidor.
LDAP_OPT_SIZELIMIT (integer)

Especifica el número máximo de entradas que pueden ser devueltas en una operación de búsqueda.

Nota: El límite de tamaño actual de las operaciones también está delimitado por el número máximo de entradas de retorno configuradas en el servidor. El menor de estos dos valores es el límite de tamaño actual.

LDAP_OPT_TIMELIMIT (integer)
Especifica el número de segundos de espera para la búsqueda de resultados.

Nota: El límite de tiempo para las operaciones también está delimitado por el tiempo máximo configurado en el servidor. El menor de estos dos valores es el límite de tiempo actual.

LDAP_OPT_NETWORK_TIMEOUT (integer)
Opción para ldap_set_option() que permite ajustar el tiempo de espera de la red. (Disponible a partir de PHP 5.3.0)
LDAP_OPT_PROTOCOL_VERSION (integer)
Especifica el protocolo LDAP a ser utilizado (V2 ó V3).
LDAP_OPT_ERROR_NUMBER (integer)
LDAP_OPT_REFERRALS (integer)
Especifica si se deben seguir automáticamente las referencias devueltas por el servidor LDAP.
LDAP_OPT_RESTART (integer)
LDAP_OPT_HOST_NAME (integer)
LDAP_OPT_ERROR_STRING (integer)
LDAP_OPT_DIAGNOSTIC_MESSAGE (integer)
LDAP_OPT_MATCHED_DN (integer)
LDAP_OPT_SERVER_CONTROLS (integer)
Especifica una lista predeterminada de controles de servidor que se envía con cada solicitud.
LDAP_OPT_CLIENT_CONTROLS (integer)
Especifica una lista predeterminada de controles de cliente a ser procesada con cada solicitud.
LDAP_OPT_DEBUG_LEVEL (integer)
Especifica un nivel de bit por bit para depurar trazas.
LDAP_OPT_X_KEEPALIVE_IDLE (integer)
Especifica el número de segundos que necesita una conexión para mantenerse en estado desocupado antes que TCP comience a enviar pruebas de que la conexión se mantiene activa.
LDAP_OPT_X_KEEPALIVE_PROBES (integer)
Especifica el número máximo de pruebas de conexión activa que TCP debería enviar antes de terminar la conexión.
LDAP_OPT_X_KEEPALIVE_INTERVAL (integer)
Especifica el intervalo en segundos entre cada prueba individual donde verifica que la conexión se mantiene activa.
LDAP_OPT_X_TLS_CACERTDIR (string)
Especifica la ruta del directorio que contiene los certificados de la CA (Autoridad Certificadora).
LDAP_OPT_X_TLS_CACERTFILE (string)
Especifica la ruta completa del fichero de certificado de la la CA (Autoridad Certificadora).
LDAP_OPT_X_TLS_CERTFILE (string)
Especifica la ruta completa del fichero de certificado.
LDAP_OPT_X_TLS_CIPHER_SUITE (string)
Especifica el conjunto de cifrado permitido.
LDAP_OPT_X_TLS_CRLCHECK (integer)
Especifica la estrategia de evaluación CRL. Esta debe ser una de las siguientes: LDAP_OPT_X_TLS_CRL_NONE,LDAP_OPT_X_TLS_CRL_PEER, LDAP_OPT_X_TLS_CRL_ALL.

Nota: Esta opción es válida solamente para OpenSSL.

LDAP_OPT_X_TLS_CRLFILE (string)
Especifica la ruta completa del fichero CRL.

Nota: Esta opción es válida solamente para GnuTLS.

LDAP_OPT_X_TLS_DHFILE (string)
Especifica la ruta completa del fichero que contiene los parámetros para el intercambio de claves efímero de Diffie-Hellman.

Nota: Esta opción es ignorada por GnuTLS y Mozilla NSS.

LDAP_OPT_X_TLS_KEYFILE (string)
Especifica la ruta completa del fichero clave de certificado.
LDAP_OPT_X_TLS_PROTOCOL_MIN (integer)
Especifica la versión mínima del protocolo. Esta puede ser una de las siguientes: LDAP_OPT_X_TLS_PROTOCOL_SSL2,LDAP_OPT_X_TLS_PROTOCOL_SSL3, LDAP_OPT_X_TLS_PROTOCOL_TLS1_0, LDAP_OPT_X_TLS_PROTOCOL_TLS1_1, LDAP_OPT_X_TLS_PROTOCOL_TLS1_2
LDAP_OPT_X_TLS_RANDOM_FILE (string)
LDAP_OPT_X_TLS_REQUIRE_CERT (integer)
Especifica el certificado que comprueba la estrategia de comprobación. Esta debe ser una de las siguientes: LDAP_OPT_X_TLS_NEVER,LDAP_OPT_X_TLS_HARD, LDAP_OPT_X_TLS_DEMAND, LDAP_OPT_X_TLS_ALLOW, LDAP_OPT_X_TLS_TRY.
GSLC_SSL_NO_AUTH (integer)
GSLC_SSL_ONEWAY_AUTH (integer)
GSLC_SSL_TWOWAY_AUTH (integer)
LDAP_EXOP_START_TLS (string)
LDAP_EXOP_MODIFY_PASSWD (string)
LDAP_EXOP_REFRESH (string)
LDAP_EXOP_WHO_AM_I (string)
LDAP_EXOP_TURN (string)
