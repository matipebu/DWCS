IntlCalendar::getWeekendTransition

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getWeekendTransition — Obtener el instante del día en la que comienza o termina el fin de semana

Descripción

Estilo orientado a objetos

public IntlCalendar::getWeekendTransition(string $dayOfWeek): int

Estilo por procedimientos

intlcal_get_weekend_transition(IntlCalendar $cal, string $dayOfWeek): int

Devuelve el número de milisegundos después de medianoche en los que comienza o termina el fin de semana.

Esto es sólo aplicable a días de la semana para los que IntlCalendar::getDayOfWeekType() devuelve IntlCalendar::DOW_TYPE_WEEKEND_OFFSET o IntlCalendar::DOW_TYPE_WEEKEND_CEASE. Invocar a esta función para otros días de la semana es una condición d error.

Esta función requiere ICU 4.4 o posterior.

Parámetros

cal

El recurso IntlCalendar.

dayOfWeek

Una de las constantes IntlCalendar::DOW_SUNDAY, IntlCalendar::DOW_MONDAY, …, IntlCalendar::DOW_SATURDAY.

Valores devueltos

El número de milisegundos dentro el día en el que comienza o termina el fin de semana o false en caso de error.
