RRDGraph::saveVerbose

(PECL rrd >= 0.9.0)

RRDGraph::saveVerbose — Guarda la consulta de la bases de datos RRD en la imagen y devuelve la información detallada acerca de gráfico generado.

Descripción

public RRDGraph::saveVerbose(): array

Guarda la consulta de bases de datos RRD en el archivo de imagen definido por el método RRDGraph::__construct() y devuelve la información detallada acerca de gráfico generado, si "-" se utiliza como nombre de archivo de imagen, los datos de la imagen también se devolverán en el array de resultados.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Se devuelve un array con información detallada acerca de la imagen generada, opcionalmente con datos de imagen, false si se produce un error.
