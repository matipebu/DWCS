DOMDocument::relaxNGValidate

(PHP 5, PHP 7, PHP 8)

DOMDocument::relaxNGValidate — Realiza una validación relaxNG del documento

Descripción

public DOMDocument::relaxNGValidate(string $filename): bool

Realiza una validación » relaxNG del documento basándose en el esquema RNG dado.

Parámetros

filename

El fichero RNG.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
