stream_context_create

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

stream_context_create — Crear un contexto de flujo

Descripción

stream_context_create(array $options = ?, array $params = ?): resource

Crea y devuelve un contexto de flujo con cualquier opción proporcionada en options.

Parámetros

options

Debe ser un array asociativo de arrays asociativos con el formato $matriz['envoltura']['opción'] = $valor.

Por defecto es un array vacío.

params

Debe ser un array asociativo en el formato $matriz['parámetro'] = $valor. Consulte parámetros de contexto para un listado de los parámetros de flujo estándar.

Valores devueltos

Un resource de contexto de flujo.

Historial de cambios

Versión Descripción
5.3.0 Se añadió el argumento opcional params.

Ejemplos

Ejemplo #1 Usar stream_context_create()

<?php
$opciones = array(
'http'=>array(
'method'=>"GET",
'header'=>"Accept-language: en\r\n" .
"Cookie: foo=bar\r\n"
)
);

$contexto = stream_context_create($opciones);

/* Envía una petición http a www.example.com
con las cabeceras adicionales mostradas arriba */
$fp = fopen('http://www.example.com', 'r', false, $contexto);
fpassthru($fp);
fclose($fp);
?>
