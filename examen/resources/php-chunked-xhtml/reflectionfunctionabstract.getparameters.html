ReflectionFunctionAbstract::getParameters

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

ReflectionFunctionAbstract::getParameters — Obtiene los parámetros

Descripción

public ReflectionFunctionAbstract::getParameters(): array

Devuelve los parámetros en forma de array de ReflectionParameter.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Parámetros, en forma de objetos ReflectionParameter.
