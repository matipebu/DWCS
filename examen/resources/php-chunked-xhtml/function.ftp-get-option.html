ftp_get_option

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

ftp_get_option — Recupera varios comportamientos de tiempo de ejecución de la secuencia FTP actual

Descripción

ftp_get_option(resource $ftp_stream, int $option): mixed

Esta función devuelve el valor para la option solicitada de la conexión FTP especificada.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

option

Actualmente, se admiten las siguientes opciones:

Opciones FTP de tiempo de ejecución soportadas
FTP_TIMEOUT_SEC Devuelve el tiempo de espera se utiliza actualmente para las operaciones de red.
FTP_AUTOSEEK Devuelve true si esta opción está activada, false de lo contrario.

Valores devueltos

Devuelve el valor en caso de tener éxito false si el option especificado no es soportado. En este último caso, un mensaje de advertencia es también lanzado.

Ejemplos

Ejemplo #1 Ejemplo de ftp_get_option()

<?php
// Obtener el tiempo de espera de la secuencia FTP dada
$timeout = ftp_get_option($conn_id, FTP_TIMEOUT_SEC);
?>
