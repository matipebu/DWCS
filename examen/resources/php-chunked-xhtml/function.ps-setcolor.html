ps_setcolor

(PECL ps >= 1.1.0)

ps_setcolor — Establecer el color actual

Descripción

ps_setcolor(
    resource $psdoc,
    string $type,
    string $colorspace,
    float $c1,
    float $c2,
    float $c3,
    float $c4
): bool

Establece el color para dibujar, rellanar, o ambos.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

type

El parámetro type puede ser both (ambos), fill (relleno), o fillstroke (contornear).

colorspace

El espacio de color debería ser gray, rgb, cmyk, spot, pattern. Dependiendo del espacio de color se usará solamente el primero parámetro, los tres primeros o todos los parámetros.

c1

Dependiendo del espacio de color es el componente rojo (rgb), el componente cian (cmyk), el valor de gris (gray), el identificador del color directo (spot) o el identificador del patrón (pattern).

c2

Dependiendo del espacio de color es el componente verde (rgb) o el componente magenta (cmyk).

c3

Dependiendo del espacio de color es el componente azul (rgb) o el componente amarillo (cmyk).

c4

Este parámetro sólo debe establecerse en el espacio de color cmyk, y especifica el componente negro.

Errores

El segundo parámetro actualmente no siempre es evaluado. El color es a veces establecido para relleno o dibujo como si se le pasara fillstroke.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
