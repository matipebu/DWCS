acos

(PHP 4, PHP 5, PHP 7, PHP 8)

acos — Arco coseno

Descripción

acos(float $num): float

Devuelve el arco coseno de num en radianes. acos() es la función complementaria de cos(), lo que quiere decir que a==cos(acos(a)) para cada valor de a en el rango de acos()'.

Parámetros

num

El argumento a procesar

Valores devueltos

The arc cosine of num in radians.
