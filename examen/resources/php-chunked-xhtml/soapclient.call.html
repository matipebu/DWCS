SoapClient::__call

(PHP 5, PHP 7, PHP 8)

SoapClient::__call — Llama a una función de SOAP (obsoleto)

Descripción

public SoapClient::__call(string $function_name, array $arguments): mixed

Llamar a este método directamente está obsoleto. Normalmente, las funciones de SOAP pueden ser invocadas como métodos del objeto SoapClient; en situaciones donde esto no es posible o son necesarias opciones adicionales, use SoapClient::__soapCall().
