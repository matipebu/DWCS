DatePeriod::getDateInterval

(PHP 5 >= 5.6.5, PHP 7, PHP 8)

DatePeriod::getDateInterval — Obtener el intervalo

Descripción

Estilo orientado a objetos

public DatePeriod::getDateInterval(): DateInterval

Obtiene un objeto DateInterval representa el intervalo empleado para el periodo.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un objeto DateInterval

Ejemplos

Ejemplo #1 Ejemplo de DatePeriod::getDateInterval()

<?php
$periodo = new DatePeriod('R7/2016-05-16T00:00:00Z/P1D');
$intervalo = $periodo->getDateInterval();
echo $intervalo->format('%d day');
?>

El resultado del ejemplo sería:

1 day
