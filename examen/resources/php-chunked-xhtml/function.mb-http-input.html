mb_http_input

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

mb_http_input — Detecta la codificación de caracteres de entrada HTTP

Descripción

mb_http_input(string $type = ""): mixed

Detecta la codificación de caracteres de entrada HTTP.

Parámetros

type

Un string de entrada que especifica el tipo de entrada. "G" para GET, "P" para POST, "C" para COOKIE, "S" para string, "L" para una lista, y "I" para una lista completa (devolverá un array). Si se omite el tipo, devolverá el último tipo de entrada procesado.

Valores devueltos

El nombre de la codificación de caracteres, según type. Si mb_http_input() no procesa la entrada HTTP especificada, devolverá false.
