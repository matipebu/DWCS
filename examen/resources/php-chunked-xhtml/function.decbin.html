decbin

(PHP 4, PHP 5, PHP 7, PHP 8)

decbin — Decimal a binario

Descripción

decbin(int $number): string

Devuelve una cadena que contiene una representación binaria del argumento number.

Parámetros

number

Ejemplos

Ejemplo #1 Ejemplo de decbin()

<?php
echo decbin(12) . "\n";
echo decbin(26);
?>
