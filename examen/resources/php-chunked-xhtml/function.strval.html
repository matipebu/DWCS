strval

(PHP 4, PHP 5, PHP 7, PHP 8)

strval — Obtiene el valor de cadena de una variable

Descripción

strval(mixed $var): string

Obtener el valor de la cadena de una variable. Vea la documentación sobre el tipo de dato string para obtener más información sobre la conversión a cadena.

Esta función no realiza ninguna de formato en el valor devuelto. Si está buscando una manera de dar formato a un valor numérico como una cadena, por favor consulte las funciones sprintf() o number_format().

Parámetros

var

La variable que se está convirtiendo en un string.

var puede ser cualquier tipo escalar o un objeto que implemente el método __toString(). No puede usar strval() en arrays u objetos que no implementen el método __toString().

Valores devueltos

El valor string de var.

Ejemplos

Ejemplo #1 strval() usando el método mágico __toString() de PHP 5

<?php
class StrValTest
{
public function __toString()
{
return __CLASS__;
}
}

// Imprime 'StrValTest'
echo strval(new StrValTest);
?>
