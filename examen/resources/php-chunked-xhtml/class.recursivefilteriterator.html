La clase RecursiveFilterIterator

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Introducción

Este iterador abstracto filtra valores no deseados de RecursiveIterator. Esta clase debería ampliarse para implementar filtros personalizados. El método RecursiveFilterIterator::accept() debe ser implementado en la subclase.
