finfo_set_flags

finfo::set_flags

(PHP >= 5.3.0, PHP 7, PHP 8, PECL fileinfo >= 0.1.0)

finfo_set_flags -- finfo::set_flags — Opciones de configuración de libmagic

Descripción

Estilo por procedimientos

finfo_set_flags(resource $finfo, int $options): bool

Estilo orientado a objetos

public finfo::set_flags(int $options): bool

Esta función establece varias opcines Fileinfo. Las opciones pueden establecerse directamente en finfo_open() u otras funciones Fileinfo.

Parámetros

finfo

Recurso devuelto por finfo_open().

options

Una o más constantes Fileinfo .

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
