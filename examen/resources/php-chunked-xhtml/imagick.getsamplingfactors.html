Imagick::getSamplingFactors

(PECL imagick 2, PECL imagick 3)

Imagick::getSamplingFactors — Obtiene el factor de muestreo horizontal y vertical

Descripción

Imagick::getSamplingFactors(): array

Obtiene el factor de muestreo horizontal y vertical

Valores devueltos

Devuelve un array asociativo con el factor de muestreo horizontal y vertical de la imagen.

Errores/Excepciones

Lanza ImagickException en caso de error.
