curl_multi_errno

(PHP 7 >= 7.1.0, PHP 8)

curl_multi_errno — Devuelve el último número de error de cURL múltiple

Descripción

curl_multi_errno(resource $mh): int

Devuelve un integer que contiene el último número de error de cURL múltiple.

Parámetros

mh

El multi recurso cURL multi devuelto por curl_multi_init().

Valores devueltos

Devuelve un integer que contiene el último número de error de cURL múltiple, o false en caso de error.
