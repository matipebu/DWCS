IntlChar::isISOControl

(PHP 7, PHP 8)

IntlChar::isISOControl — Comprobar si un punto de código es un código de control de ISO

Descripción

public static IntlChar::isISOControl(mixed $codepoint): bool

Determina si el punto de código especificado es un código de control de ISO.

true para U+0000..U+001f y U+007f..U+009f (categoría general "Cc").

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devueve true si codepoint es un código de control de ISO, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isISOControl(" "));
var_dump(IntlChar::isISOControl("\n"));
var_dump(IntlChar::isISOControl("\u{200e}"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(false)
