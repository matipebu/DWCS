xml_parser_free

(PHP 4, PHP 5, PHP 7, PHP 8)

xml_parser_free — Liberar una analizador XML

Descripción

xml_parser_free(resource $parser): bool

Liberar el parser XML dado.

Parámetros

parser
Una referencia a el analizador a liberar.

Valores devueltos

Ésta función devuelve false si parser no refiere a un analizador valido, o sino lo libera al analizador y devuelve true.
