XMLWriter::endComment

xmlwriter_end_comment

(PHP 5 >= 5.1.2, PHP 7, PHP 8, PECL xmlwriter >= 1.0.0)

XMLWriter::endComment -- xmlwriter_end_comment — Crea un comentario final

Descripción

Estilo orientado a objetos

public XMLWriter::endComment(): bool

Estilo por procedimientos

xmlwriter_end_comment(XMLWriter $writer): bool

Crea un comentario final.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Historial de cambios

Versión Descripción
8.0.0 writer expects an XMLWriter instance now; previously, a resource was expected.
