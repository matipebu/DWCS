SoapServer::handle

(PHP 5, PHP 7, PHP 8)

SoapServer::handle — Controla la petición SOAP

Descripción

public SoapServer::handle(string $soap_request = ?): void

Procesa una petición SOAP, llama a las funciones necesarias y envía una respuesta.

Parámetros

soap_request

La petición SOAP. Si se omite este argumento, se asume que la petición está en los datos POST de la petición HTTP.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de SoapServer::handle()

<?php
function test($x)
{
return $x;
}

$server = new SoapServer(null, array('uri' => "http://test-uri/"));
$server->addFunction("test");
$server->handle();
?>
