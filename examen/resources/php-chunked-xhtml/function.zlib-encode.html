zlib_encode

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

zlib_encode — Comprime datos con la codificación especificada

Descripción

zlib_encode(string $data, int $encoding, int $level = -1): string|false

Comprime datos con la codificación especificada.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

data

Los datos a comprimir.

encoding

El algoritmo de compresión. O bien ZLIB_ENCODING_RAW, ZLIB_ENCODING_DEFLATE o ZLIB_ENCODING_GZIP.

level

Valores devueltos

Ejemplos

Ejemplo #1 Ejemplo de zlib_encode()

<?php
$str = 'hello world';
$enc = zlib_encode($str, ZLIB_ENCODING_DEFLATE);
echo bin2hex($enc);
?>

El resultado del ejemplo sería:

789ccb48cdc9c95728cf2fca4901001a0b045d
