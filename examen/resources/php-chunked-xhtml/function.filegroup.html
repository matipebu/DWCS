filegroup

(PHP 4, PHP 5, PHP 7, PHP 8)

filegroup — Obtiene el grupo de un archivo

Descripción

filegroup(string $filename): int

Obtiene el grupo del archivo. El ID de grupo es devuelto en formato numérico, use posix_getgrgid() para resolverlo a un nombre de grupo.

Parámetros

filename

Ruta del archivo.

Valores devueltos

Devuelve el ID de grupo del archivo, o false en caso de que se produzca un error. El ID de grupo es devuelto en formato numérico, use posix_getgrgid() para resolverlo a un nombre de grupo. Si falla, se devuelve false

Ejemplos

Ejemplo #1 Encontrar el grupo de un archivo

<?php
$nombre_archivo = 'index.php';
print_r(posix_getgrgid(filegroup($nombre_archivo)));
?>

Errores/Excepciones

Si falla, se emite un E_WARNING.

Notas

Nota: Los resultados de esta función son guardados en caché. Para más información ver clearstatcache().

Sugerencia

Desde PHP 5.0.0, esta función también puede usarse con algunos URL wrappers. Consultar Protocolos y Envolturas soportados para deter4minar qué envolturas soportan la familia de funcionalidad de stat().
