posix_setpgid

(PHP 4, PHP 5, PHP 7, PHP 8)

posix_setpgid — Establecer el id de grupo de procesos para el control de trabajo

Descripción

posix_setpgid(int $pid, int $pgid): bool

Permite al proceso pid unirse al grupo de procesos pgid.

Parámetros

pid

El id del proceso.

pgid

El id de grupo de procesos.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ver también

  • Véase POSIX.1 y la página del manual setsid(2) del sistema POSIX para más información sobre grupos de proceoss y control de trabajo.
