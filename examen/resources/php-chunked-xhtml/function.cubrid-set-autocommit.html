cubrid_set_autocommit

(PECL CUBRID >= 8.4.0)

cubrid_set_autocommit — Establece el modo de confirmación automática de la conexión

Descripción

cubrid_set_autocommit(resource $conn_identifier, bool $mode): bool

La función cubrid_set_autocommit() es usada para establecer el modo de confirmación automática de la conexión de base de datos actual.

En CUBRID PHP, el modo de confirmación automática está desactivado por defecto para el manejo de la transacción. Cuando el modo de confirmación automática es encendido, cualquier trabajo pendiente es confirmado automáticamente.

Parámetros

conn_identifier

Identifiador de conexión.

mode

Modo de confirmación automática. Las siguientes constantes podrán ser utilizadas:

  • CUBRID_AUTOCOMMIT_FALSE
  • CUBRID_AUTOCOMMIT_TRUE

Valores devueltos

true, cuando el proceso ha sido exitoso.

false, cuando el proceso no tuvo éxito.
