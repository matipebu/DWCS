ImagickDraw::getVectorGraphics

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getVectorGraphics — Devuelve una cadena que contiene gráficos vectoriales

Descripción

public ImagickDraw::getVectorGraphics(): string
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve un string que especifica los gráficos vectoriales generados por cualquier llamada a gráficos hecha desde que el objeto ImagickDraw fue instanciado.

Valores devueltos

Devuelve un string que contiene los gráficos vectoriales.
