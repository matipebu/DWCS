ArrayObject::__construct

(PHP 5, PHP 7, PHP 8)

ArrayObject::__construct — Construir un nuevo objeto Array

Descripción

public ArrayObject::__construct(mixed $input = array(), int $flags = 0, string $iterator_class = "ArrayIterator")

Esto crea un nuevo objeto array.

Parámetros

input

Este parámetro de input (entrada) acepta un array o un objeto.

flags

Flags para controlar el comportamiento del objeto ArrayObject.

iterator_class

Especifica la clase que va a ser usada para la iteración del objeto ArrayObject. ArrayIterator es la clase usada por omisión.

Valores devueltos

Devuelve un objeto ArrayObject en caso de éxito.

Errores/Excepciones

Lanza una InvalidArgumentException cuando:

  • input no es un array u objeto
  • flags no es un integer
  • iterator_class no es un objeto que implementa Iterator

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::__construct()

<?php
$array = array('1' => 'uno',
'2' => 'dos',
'3' => 'tres');

$arrayobject = new ArrayObject($array);

var_dump($arrayobject);
?>

El resultado del ejemplo sería:

object(ArrayObject)#1 (3) {
  [1]=>
  string(3) "uno"
  [2]=>
  string(3) "dos"
  [3]=>
  string(5) "tres"
}
