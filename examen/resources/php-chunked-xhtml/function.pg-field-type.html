pg_field_type

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

pg_field_type — Devuelve el nombre de tipo para el número de campo correspondiente

Descripción

pg_field_type(resource $result, int $field_number): string

pg_field_type() Devuelve un string que contiene el nombre del tipo base para el field_number en el resultado de consulta result dado de PostgreSQL.

Nota:

Si el campo utiliza un dominio PostgreSQL (en lugar de un tipo básico), es el nombre del tipo subyacente del dominio el que se devuelve, en lugar del nombre del propio dominio.

Nota:

Esta función solía llamarse pg_fieldtype().

Parámetros

result

Resultado de consulta de PostgreSQL, devuelto por pg_query(), pg_query_params() o pg_execute() (entre otros).

field_number

Número de campo, a partir de 0.

Valores devueltos

Un string que contiene el nombre base del tipo del campo, o false En caso de Fallo.

Ejemplos

Ejemplo #1 Obtención de información sobre los campos

<?php
$dbconn = pg_connect("dbname=publisher") or die("Could not connect");

// Assume 'title' is a varchar type
$res = pg_query($dbconn, "select title from authors where author = 'Orwell'");

echo "Campo title de tipo: ", pg_field_type($res, 0);
?>

El resultado del ejemplo sería:

Title field type: varchar
