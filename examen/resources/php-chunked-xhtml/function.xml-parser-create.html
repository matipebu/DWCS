xml_parser_create

(PHP 4, PHP 5, PHP 7, PHP 8)

xml_parser_create — Crea un intérprete XML

Descripción

xml_parser_create(string $encoding = ?): resource

xml_parser_create() crea un nuevo intérprete XML y devuelve un identificador del recurso al que hace referencia, para ser usado por otras funciones XML.

Parámetros

encoding

El parámetro opcional encoding especifica la codificación de caracteres para entrada y salida en PHP4. A partir de PHP5, la codificación de entrada se detecta automaticamente, así que el parámetro encoding especifica solo la codificación de salida. En PHP 4, la codificación por defecto de salida es la misma que la del juego de caracteres de la entrada. Si se le pasa una cadena vacia, el intérprete intenta identificar cual es la codificación del documento mirando los 3 o 4 primeros bytes. En PHP 5.0.0 y 5.0.1, el juego de caracteres por defecto es ISO-8859-1, mientras que en PHP 5.0.2 y superior es UTF-8. Las codificaciones soportadas son ISO-8859-1, UTF-8 y US-ASCII.

Valores devueltos

Devuelve un identificador del recurso para el nuevo intérprete XML.
