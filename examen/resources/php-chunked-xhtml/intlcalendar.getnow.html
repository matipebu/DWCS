IntlCalendar::getNow

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getNow — Obtener el número que representa el instante actual

Descripción

Estilo orientado a objetos

public static IntlCalendar::getNow(): float

Estilo por procedimientos

intlcal_get_now(): float

El número de milisegundos que han pasado desde la fecha de referencia. Este número es derivado del instante del sistema.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un valor de tipo float que representa el número de milisegundos desde la época, sin incluir los segundos intercalares.

Ejemplos

Ejemplo #1 IntlCalendar::getNow()

<?php
$formatter = IntlDateFormatter::create('es_ES',
IntlDateFormatter::FULL,
IntlDateFormatter::FULL,
'Europe/Madrid');

$val = IntlCalendar::getNow();

var_dump($val);
echo $formatter->format(IntlCalendar::getNow() / 1000.), "\n";

El resultado del ejemplo sería:

float(1371425814666)
lunes, 17 de junio de 2013 01:36:54 Hora de verano de Europa central
