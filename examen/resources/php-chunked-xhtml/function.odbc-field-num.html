odbc_field_num

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_field_num — Devolver el número de columna

Descripción

odbc_field_num(resource $result_id, string $field_name): int

Obtiene el número del espacio de columna que se corresponde con el campo nominado del identificador de resultado dado.

Parámetros

result_id

El identificador de resultado.

field_name

El nombre del campo.

Valores devueltos

Devuelve el número de campo como un valor entero, o false si se produjo un error. La numeración de campos comienza en 1.
