pspell_config_repl

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

pspell_config_repl — Establecer un fichero que contiene pares de sustitución

Descripción

pspell_config_repl(int $dictionary_link, string $file): bool

Establece un fichero que contiene pares de sustitución.

Los pares de sustitución mejoran la calidad del corrector ortográfico. Cuando una palabra está mal escrita y no se ha encontrado una sugerencia apropiada en la lista, se puede usar pspell_store_replacement() para almacenar un par de sustitución y después pspell_save_wordlist() para guardar la lista de palabras junto con los pares de sustitución.

pspell_config_repl() debería usarse con una configuración antes de llamar a pspell_new_config().

Parámetros

dictionary_link

file

El fichero debería ser modificable por quienquiera que ejecute PHP (p.ej. nadie).

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 pspell_config_repl()

<?php
$pspell_config = pspell_config_create("en");
pspell_config_personal($pspell_config, "/var/dictionaries/custom.pws");
pspell_config_repl($pspell_config, "/var/dictionaries/custom.repl");
$pspell_link = pspell_new_config($pspell_config);
pspell_check($pspell_link, "thecat");
?>

Notas

Nota:

Esta función no funcionará a menos que tenga pspell .11.2 y aspell .32.5 o posterior.
