enchant_broker_free

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_broker_free — Liberar el recurso de agente y sus diccionarios

Descripción

enchant_broker_free(resource $broker): bool

Libera un recurso de agente con todos sus diccionarios.

Parámetros

broker

Recurso de agente

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
