msg_set_queue

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

msg_set_queue — Establecer información en la estructura de datos de la cola de mensajes

Descripción

msg_set_queue(resource $queue, array $data): bool

msg_set_queue() permite que se cambien los valores de los campos msg_perm.uid, msg_perm.gid, msg_perm.mode y msg_qbytes fields de la estructura de datos de la cola de mensajes subyacente.

Cambiar la estructura de datos requerirá que PHP sea ejecutado mediante el mismo usuario que creó la cola, los propietarios de la cola (como está determinado por los campos msg_perm.xxx existentes), o sea ejecutado con privilegios root. Se requieren los privilegios root para aumentar los valores de msg_qbytes por encima del límite definido por el sistema.

Parámetros

queue

Gestor de recurso de cola de mensajes

data

Se especifican los valores que se requieren estableciendo el valor de las claves que se neceditan en la matriz data.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
