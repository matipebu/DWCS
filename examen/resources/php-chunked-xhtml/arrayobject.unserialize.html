ArrayObject::unserialize

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

ArrayObject::unserialize — Deserializar el objeto ArrayObject

Descripción

public ArrayObject::unserialize(string $serialized): void

Deserializa un objeto ArrayObject serializado.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

serialized

El objeto ArrayObject serializado.

Valores devueltos

El objeto ArrayObject deserializado.
