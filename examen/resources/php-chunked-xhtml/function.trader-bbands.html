trader_bbands

(PECL trader >= 0.2.0)

trader_bbands — Bandas de Bollinger

Descripción

trader_bbands(
    array $real,
    int $timePeriod = ?,
    float $nbDevUp = ?,
    float $nbDevDn = ?,
    int $mAType = ?
): array

Parámetros

real

Array de valores reales.

timePeriod

Número del periodo. Rango válido desde 2 hasta 100000.

nbDevUp

Desviación multiplicadora para la banda superior. Rango válido desde TRADER_REAL_MIN hasta TRADER_REAL_MAX.

nbDevDn

Desviación multiplicadora para la banda inferior. Rango válido desde TRADER_REAL_MIN hasta TRADER_REAL_MAX.

mAType

Tipo de media móvil. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de fallo.
