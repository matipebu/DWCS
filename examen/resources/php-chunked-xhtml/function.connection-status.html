connection_status

(PHP 4, PHP 5, PHP 7, PHP 8)

connection_status — Devuelve el campo de bits de status de conexión

Descripción

connection_status(): int

Obtiene el campo de bits de status de la conexión.

Valores devueltos

Devuelve el campo de bits de status de la conexión, el cual puede ser usado junto con las constantes CONNECTION_XXX para determinar el status de conexión.
