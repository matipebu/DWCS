rmdir

(PHP 4, PHP 5, PHP 7, PHP 8)

rmdir — Elimina un directorio

Descripción

rmdir(string $directory, resource $context = ?): bool

Intenta eliminar el directorio nombrado por directory. El directorio debe estar vacío, y los permisos relevantes deben permitirlo. Un error de nivel E_WARNING será generado si se produce un error.

Parámetros

directory

Ruta del directorio

context

Nota: Soporte para context fue añadido en PHP 5.0.0. Para una descripción de contexts, refiérase a Flujos.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de rmdir()

<?php
if (!is_dir('ejemplos')) {
mkdir('ejemplos');
}

rmdir('ejemplos');
?>
