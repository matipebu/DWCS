trader_macdext

(PECL trader >= 0.2.0)

trader_macdext — MACD con tipo MA controlable

Descripción

trader_macdext(
    array $real,
    int $fastPeriod = ?,
    int $fastMAType = ?,
    int $slowPeriod = ?,
    int $slowMAType = ?,
    int $signalPeriod = ?,
    int $signalMAType = ?
): array

Parámetros

real

Array de valores reales.

fastPeriod

Número del periodo para el MA rápido. Rango válido desde 2 hasta 100000.

fastMAType

Tipo de media móvil para MA rápido. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

slowPeriod

Número del periodo para el MA lento. Rango válido desde 2 hasta 100000.

slowMAType

Tipo de media móvil para MA lento. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

signalPeriod

Suavizado para la línea de señal (nb de periodo). Rango válido desde 1 hasta 100000.

signalMAType

Tipo de media móvil para la línea de señal. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
