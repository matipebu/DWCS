gmp_divexact

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_divexact — División exacta de números

Descripción

gmp_divexact(GMP|int|string $num1, GMP|int|string $num2): GMP

Divide num1 por num2, usando el rápido algoritmo de "división exacta". Ésta función produce el resultado correcto solo cuando es conocida por adelantado num2 divide a num1.

Parámetros

num1

El número que es dividido.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

num2

El número que es dividio por a.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

resource de número de GMP en PHP 5.5 y anterior, o un objeto GMP en PHP 5.6 y posterior.

Ejemplos

Ejemplo #1 Ejemplo degmp_divexact()

<?php
$div1 = gmp_divexact("10", "2");
echo gmp_strval($div1) . "\n";

$div2 = gmp_divexact("10", "3"); // resultado falso
echo gmp_strval($div2) . "\n";
?>

El resultado del ejemplo sería:

5
