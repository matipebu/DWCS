Soporte para Windows

configure flags

configure ahora considera a los conjuntos adicionales como CFLAGS y LDFLAGS variables de entorno.

Manejo de CTRL

CTRL+C y CTRL+BREAK en la consola pueden ser capturados mediante la configuración de una función de manejo con sapi_windows_set_ctrl_handler().

proc_open() en Windows se puede pasar una opción "create_process_group". Se requiere si el proceso del hijo se supone que maneja eventos CTRL.

OPcache

OPcache ahora soporta un número arbitrario de cachés separados por usuario a través de la directiva INI opcache.cache_id. Todos los procesos con el mismo ID de caché y el usuario comparten una instancia OPcache.

stat

La implementación de la estadística se ha refactorizado.

  • Un número inode An inode number se entregado y se basa en el índice de archivos NTFS.
  • El número del dispositivo se basa ahora en el número de serie del volumen.

Obsérvese que ambos valores se derivan del sistema y se proporcionan tal cual en los sistemas de 64 bits. En sistemas de 32 bits, estos valores podrían desbordar el entero de 32 bits en PHP, así que son falsos.

libsqlite3

libsqlite3 ya no está compilada estáticamente en php_sqlite3.dll y php_pdo_sqlite.dll, pero más bien disponible como libsqlite3.dll. Consulte las instrucciones de instalación para SQLite3 y PDO_SQLITE, respectivamente.
