ftp_rawlist

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_rawlist — Devuelve una lista detallada de archivos en el directorio especificado

Descripción

ftp_rawlist(resource $ftp_stream, string $directory, bool $recursive = false): mixed

ftp_rawlist() ejecuta el comando FTP LIST, y devuelve el resultado como un array.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

directory

La ruta del directorio. Puede incluir argumentos para el comando LIST.

recursive

Si se establece a true, el comando emitido será LIST -R.

Valores devueltos

Devuelve un array donde cada elemento corresponde a una línea de texto. Devuelve false al pasar un directory no válido.

La salida no se analiza de forma alguna. El identificador del tipo de sistema devuelto por ftp_systype() se puede utilizar para determinar cómo los resultados deben ser interpretados.

Ejemplos

Ejemplo #1 Ejemplo de ftp_rawlist()

<?php

// establecer una conexión básica
$conn_id = ftp_connect($ftp_server);

// iniciar sesión con nombre de usuario y contraseña
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// obtener la lista de archivos de /
$buff = ftp_rawlist($conn_id, '/');

// cerrar la conexión ftp
ftp_close($conn_id);

// imprimir el contenido del buffer
var_dump($buff);
?>

El resultado del ejemplo sería algo similar a:

array(3) {
  [0]=>
  string(65) "drwxr-x---   3 vincent  vincent      4096 Jul 12 12:16 public_ftp"

  [2]=>
  string(73) "lrwxrwxrwx   1 vincent  vincent        11 Jul 12 12:16 www -> public_html"
}
