ftp_raw

(PHP 5, PHP 7, PHP 8)

ftp_raw — Envía un comando arbitrario a un servidor FTP

Descripción

ftp_raw(resource $ftp_stream, string $command): array

Envía un command arbitrario al servidor FTP.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

command

El comando a ejecutar.

Valores devueltos

Devuelve la respuesta del servidor como una matriz de cadenas. No se realiza procesamiento alguno con la cadena de respuesta, ni ftp_raw() determina si el comando tuvo éxito.

Ejemplos

Ejemplo #1 Uso de ftp_raw() para iniciar sesión en un servidor FTP de forma manual.

<?php
$fp = ftp_connect("ftp.example.com");

/* Esto es lo mismo que:
ftp_login($fp, "joeblow", "secreto"); */
ftp_raw($fp, "USER joeblow");
ftp_raw($fp, "PASS secreto");
?>
