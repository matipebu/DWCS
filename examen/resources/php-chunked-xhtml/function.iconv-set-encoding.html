iconv_set_encoding

(PHP 4 >= 4.0.5, PHP 5, PHP 7, PHP 8)

iconv_set_encoding — Establece las opciones para la conversión de codificación de caracteres

Descripción

iconv_set_encoding(string $type, string $charset): bool

Cambia el valor de la variable de configuración interna especificada por type a charset.

Parámetros

type

El valor de type puede ser cualquiera de los siguientes:

  • input_encoding
  • output_encoding
  • internal_encoding

charset

La codificación de caracteres.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de iconv_set_encoding()

<?php
iconv_set_encoding("internal_encoding", "UTF-8");
iconv_set_encoding("output_encoding", "ISO-8859-1");
?>
