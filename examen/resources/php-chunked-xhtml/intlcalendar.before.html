IntlCalendar::before

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::before — Indica si el instante de este objeto es anterior al del proporcionado

Descripción

Estilo orientado a objetos

public IntlCalendar::before(IntlCalendar $other): bool

Estilo por procedimientos

intlcal_before(IntlCalendar $cal, IntlCalendar $other): bool

Devuelve si el instante de este objeto es anterior al instante del argumento.

Parámetros

cal

El recurso IntlCalendar.

other

El calendario cuyo instante será comprobado con el instante del objeto primario.

Valores devueltos

Devuelve true si el instante de este objeto es anterior al instante del argumento calendar. Devuelve false en caso contrario. También devuelve false en caso de error. Se puede utilizar exceptions o intl_get_error_code() para detectar condiciones de error.
