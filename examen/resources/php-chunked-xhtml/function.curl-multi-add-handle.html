curl_multi_add_handle

(PHP 5, PHP 7, PHP 8)

curl_multi_add_handle — Añade un recurso cURL a un grupo de recursos cURL

Descripción

curl_multi_add_handle(resource $mh, resource $ch): int

Añade el recuso ch al grupo de recursos mh

Parámetros

mh

El multi recurso cURL multi devuelto por curl_multi_init().

ch

El recurso cURL devuelto por curl_init().

Valores devueltos

En caso de éxito devuelve 0, sino devuelve alguno de los códigos de error CURLM_XXX

Ejemplos

Ejemplo #1 curl_multi_add_handle() example

Este ejemplo creará dos recursos cURL, los añadirá a un grupo de recursos y entoces los ejecutará de forma asíncrona.

<?php
// Crea los dos recursos cURL
$ch1 = curl_init();
$ch2 = curl_init();

// Establecer la URL y otras opciones apropiadas
curl_setopt($ch1, CURLOPT_URL, "http://www.example.com/");
curl_setopt($ch1, CURLOPT_HEADER, 0);
curl_setopt($ch2, CURLOPT_URL, "http://www.php.net/");
curl_setopt($ch2, CURLOPT_HEADER, 0);

// Crea el recurso cURL múltiple
$mh = curl_multi_init();

// Añade los dos recursos
curl_multi_add_handle($mh,$ch1);
curl_multi_add_handle($mh,$ch2);

// Ejecuta los recursos múltiples
do {
$status = curl_multi_exec($mh, $active);
if ($active) {
curl_multi_select($mh);
}
} while ($active && $status == CURLM_OK);

// Cierra todos los recursos
curl_multi_remove_handle($mh, $ch1);
curl_multi_remove_handle($mh, $ch2);
curl_multi_close($mh);
?>
