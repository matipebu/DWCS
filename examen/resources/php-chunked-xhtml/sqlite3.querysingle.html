SQLite3::querySingle

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SQLite3::querySingle — Ejecutar una consulta y devolver un único resultado

Descripción

public SQLite3::querySingle(string $query, bool $entire_row = false): mixed

Ejecuta una consulta y devuelve un único resultado.

Parámetros

query

La consulta SQL a ejecutar.

entire_row

Por omisión, querySingle() devuelve el valor de la primera columna devuelta por la consulta. Si entire_row es true, devuelve un array con la primera fila completa.

Valores devueltos

Devuelve el valor de la primera columna de resultados o un array con la primera fila completa (si entire_row es true).

Si la consulta es válida pero no se devuelven resultados, se devolverá null si entire_row es false, de otro modo se devolverá un array vacío.

Las consultas no válidas o defectuosas devolverán false.

Ejemplos

Ejemplo #1 Ejemplo de SQLite3::querySingle()

<?php
$bd = new SQLite3('mibdsqlite.db');

var_dump($bd->querySingle('SELECT username FROM user WHERE userid=1'));
print_r($bd->querySingle('SELECT username, email FROM user WHERE userid=1', true));
?>

El resultado del ejemplo sería algo similar a:

string(5) "Scott"
Array
(
    [username] => Scott
    [email] => scott@example.com
)
