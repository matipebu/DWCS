IntlChar::isdefined

(PHP 7, PHP 8)

IntlChar::isdefined — Comprobar si un punto de código está definido

Descripción

public static IntlChar::isdefined(mixed $codepoint): bool

Determina si el punto de código especificado está «definido», lo que normalmente significa que tiene asignado un caráter.

true para las catergorías generales distintas de "Cn" (otros, no asignado).

Nota:

Observe que los puntos de código que no son un carácter (p.ej., U+FDD0) no están «definidos» (son Cn), mientras que los puntos de código sustitutos están «definidos» (Cs).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint es un carácter definido, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isdefined("A"));
var_dump(IntlChar::isdefined(" "));
var_dump(IntlChar::isdefined("\u{FDD0}"));
?>

El resultado del ejemplo sería:

bool(true)
bool(true)
bool(false)
