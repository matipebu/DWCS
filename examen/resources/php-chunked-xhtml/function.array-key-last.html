array_key_last

(PHP 7 >= 7.3.0, PHP 8)

array_key_last — Obtiene la última clave de un array

Descripción

array_key_last(array $array): mixed

Obtener la última clave del array dado sin afectar el puntero del array interno.

Parámetros

array

Un array;

Valores devueltos

Devuelve la última clave del array si el array no está vacío; null en caso contrario.
