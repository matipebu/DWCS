OAuth::disableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::disableSSLChecks — Desactiva la verificación SSL

Descripción

public OAuth::disableSSLChecks(): bool

Desactiva la verificación del certificado y servidor SSL, esto no se recomiendo para entornos en producción. Alternativamente, el miembro sslChecks puede ser puesto false para desactivar la revisión.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

true

Historial de cambios

Versión Descripción
0.99.8 El miembro sslChecks fue agregado
