gmp_random_seed

(PHP 7, PHP 8)

gmp_random_seed — Establece la semilla RNG

Descripción

gmp_random_seed(GMP|int|string $seed): void

Parámetros

seed

La semilla para las funciones gmp_random(), gmp_random_bits(), y gmp_random_range() functions.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

devuelve null en caso de éxito o false en caso de error.

Errores/Excepciones

Emite un E_WARNING y devuelve false si seed no es válido.

Ejemplos

Ejemplo #1 Ejemplo de gmp_random_seed()

<?php
// establecer la semilla
gmp_random_seed(100);

var_dump(gmp_strval(gmp_random(1)));

// establecer la semilla para algo más
gmp_random_seed(gmp_init(-100));

var_dump(gmp_strval(gmp_random_bits(10)));

// establecer la semilla a algo inválido
var_dump(gmp_random_seed('not a number'));

El resultado del ejemplo sería:

string(20) "15370156633245019617"
string(3) "683"
