GearmanTask::taskDenominator

(PECL gearman >= 0.5.0)

GearmanTask::taskDenominator — Obtiene el denominador del porcentaje de tarea completada

Descripción

public GearmanTask::taskDenominator(): int

Retorna el denominador del tanto por ciento completado de la tarea, expresado como una fracción.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un número entre 0 y 100 o false si no puede ser determinado.
