IntlChar::isMirrored

(PHP 7, PHP 8)

IntlChar::isMirrored — Comprobar si un punto de código posee la propiedad Bidi_Mirrored

Descripción

public static IntlChar::isMirrored(mixed $codepoint): bool

Determina si el punto de código posee la propiedad Bidi_Mirrored.

Esta propiedad se establece para caracteres que se utilizan normalmente en contextos Deracha-Izquierda, siendo necesario mostrarlos con un glifo «reflejado».

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint posee la propiedad Bidi_Mirrored, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isMirrored("A"));
var_dump(IntlChar::isMirrored("<"));
var_dump(IntlChar::isMirrored("("));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(true)
