Introducción

PHP incorpora soporte para funciones de entrada/salida directa tal como se describe en el estándar Posix (Sección 6) para realizar funciones de E/S a un nivel más bajo bajo que los flujos de E/S de las funciones en C (fopen(), fread(),..). Sólo debe considerarse usar las funciones DIO cuando sea necesario tener control directo sobre un dispositivo. En cualquier otro caso, las funciónes estándar del sistema de ficheros son más que suficiente.

Nota:

Esta extensión se ha movido al repositorio de » PECL y ya no se distribuye con PHP a partir de PHP 5.1.0.
