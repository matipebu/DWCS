dba_popen

(PHP 4, PHP 5, PHP 7, PHP 8)

dba_popen — Abrir una base de datos de forma persistente

Descripción

dba_popen(
    string $path,
    string $mode,
    string $handler = ?,
    mixed $... = ?
): resource

dba_popen() establece una instancia de base de datos persistente para el archivo en path con el mode indicado, usando handler.

Parámetros

path

Usualmente una ruta normal en su sistema de archivos.

mode

Es r para acceso de lectura, w para acceso de lectura/escritura a una base de datos existente, c para acceso de lectura/escritura y creación de base de datos si no existe actualmente, y n para crear, truncar y tener acceso de lectura/escritura.

handler

El nombre del gestor que será usado para acceder a path. A él se le pasan todos los parámetros opcionales dados a dba_popen() y puede actuar sobre ellos.

Valores devueltos

Devuelve un gestor positivo en caso de éxito, o o false en caso de error.
