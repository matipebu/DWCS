Yaf_Dispatcher::setErrorHandler

(Yaf >=1.0.0)

Yaf_Dispatcher::setErrorHandler — Establece el gestor de errores

Descripción

public Yaf_Dispatcher::setErrorHandler(call $callback, int $error_types): Yaf_Dispatcher

Establece el gestor de errores de Yaf. Cuando application.dispatcher.throwException está desactivada, Yaf provocará errores capturables mientras ocurran errores inesperados.

Por lo tanto, este gestor de errores será invocado mientras se produce el error.

Parámetros

callback

Una llamada de retorno de tipo callable.

error_types

Valores devueltos
