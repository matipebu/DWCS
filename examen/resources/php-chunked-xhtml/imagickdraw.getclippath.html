ImagickDraw::getClipPath

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getClipPath — Obtiene el ID del trazado de recorte actual

Descripción

ImagickDraw::getClipPath(): string
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Obtiene el ID del trazado de recorte actual.

Valores devueltos

Devuelve un string que contiene el ID del trazado de recorte o false si no existe ningún trazado de recorte.
