oci_lob_copy

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

oci_lob_copy — Copia objetos grandes

Descripción

oci_lob_copy(OCILob $to, OCILob $from, ?int $length = null): bool

Copia un objeto grande, o una parte del mismo, a otro objeto grande. Los datos de recipientes LOB antiguos serán sobrescritos.

Si fuera necesario copiar una parte en particular de un LOB a una posición en particular de otro LOB, use OCILob::seek() para mover el puntero interno del LOB.

Parámetros

lob_to

El LOB de destino.

lob_from

El LOB copiado.

length

Indica la longitud de los datos a copiar.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 length es ahora nullable.

Notas

Nota:

La clase OCILob se llamó OCI-Lob antes de PHP 8 y OCI8 3.0.0.
