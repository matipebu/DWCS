IntlChar::getIntPropertyMaxValue

(PHP 7, PHP 8)

IntlChar::getIntPropertyMaxValue — Obtener el valor máximo de una propiedad de Unicode

Descripción

public static IntlChar::getIntPropertyMaxValue(int $property): int

Obtiene el valor máximo de una propiedad de Unicode enumerada/entera/binaria.

Parámetros

property

La propiedad de Unicode a consultar (véanse las constantes IntlChar::PROPERTY_*).

Valores devueltos

El valor máximo devuelto por IntlChar::getIntPropertyValue() para una propiedad de Unicode. <=0 si el selector de la propiedad está fuera de rango.

Ejemplos

Ejemplo #1 Probar diferentes propiedades

<?php
var_dump(IntlChar::getIntPropertyMaxValue(IntlChar::PROPERTY_BIDI_CLASS));
var_dump(IntlChar::getIntPropertyMaxValue(IntlChar::PROPERTY_SCRIPT));
var_dump(IntlChar::getIntPropertyMaxValue(IntlChar::PROPERTY_IDEOGRAPHIC));
var_dump(IntlChar::getIntPropertyMaxValue(999999999)); // Un valor inventado
?>

El resultado del ejemplo sería:

int(22)
int(166)
int(1)
int(-1)
