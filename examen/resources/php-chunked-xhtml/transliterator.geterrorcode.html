Transliterator::getErrorCode

transliterator_get_error_code

(PHP 5 >= 5.4.0, PHP 7, PHP 8, PECL intl >= 2.0.0)

Transliterator::getErrorCode -- transliterator_get_error_code — Obtener el código del último error

Descripción

Estilo orientado a objetos

public Transliterator::getErrorCode(): int

Estilo por procedimientos

transliterator_get_error_code(): int

Obtiene el código del último error de este transliterador.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El código del error en caso de éxito, o false si no existe o en caso de error.
