ob_get_contents

(PHP 4, PHP 5, PHP 7, PHP 8)

ob_get_contents — Devolver el contenido del búfer de salida

Descripción

ob_get_contents(): string

Obtiene el contenido del búfer de salida, sin borrarlo.

Valores devueltos

Devolverá el contenido del búfer de salida, o false si el almacenameinto en el búfer de salida no está activo.

Ejemplos

Ejemplo #1 Un ejemplo sencillo de ob_get_contents()

<?php

ob_start();

echo "Hola ";

$salida1 = ob_get_contents();

echo "Mundo";

$salida2 = ob_get_contents();

ob_end_clean();

var_dump($salida1, $salida2);
?>

El resultado del ejemplo sería:

string(5) "Hola "
string(10) "Hola Mundo"
