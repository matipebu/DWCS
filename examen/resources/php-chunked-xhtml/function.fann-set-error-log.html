fann_set_error_log

(PECL fann >= 1.0.0)

fann_set_error_log — Establece dónde registrar los errores

Descripción

fann_set_error_log(resource $errdat, string $log_file): void

Establece dónde registrar los errores.

Parámetros

errdat

Un resource de red neuronal o un resource de datos de entrenamiento de red neuronal.

log_file

La ruta al fichero de registro.

Valores devueltos

No se devuelve ningún valor.
