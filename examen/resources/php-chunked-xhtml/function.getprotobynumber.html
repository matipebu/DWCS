getprotobynumber

(PHP 4, PHP 5, PHP 7, PHP 8)

getprotobynumber — Obtiene el nombre de protocolo asociado con un número de protocolo

Descripción

getprotobynumber(int $number): string

getprotobynumber() devuelve el nombre de protocolo asociado con el numero de procotolo, como lo indica /etc/protocols.

Parámetros

number

El número del protocolo.

Valores devueltos

Devuelve el nombre del protocolo como una cadena, o false en caso de error.
