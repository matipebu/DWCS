Imagick::mapImage

(PECL imagick 2, PECL imagick 3)

Imagick::mapImage — Reemplaza los colores de una imagen con el color más cercano de una imagen de referencia

Advertencia

This function has been DEPRECATED as of Imagick 3.4.4. Relying on this function is highly discouraged.

Descripción

public Imagick::mapImage(Imagick $map, bool $dither): bool

Parámetros

map

dither

Valores devueltos

Devuelve true en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
