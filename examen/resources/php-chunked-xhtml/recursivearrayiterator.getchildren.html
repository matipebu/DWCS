RecursiveArrayIterator::getChildren

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

RecursiveArrayIterator::getChildren — Devuelve un iterador para la entrada actual si este es un array o un object

Descripción

public RecursiveArrayIterator::getChildren(): ?RecursiveArrayIterator

Devuelve un iterador interno para la entrada actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

An iterator for the current entry, if it is an array or object; or null on failure.

Errores/Excepciones

Lanza una InvalidArgumentException si la entrada actual no contiene un array o un object.

Ejemplos

Ejemplo #1 Ejemplo de RecursiveArrayIterator::getChildren()

<?php
$fruits = array("a" => "limon", "b" => "naranja", array("a" => "manzana", "p" => "pera"));

$iterator = new RecursiveArrayIterator($fruits);

while ($iterator->valid()) {

if ($iterator->hasChildren()) {
// imprimir todos los hijos
foreach ($iterator->getChildren() as $key => $value) {
echo $key . ' : ' . $value . "\n";
}
} else {
echo "No hijos.\n";
}

$iterator->next();
}
?>

El resultado del ejemplo sería:

No hijos.
No hijos.
a : manzana
p : pera
