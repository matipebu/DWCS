trader_div

(PECL trader >= 0.2.0)

trader_div — División aritmética de vectores

Descripción

trader_div(array $real0, array $real1): array

Divide cada valor de real0 por el valor correspondiente de real1 y devuelve el array resultante.

Parámetros

real0

Array de valores reales.

real1

Array de valores reales.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de fallo.
