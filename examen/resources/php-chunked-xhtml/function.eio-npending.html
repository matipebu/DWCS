eio_npending

(PECL eio >= 0.0.1dev)

eio_npending — Devolver el número de peticiones, excepto las no manejadas

Descripción

eio_npending(): int

eio_npending() devuelve el número de peticiones, excepto las no manejadas.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

eio_npending() devuelve el número de peticiones, excepto las no manejadas.
