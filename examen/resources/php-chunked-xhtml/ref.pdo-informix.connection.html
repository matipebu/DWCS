DSN de PDO_INFORMIX — Conectarse a bases de datos de Informix

Descripción

El Nombre del Origen de Datos (DSN) de PDO_INFORMIX está basado en la cadena del DSN de ODBC de Informix. Los detalles sobre la configuración de un DSN de ODBC de Informix están disponbles en el » Centro de información de servidores dinámicas de Informix. Los pricipales componentes del DSN de PDO_INFORMIX son:

Prefijo DSN

El prefijo DSN es informix:.

DSN

El DSN puede ser una configuración de origen de datos usando odbc.ini, o una » cadena de conexión completa.

Ejemplos

Ejemplo #1 Ejemplo de DSN de PDO_INFORMIX usando odbc.ini

El siguiente ejemplo muestra un DSN de PDO_INFORMIX para conectarse a una base de datos de Informix catalogada como Infdrv33 en odbc.ini: 

$bd = new PDO("informix:DSN=Infdrv33", "", "");

[ODBC Data Sources]
Infdrv33=INFORMIX 3.3 32-BIT

[Infdrv33]
Driver=/opt/informix/csdk_2.81.UC1G2/lib/cli/iclis09b.so
Description=INFORMIX 3.3 32-BIT
Database=common_db
LogonID=testuser
pwd=testpass
Servername=ids_server
DB_LOCALE=en_US.819
OPTIMIZEAUTOCOMMIT=1
ENABLESCROLLABLECURSORS=1

Ejemplo #2 Ejemplo de DSN de PDO_INFORMIX usando una cadena de conexión

El siguiente ejemplo muestra un DSN de PDO_INFORMIX DSN para conectarse a una base de datos de Informix llamada common_db usando la sintaxis de cadena de conexión de Informix. 

$bd = new PDO("informix:host=host.domain.com; service=9800;
    database=common_db; server=ids_server; protocol=onsoctcp;
    EnableScrollableCursors=1", "testuser", "tespass");
