pspell_suggest

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

pspell_suggest — Sugerir ortografías de una palabra

Descripción

pspell_suggest(int $dictionary_link, string $word): array

pspell_suggest() devuelve un array de posibles ortografías para la palabra dada.

Parámetros

dictionary_link

word

La palabra comprobada.

Valores devueltos

Devuelve un array de posibles ortografías.

Ejemplos

Ejemplo #1 Ejemplo de pspell_suggest()

<?php
$pspell_link = pspell_new("en");

if (!pspell_check($pspell_link, "testt")) {
$sugerencias = pspell_suggest($pspell_link, "testt");

foreach ($sugerencias as $sugerencia) {
echo "Ortografía posible: $sugerencia<br />";
}
}
?>
