Imagick::setSizeOffset

(PECL imagick 2, PECL imagick 3)

Imagick::setSizeOffset — Establece el tamaño y el índice del objeto Imagick

Descripción

public Imagick::setSizeOffset(int $columns, int $rows, int $offset): bool

Establece el tamaño y el índice del objeto Imagick. Se debe establecer antes de leer un formato de imagen en bruto como RGB, GRAY, o CMYK. Este método está disponible si Imagick ha sido compilado con la versión 6.2.9 o superior de ImageMagick.

Parámetros

columns

El ancho en píxeles.

rows

El alto en píxeles.

offset

El índice de la imagen.

Valores devueltos

Devuelve true en caso de éxito.
