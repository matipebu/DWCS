Reflection::getModifierNames

(PHP 5, PHP 7, PHP 8)

Reflection::getModifierNames — Obtener nombres de modificadores

Descripción

public static Reflection::getModifierNames(int $modifiers): array

Obtiene los nombres de los modificadores.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

modifiers

Modificadores que se desean obtener, a partir de un valor numérico.

Valores devueltos

Un array con nombres de modificadores.
