IntlCalendar::setRepeatedWallTimeOption

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::setRepeatedWallTimeOption — Establecer el comportamiento para tratar horas repetidas en transiciones negativas de índices de zonas horarias

Descripción

Estilo orientado a objetos

public IntlCalendar::setRepeatedWallTimeOption(int $wallTimeOption): bool

Estilo por procedimientos

intlcal_set_repeated_wall_time_option(IntlCalendar $cal, int $wallTimeOption): bool

Establece la estrategia a usar para tratar con las horas repetidas siempre que el reloj se atrase durante las transiciones del final del horario de verano (DST). El valor predeterminado es IntlCalendar::WALLTIME_LAST (tomado como si fuera el instante pos-DST). El otro valor posible es IntlCalendar::WALLTIME_FIRST (tomando el mismo instante que ocurre durante el DST).

Esta función requiere ICU 4.9 o posterior.

Parámetros

cal

El recurso IntlCalendar.

wallTimeOption

Una de las constantes IntlCalendar::WALLTIME_FIRST o IntlCalendar::WALLTIME_LAST.

Valores devueltos

Devuelve true en caso de éxito. Los errores solamente pueden ocurrir debido a parámetros no válidos.
