openssl_x509_export_to_file

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

openssl_x509_export_to_file — Exporta un certificado a un archivo

Descripción

openssl_x509_export_to_file(mixed $x509, string $outfilename, bool $notext = true): bool

openssl_x509_export_to_file() guarda x509 en un archivo nombrado por outfilename en un formato PEM codificado.

Parámetros

x509

Ver parámetros Key/Certificate para un listado de valores válidos.

outfilename

Ruta del archivo de salida.

notext

El parámetro opcional notext afecta la verbosidad de la salida; si es false, se incluirá información adicional en la salida entendible por seres humanos. El valor por defecto para notext es true.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
