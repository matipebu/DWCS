getprotobyname

(PHP 4, PHP 5, PHP 7, PHP 8)

getprotobyname — Obtener el número de protocolo asociado con el nombre de protocolo

Descripción

getprotobyname(string $name): int

getprotobyname() devuelve el número de protocolo asociado con el name de protocolo, indicado por /etc/protocols.

Parámetros

name

El nombre del protocolo.

Valores devueltos

Devuelve el número del protocolo, o false en caso de error.
