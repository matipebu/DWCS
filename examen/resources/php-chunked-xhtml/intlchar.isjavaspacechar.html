IntlChar::isJavaSpaceChar

(PHP 7, PHP 8)

IntlChar::isJavaSpaceChar — Comprobar si un punto de código es un carácter de espacio según Java

Descripción

public static IntlChar::isJavaSpaceChar(mixed $codepoint): bool

Determina si es punto de código especificado es un carácter de espacio según Java.

true para caracterse con la categoría general "Z" (separadores), lo que no incluye códigos de control (p.ej., TAB o Line Feed).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint es un carácter de espacio según Java, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isJavaSpaceChar("A"));
var_dump(IntlChar::isJavaSpaceChar(" "));
var_dump(IntlChar::isJavaSpaceChar("\n"));
var_dump(IntlChar::isJavaSpaceChar("\t"));
var_dump(IntlChar::isJavaSpaceChar("\u{00A0}"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(false)
bool(false)
bool(true)
