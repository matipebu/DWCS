ldap_control_paged_result_response

(PHP 5 >= 5.4.0, PHP 7)

ldap_control_paged_result_response — Recuperar la cookie de paginación LDAP

Descripción

ldap_control_paged_result_response(
    resource $link,
    resource $result,
    string &$cookie = ?,
    int &$estimated = ?
): bool

Recuperar la información de paginación enviada por el servidor.

Parámetros

link

Un identificador de enlace LDAP, devuelto por ldap_connect().

result

cookie

Una estructura opaca enviada por el servidor.

estimated

El número estimado de entradas a recuperar.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
