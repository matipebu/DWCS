pg_get_pid

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

pg_get_pid — Obtiene la ID del proceso del backend

Descripción

pg_get_pid(resource $connection): int

pg_get_pid() Obtiene el backend (proceso de servidor de base de datos) PID. El PID es útil para determinar si un NOTIFY mensaje recibido a través de pg_get_notify() se envia desde otro proceso o no.

Parámetros

connection

Recurso de conexión de base de datos PostgreSQL.

Valores devueltos

La ID del proceso de la base de datos backend.

Ejemplos

Ejemplo #1 PostgreSQL backend PID

<?php
$conn = pg_pconnect("dbname=publisher");
if (!$conn) {
echo "An error occurred.\n";
exit;
}

// Backend process PID. Use PID with pg_get_notify()
$pid = pg_get_pid($conn);
?>
