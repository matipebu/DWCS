phar://

phar:// — Archivo PHP

Descripción

La envoltura de flujo phar:// está disponible desde PHP 5.3.0. Para consultar una descripción detallada, revise envoltura del flujo Phar.

Uso

  • phar://

Opciones

Resumen de la Envoltura
Atributo Permitido
Restringido por allow_url_fopen No
Restringido por allow_url_include No
Permite Lecturas Sí
Permite Escrituras Sí
Permite Añadir contenidos No
Permite Lectura y Escritura Simultanea Sí
Permite usar la función stat() Sí
Permite usar la función unlink() Sí
Permite usar la función rename() Sí
Permite usar la función mkdir() Sí
Permite usar la función rmdir() Sí
