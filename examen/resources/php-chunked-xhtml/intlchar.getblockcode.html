IntlChar::getBlockCode

(PHP 7, PHP 8)

IntlChar::getBlockCode — Obtener el bloque de asignación de Unicode que contiene a un punto de código

Descripción

public static IntlChar::getBlockCode(mixed $codepoint): int

Devuelve el bloque de asignación de Unicode que contiene al carácter.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve el valor del bloque para codepoint. Véanse las constantes IntlChar::BLOCK_CODE_* para los posibles valores devueltos.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::getBlockCode("A") === IntlChar::BLOCK_CODE_BASIC_LATIN);
var_dump(IntlChar::getBlockCode("Φ") === IntlChar::BLOCK_CODE_GREEK);
var_dump(IntlChar::getBlockCode("\u{2603}") === IntlChar::BLOCK_CODE_MISCELLANEOUS_SYMBOLS);
?>

El resultado del ejemplo sería:

bool(true)
bool(true)
bool(true)
