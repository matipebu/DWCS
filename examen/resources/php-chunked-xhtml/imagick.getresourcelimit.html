Imagick::getResourceLimit

(PECL imagick 2, PECL imagick 3)

Imagick::getResourceLimit — Devuelve el límite del recurso especificado

Descripción

Imagick::getResourceLimit(int $type): int

Devuelve el límite del recurso especificado.

Parámetros

type

Consulte la lista de constantes de tipos de recursos.

Valores devueltos

Devuelve el límite del recurso especificado en megabytes.

Errores/Excepciones

Lanza ImagickException en caso de error.
