IntlChar::getBidiPairedBracket

(PHP 7, PHP 8)

IntlChar::getBidiPairedBracket — Obtener la pareja del carácter parentético de un punto de código

Descripción

public static IntlChar::getBidiPairedBracket(mixed $codepoint): mixed

Hace corresponder el carácter especificado con su pareja parentética.

Para Bidi_Paired_Bracket_Type!=None, esto es lo mismo que IntlChar::charMirror(). De lo contrario se devuelve el mismo codepoint.

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve el punto de código de la pareja parentética, o el mismo codepoint si no existe tal correspondencia.

El tipo devuelto será integer a menos que el punto de código se pase como un string UTF-8, en cuyo caso será devuelto un string.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::getBidiPairedBracket(91));
var_dump(IntlChar::getBidiPairedBracket('['));
?>

El resultado del ejemplo sería:

int(93)
string(1) "]"
