SolrInputDocument::__clone

(PECL solr >= 0.9.2)

SolrInputDocument::__clone — Crea una copia de un objeto SolrDocument

Descripción

public SolrInputDocument::__clone(): void

No se debería llamar directamente. Se usa para crear una copia profunda de un objeto SolrInputDocument.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Crea una nueva instancia de SolrInputDocument.
