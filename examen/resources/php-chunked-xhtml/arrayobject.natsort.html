ArrayObject::natsort

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

ArrayObject::natsort — Ordena entradas usando un algoritmo de orden natural

Descripción

public ArrayObject::natsort(): void

Este método implementa un algoritmo que ordena string alfanuméricas en la forma en que un humano seguiría manteniendo la asociación clave/valor. Esto se describe como un "orden natural". Un ejemplo de la diferencia entre este algoritmo y los algoritmos comunes de computadoras para ordenar string. (Usado en el método ArrayObject::asort) puede verse en el siguiente ejemplo.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::natsort()

<?php
$array = array("img12.png", "img10.png", "img2.png", "img1.png");

$arr1 = new ArrayObject($array);
$arr2 = clone $arr1;

$arr1->asort();
echo "Ordenamiento estándar\n";
print_r($arr1);

$arr2->natsort();
echo "\nOrdenamiento de orden naturar\n";
print_r($arr2);
?>

El resultado del ejemplo sería:

Ordenamiento estándar
ArrayObject Object
(
    [3] => img1.png
    [1] => img10.png
    [0] => img12.png
    [2] => img2.png
)

Ordenamiento de orden naturar
ArrayObject Object
(
    [3] => img1.png
    [2] => img2.png
    [1] => img10.png
    [0] => img12.png
)
