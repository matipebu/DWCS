DirectoryIterator::next

(PHP 5, PHP 7, PHP 8)

DirectoryIterator::next — Avanza al siguiente elemento DirectoryIterator

Descripción

public DirectoryIterator::next(): void

Avanzar al siguiente elemento DirectoryIterator.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de DirectoryIterator::next()

List the contents of a directory using a while loop.

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
while($iterator->valid()) {
echo $iterator->getFilename() . "\n";
$iterator->next();
}
?>

El resultado del ejemplo sería algo similar a:

.
..
manzana.jpg
banana.jpg
index.php
pera.jpg
