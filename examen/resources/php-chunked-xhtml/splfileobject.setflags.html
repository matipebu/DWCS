SplFileObject::setFlags

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

SplFileObject::setFlags — Establece flags para el SplFileObject

Descripción

public SplFileObject::setFlags(int $flags): void

Establece las flags a ser usadas por el SplFileObject.

Parámetros

flags

Bitmask de las flags a establecer. Véase las Constantes SplFileObject para una lista de flags disponibles.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de SplFileObject::setFlags()

<?php
$file = new SplFileObject("datos.csv");
$file->setFlags(SplFileObject::READ_CSV);
foreach ($file as $fields) {
var_dump($fields);
}
?>
