curl_multi_strerror

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_multi_strerror — Devuelve un string con la descripción del código de error

Descripción

curl_multi_strerror(int $errornum): string

Devuelve un mensaje de error que describe el código de error CURLM dado.

Parámetros

errornum

Una de las constantes de » códigos de error CURLM .

Valores devueltos

Devuelve un string con la descripción del código de error válido, null si no lo es.

Ejemplos

Ejemplo #1 Ejemplo con curl_multi_strerror()

<?php
// Crea gestores cURL
$ch1 = curl_init("http://example.com"/);
$ch2 = curl_init("http://php.net/");

// Crea un multi gestor cURL
$mh = curl_multi_init();

// Añade los gestores al multi gestor
curl_multi_add_handle($mh, $ch1);
curl_multi_add_handle($mh, $ch2);

// Ejecuta el multi gestor
do {
$status = curl_multi_exec($mh, $active);
// Revisa si se ha producido errores
if($status > 0) {
// Muestra mensaje de error
echo "ERROR!\n " . curl_multi_strerror($status);
}
} while ($status === CURLM_CALL_MULTI_PERFORM || $active);
?>
