openssl_error_string

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

openssl_error_string — Devolver un mensaje de error openSSL

Descripción

openssl_error_string(): string

openssl_error_string() devuelve el último error de la biblioteca openSSL. Los mensajes de error se ponen en cola, por lo que esta función debería ser llamada múltiples veces para reunir toda la información. El último error será la más reciente.

Valores devueltos

Devuelve una cadena de mensaje de error, o false si no hay más mensajes de error a devolver.

Ejemplos

Ejemplo #1 Ejemplo de openssl_error_string()

<?php
// asumimos que se llamó a una función openssl que falló
while ($msg = openssl_error_string())
echo $msg . "<br />\n";
?>
