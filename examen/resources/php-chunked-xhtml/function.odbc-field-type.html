odbc_field_type

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_field_type — Tipo de datos de un campo

Descripción

odbc_field_type(resource $result_id, int $field_number): string

Obtiene el tipo SQL de un campo referenciado mediante número en el identificador de resultado dado.

Parámetros

result_id

El identificador de resultado.

field_number

El número de campo. La numeración de campos comienza en 1.

Valores devueltos

Devuelve el tipo de campo como cadena, o false si se produjo un error.
