Formatos

El manual de PHP se encuentra disponible en diferentes formatos. Estos formatos se pueden dividir en dos grupos: formato accesible online y ficheros descargables.

Nota:

Algunas editoriales han publicado versiones impresas de este manual. No podemos recomendar ninguna de ellas, ya que suelen quedarse obsoletas rápidamente.

El manual puede ser leído online en » sitio web de PHP.net y en numerosos mirrors. Para un mejor funcionamiento, debería utilizar el mirror más cercano. La versión online del manual PHP tiene dos hojas de estilos CSS CSS, una para web y la otra para imprimir.

La ventaja del manual online sobre la mayoría de formatos offline es que contiene los comentarios/notas de usuarios y los » enlaces URL que pueden ser usados para llegar rápidamente hasta las partes deseadas del manual. Y obviamente, la desventaja es que se requiere estár online para ver este manual.

Existen varios formatos offline. El más apropiado para cada usuario depende del sistema operativo que use y de sus preferencias a la hora de leer. La información sobre como el manual es generado en tantos formatos diferentes se puede obtener en la sección de este apéndice titulada 'Cómo generamos los diferentes formatos' sección de su apéndice.

El formato más accesible desde diferentes plataformas es la versión HTML. Este manual se ofrece como un único fichero en HTML o como un paquete que contiene un fichero HTML por cada sección (colección de varios cientos de ficheros). Se proporcionan estas versiones de forma comprimida, por tanto, se require una utilidad para obtener los ficheros desde el archivo comprimido.

Para plataformas Windows, la versión Windows HTML Help Workshop mejora la versión del manual en HTML, al poder usar la aplicación Windows HTML Help Workshop. Esta versión proporciona: búsqueda completa de texto, índice completo y favoritos. Muchos entornos de desarrollo para PHP incorporan esta versión de la documentación para proporcionar un acceso fácil . También hay visores CHM disponibles para Linux. Dale un vistazo a » xCHM o » GnoCHM.

También está disponible la » versión extendida de CHM , la cual se actualiza menos frecuentemente, pero proporciona otras muchas opciones. Solamente funciona con Microsoft Windows debido a la tecnología utilizada para crear estas páginas de ayuda.
