OCICollection::size

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCICollection::size — Devuelve el tamaño de la colección

Descripción

public OCICollection::size(): int|false

Devuelve el tamaño de la colección.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el número de elementos de la colección o false en caso de error.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 La clase OCI-Collection ha sido renombrada a OCICollection para ajustarse a los estándares de nomenclatura de PHP.
