RRDCreator::addDataSource

(PECL rrd >= 0.9.0)

RRDCreator::addDataSource — Añade definición de origen de datos para la base de datos RRD

Descripción

public RRDCreator::addDataSource(string $description): void

RRD puede aceptar la entrada desde varias fuentes de datos (DS), por ejemplo, tráfico entrante y saliente. Este método añade fuente de datos por descripción. Necesita llamar a este método para cada fuente de datos.

Parámetros

description

Definición de origen de datos - DS. Esta tiene el mismo formato como la definición DS en el comando rrd create. Ver página de manual de rrd create para obtener más detalles.

Valores devueltos

No devuelve ningún valor.
