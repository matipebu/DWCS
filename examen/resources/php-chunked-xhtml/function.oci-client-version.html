oci_client_version

(PHP 5 >= 5.3.7, PHP 7, PHP 8, PECL OCI8 >= 1.4.6)

oci_client_version — Devuelve la versión de la biblioteca cliente de Oracle

Descripción

oci_client_version(): string

Devuelve un string que contiene el número de la versión de la biblioteca cliente en C de Oracle a la que está vinculada PHP.

Parámetros

Ninguno

Valores devueltos

Devuelve el número de la versión como un string.

Ejemplos

Ejemplo #1 Ejemplo de oci_client_version()

<?php
echo "Versión cliente: " . oci_client_version(); // Versión cliente: 11.2.0.2
?>

Notas

Nota:

Las bibliotecas de Oracle anteriores a 10gR2 no tienen la funcionalidad subyacente para obtener el número de la versión de la biblioteca cliente. En este caso será devuelto la cadena "Unknown".
