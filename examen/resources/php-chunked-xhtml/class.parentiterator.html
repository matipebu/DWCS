La clase ParentIterator

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Introducción

Esta FilterIterator ampliada permite una iteración recursiva usando RecursiveIteratorIterator, que solamente muestra los elementos que tienen hijos.
