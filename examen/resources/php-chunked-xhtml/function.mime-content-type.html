mime_content_type

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

mime_content_type — Detecta el MIME Content-type para un fichero

Descripción

mime_content_type(string $filename): string

Devuelve el MIME content type para un fichero determinado usando información desde el fichero magic.mime.

Parámetros

filename

Ruta para comprobar fichero.

Valores devueltos

Devuelve el contenido en formato MIME como, text/plain o application/octet-stream, o false en caso de error.

Errores/Excepciones

En caso de fallo, se emite un E_WARNING.

Ejemplos

Ejemplo #1 Ejemplo mime_content_type()

<?php
echo mime_content_type('php.gif') . "\n";
echo mime_content_type('test.php');
?>

El resultado del ejemplo sería:

image/gif
text/plain
