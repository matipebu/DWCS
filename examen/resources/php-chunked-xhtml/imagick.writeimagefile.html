Imagick::writeImageFile

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::writeImageFile — Escribe una imagen a un gestor de archivo

Descripción

public Imagick::writeImageFile(resource $filehandle, string $format = ?): bool

Escribe una secuencia de imágenes a un gestor de archivo abierto. El gestro debe ser abierto con, por ejemplo, fopen. Este método está disponible si Imagick ha sido compilado con la versión 6.3.6 o superior de ImageMagick.

Parámetros

filehandle

El gestor de archivo donde se va a escribir la imagen

Valores devueltos

Devuelve true en caso de éxito.
