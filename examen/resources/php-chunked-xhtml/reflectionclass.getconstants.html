ReflectionClass::getConstants

(PHP 5, PHP 7, PHP 8)

ReflectionClass::getConstants — Obtener constantes

Descripción

public ReflectionClass::getConstants(?int $filter = null): array

Obtiene todas las constantes definidas de una clase, independientemente de su visibilidad.

Parámetros

filter

El filtro opcional, para filtrar las visibilidades constantes deseadas. Se configura utiliando las constantes de ReflectionClassConstant, y por omisión tiene todas las constantes visibles.

Valores devueltos

Un array de constantes. El nombre de la constante en la clave, y en el valor, el valor de la constante.

Historial de cambios

Versión Descripción
8.0.0 Se añadió el parámetro filter.
