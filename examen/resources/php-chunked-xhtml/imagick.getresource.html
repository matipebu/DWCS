Imagick::getResource

(PECL imagick 2, PECL imagick 3)

Imagick::getResource — Devuelve el uso de memoria del recurso especificado

Descripción

Imagick::getResource(int $type): int

Devuelve el uso de memoria del recurso especificado en megabytes.

Parámetros

type

Consulte esta lista de constantes de tipos de recursos.

Valores devueltos

Devuelve el uso de memoria del recurso especificado en megabytes.

Errores/Excepciones

Lanza ImagickException en caso de error.
