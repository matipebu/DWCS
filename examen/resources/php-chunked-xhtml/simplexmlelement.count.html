SimpleXMLElement::count

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SimpleXMLElement::count — Cuenta los hijos de un elemento

Descripción

public SimpleXMLElement::count(): int

Este método cuenta el número de hijos de un elemento.

Valores devueltos

Devuelve el número de elementos de un elemento.

Ejemplos

Ejemplo #1 Cuenta el número de hijos

<?php
$xml = <<<EOF
<people>
<person name="Persona 1">
<child/>
<child/>
<child/>
</person>
<person name="Persona 2">
<child/>
<child/>
<child/>
<child/>
<child/>
</person>
</people>
EOF;

$elem = new SimpleXMLElement($xml);

foreach ($elem as $person) {
printf("%s tiene %d hijos.\n", $person['name'], $person->count());
}
?>

El resultado del ejemplo sería:

Persona 1 tiene 3 hijos.
Persona 2 tiene 5 hijos.
