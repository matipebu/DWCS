Gender\Gender::similarNames

(PECL gender >= 0.9.0)

Gender\Gender::similarNames — Obtener nombres similares

Descripción

public Gender\Gender::similarNames(string $name, integer $country = ?): array

Obtiene nombres similares para el nombre y el país dado.

Parámetros

name

El nombre a comprobar.

country

El Id del país identificado por una constante la clase Gender. Si se omite, se usa ANY_COUNTRY.

Valores devueltos

Devuelve un array con los nombres similares encontrados.
