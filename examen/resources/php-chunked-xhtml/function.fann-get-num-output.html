fann_get_num_output

(PECL fann >= 1.0.0)

fann_get_num_output — Obtener el número de neuronas de salida

Descripción

fann_get_num_output(resource $ann): int

Obtener el número de neuronas de salida.

Parámetros

ann

Un resource de red neuronal.

Valores devueltos

El número de neuronas de salida, o false en caso de error.
