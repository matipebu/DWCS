stream_set_read_buffer

(PHP 5 >= 5.3.3, PHP 7, PHP 8)

stream_set_read_buffer — Establece el búfer de lectura de archivos en el flujo dado

Descripción

stream_set_read_buffer(resource $stream, int $buffer): int

Establece el búfer de lectura. Es equivalente a stream_set_write_buffer(), excepto que es para operaciones de escritura-

Parámetros

stream

El puntero al archivo.

buffer

El número de bytes del buffer. Si buffer es 0 las operaciones de lectura no son volcadas al buffer. Esto asegura que todas las lecturas hechas con fread() son completadas antes de que a otros procesos se les permita leer en ese flujo de entrada.

Valores devueltos

Devuelve 0 en caso de éxito, u otro valor si la petición no se pudo satisfacer.
