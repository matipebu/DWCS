PDO::pgsqlCopyToArray

(PHP 5 >= 5.3.3, PHP 7, PHP 8)

PDO::pgsqlCopyToArray — Copiar datos de una tabla de una base de datos a un array de PHP

Descripción

public PDO::pgsqlCopyToArray(
    string $table_name,
    array $rows,
    string $delimiter = '\t',
    string $null_as = "\\\\N",
    string $fields = ?
): array

Copia los datos de la tabla dada por table a un array empleando delimiter como delimitador de campos y la lista de campos fields

Parámetros

table_name

Una cadena que contiene el nombre de la tabla

delimiter

El delimitador empleado en el array rows

null_as

Cómo interpretar valores nulos

fields

El listado de campos a exportar

Valores devueltos

Devuelve un array de filas o false en caso de error.
