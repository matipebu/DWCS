trader_macdfix

(PECL trader >= 0.2.0)

trader_macdfix — Convergencia/divergencia fija 12/26 de la media móvil

Descripción

trader_macdfix(array $real, int $signalPeriod = ?): array

Parámetros

real

Array de valores reales.

signalPeriod

Suavizado para la línea de señal (nb de periodo). Rango válido desde 1 hasta 100000.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
