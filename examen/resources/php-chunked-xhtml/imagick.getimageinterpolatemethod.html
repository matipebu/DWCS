Imagick::getImageInterpolateMethod

(PECL imagick 2, PECL imagick 3)

Imagick::getImageInterpolateMethod — Devuelve el método de interpolación

Descripción

Imagick::getImageInterpolateMethod(): int

Devuelve el método de interpolación de la imagen especificada. El método es una de las constantes Imagick::INTERPOLATE_*.

Valores devueltos

Devuelve el método de interpolación si se tuvo éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
