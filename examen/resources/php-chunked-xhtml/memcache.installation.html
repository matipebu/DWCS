Instalación

Esta extensión » PECL no se distribuye con PHP. Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » https://pecl.php.net/package/memcache.

Nota:

Es posible desactivar el gestor de sesiones memcache. La opción 'pecl install' pregunta por esta opción (por defecto está activada) de todas formas cuando se compila estáticamente en PHP se puede utilizar la opción de configuración --disable-memcache-session.
