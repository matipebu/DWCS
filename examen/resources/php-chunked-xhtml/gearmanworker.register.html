GearmanWorker::register

(PECL gearman >= 0.6.0)

GearmanWorker::register — Registra una función en el servidor de trabajo

Descripción

public GearmanWorker::register(string $function_name, int $timeout = ?): bool

Registra el nombre de una función en el servidor de trabajo con un tiempo máximo de espera opcional. El tiempo de espera indica cuantos segundos debe esperar el servidor antes de marcar el trabajo como fallo. Indicar un tiempo de espera de cero implica deshabilitar la espera.

Parámetros

function_name

El nombre de una función a registrar en el servidor de trabajo

timeout

Un intervalo de tiempo en segundos

Valores devueltos

Un valor de retorno estándar de Gearman.

Ver también

  • GearmanWorker::unregister() - Elimina el registro del nombre de una función en los servidores de trabajo
  • GearmanWorker::unregisterAll() - Elimina el registro de todas los nombres de funciones en los servidores de trabajo
