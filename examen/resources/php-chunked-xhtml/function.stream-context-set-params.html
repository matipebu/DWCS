stream_context_set_params

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

stream_context_set_params — Establecer parámetros para un flujo/envoltura/contexto

Descripción

stream_context_set_params(resource $stream_or_context, array $params): bool

Establece parámetros en el contexto especificado.

Parámetros

stream_or_context

El flujo o el contexto para aplicar los parámetros.

params

Un array de parámetros para configurar.

Nota:

params debería ser un array asociativo con la estructura: $parámetros['nombre_parametro'] = "valor_parametro";.

Parámetros soportados

Parámetros Propósito
notification Nombre de la función de llamada de retorno definida por el usuario a ser llamada siempre que un flujo provoque una notificación. Sólo se soporta para los stream wrappers http:// y ftp://.
options Array de opciones como en opciones y parámetros de contexto.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
