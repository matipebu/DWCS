Gmagick::getversion

(PECL gmagick >= Unknown)

Gmagick::getversion — Devuelve la versión de la API GraphicsMagick

Descripción

public Gmagick::getversion(): array

Devuelve la versión de la API GraphicsMagick como una cadena y como un número.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la versión de la API GraphicsMagick como una cadena y como un número.

Errores/Excepciones

Arroja un GmagickException en caso de error.
