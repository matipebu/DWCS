gethostbyaddr

(PHP 4, PHP 5, PHP 7, PHP 8)

gethostbyaddr — Obtener el nombre del host de Internet correspondiente a una dirección IP dada

Descripción

gethostbyaddr(string $ip_address): string

Devuelve el nombre del host de Internet especificado por ip_address.

Parámetros

ip_address

La dirección IP del servidor huésped.

Valores devueltos

Devuelve el nombre de host en caso de éxito, la ip_address sin modificar en caso de fallo, o false en caso de que la entrada tenga un formato incorrecto.

Ejemplos

Ejemplo #1 Un ejemplo sencillo de gethostbyaddr()

<?php
$nombre_host = gethostbyaddr($_SERVER['REMOTE_ADDR']);

echo $nombre_host;
?>
