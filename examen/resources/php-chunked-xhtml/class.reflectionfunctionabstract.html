Clase ReflectionFunctionAbstract

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

Introducción

Clase padre de ReflectionFunction. Lea su descripción para conocer más detalles.

Propiedades

name

Nombre de la función. De sólo lectura, lanza una ReflectionException en el intento de escribir.
