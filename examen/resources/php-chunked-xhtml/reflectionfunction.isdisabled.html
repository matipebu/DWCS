ReflectionFunction::isDisabled

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

ReflectionFunction::isDisabled — Comprueba si una función está desactivada

Descripción

public ReflectionFunction::isDisabled(): bool

Comprueba si una función está desactivada, mediante la directiva disable_functions.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

true si está desactivadao, o false en caso contrario.
