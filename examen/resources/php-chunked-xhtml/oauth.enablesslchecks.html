OAuth::enableSSLChecks

(PECL OAuth >= 0.99.5)

OAuth::enableSSLChecks — Activa la verificación SSL

Descripción

public OAuth::enableSSLChecks(): bool

Activa la verificación del certificado y servidor SSL (activado por omisión). Alternativamente, el miembro sslChecks puede ser puesto a un valor distinto a false para desactivar la revisión.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

true

Historial de cambios

Versión Descripción
0.99.8 El miembro sslChecks fue agregado
