ldap_first_attribute

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_first_attribute — Devolver el primer atributo

Descripción

ldap_first_attribute(resource $link_identifier, resource $result_entry_identifier): string

Obtiene el primer atributo de la entrada proporcionada. Los atributos restantes son devueltos al invocar ldap_next_attribute() sucesivamente.

Similar a la lectura de las entradas, los atributos son leídos uno por uno desde una entrada en particular.

Parámetros

link_identifier

Un enlace identificador de LDAP, devuelto por ldap_connect().

result_entry_identifier

ber_identifier

ber_identifier es el identificador hacia la ubicación del puntero de memoria interna. Este se pasa por referencia. El mismo ber_identifier se pasa a ldap_next_attribute(), el cual modifica ese puntero.

Nota:

Este parámetro ya no se utiliza ya que ahora es manejado automáticamente por PHP. Por compatibilidad con versiones anteriores PHP no lanzará un error si se pasa este parámetro.

Valores devueltos

Devuelve el primer atributo de la entrada en caso de éxito y false en caso de error.

Historial de cambios

Versión Descripción
5.2.4 ber_identifier ha sido retirado. Este ahora es manejado automáticamente por PHP.
