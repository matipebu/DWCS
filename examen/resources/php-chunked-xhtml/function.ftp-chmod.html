ftp_chmod

(PHP 5, PHP 7, PHP 8)

ftp_chmod — Establecer permisos en un archivo via FTP

Descripción

ftp_chmod(resource $ftp_stream, int $mode, string $filename): int

Establece los permisos en el archivo remoto especificado en mode.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

mode

Los nuevos permisos, dados como un valor octal.

filename

El archivo remoto.

Valores devueltos

Devuelve los nuevos permisos del archivo en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de ftp_chmod()

<?php
$file = 'public_html/index.php';

// configurar conexión básica
$conn_id = ftp_connect($ftp_server);

// iniciar sesión con nombre de usuario y contraseña
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// intentar cambiar el chmod del archivo $file a 644
if (ftp_chmod($conn_id, 0644, $file) !== false) {
echo "El chmod del archivo $file fué satisfactoriamente cambiado a 644\n";
} else {
echo "No se pudo cambiar el chmod del archivo $file\n";
}

// cerrar la conexión ftp
ftp_close($conn_id);
?>
