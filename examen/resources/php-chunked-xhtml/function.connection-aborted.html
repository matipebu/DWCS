connection_aborted

(PHP 4, PHP 5, PHP 7, PHP 8)

connection_aborted — Verifica si el cliente se desconectó

Descripción

connection_aborted(): int

Verifica si el cliente se ha desconectado.

Valores devueltos

Devuelve 1 si el cliente se desconectó, 0 de lo contrario.
