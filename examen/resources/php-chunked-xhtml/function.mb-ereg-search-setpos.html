mb_ereg_search_setpos

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

mb_ereg_search_setpos — Establece el punto de inicio de la siguiente comparación de una expresión regular

Descripción

mb_ereg_search_setpos(int $position): bool

mb_ereg_search_setpos() establece el punto de inicio de una comparación con mb_ereg_search().

Parámetros

position

La posición a establecer.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Notas

Nota:

La codificación interna o la codificación especificada por mb_regex_encoding() será usada en esta función.
