Constantes predefinidas

Constantes Memcache
Nombre Descripción
MEMCACHE_COMPRESSED (integer) Sirve para activar compresión de datos al vuelo con Memcache::set(), Memcache::add() y Memcache::replace().
MEMCACHE_HAVE_SESSION (integer) 1 si el gestor de sesiones de Memcache está disponible, 0 si no.
MEMCACHE_USER1 (integer) Sirve para activar la bandera de aplicación definida por el usuario con Memcache::set(), Memcache::add() y Memcache::replace().
MEMCACHE_USER2 (integer) Sirve para activar la bandera de aplicación definida por el usuario con Memcache::set(), Memcache::add() y Memcache::replace().
MEMCACHE_USER3 (integer) Sirve para activar la bandera de aplicación definida por el usuario con Memcache::set(), Memcache::add() y Memcache::replace().
MEMCACHE_USER4 (integer) Sirve para activar la bandera de aplicación definida por el usuario con Memcache::set(), Memcache::add() y Memcache::replace().
