iptcembed

(PHP 4, PHP 5, PHP 7, PHP 8)

iptcembed — Incluir información IPTC binaria en una imagen JPEG

Descripción

iptcembed(string $iptcdata, string $jpeg_file_name, int $spool = 0): mixed

Incluye información IPTC binaria en una imagen JPEG.

Parámetros

iptcdata

La información que va a ser escrita.

jpeg_file_name

Ruta a la imagen JPEG.

spool

Bandera de spool. Si la bandera de spool es menor que 2, el JPEG será devuelto como una cadena. De lo contrario el JPEG será enviado a STDOUT.

Valores devueltos

Si spool es menor que 2, se devolverá el JPEG, o false en caso de error. De lo contrario devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Incluir información IPTC a un JPEG

<?php

// Función iptc_make_tag() por Thies C. Arntzen
function iptc_make_tag($rec, $data, $value)
{
$length = strlen($value);
$retval = chr(0x1C) . chr($rec) . chr($data);

if($length < 0x8000)
{
$retval .= chr($length >> 8) . chr($length & 0xFF);
}
else
{
$retval .= chr(0x80) .
chr(0x04) .
chr(($length >> 24) & 0xFF) .
chr(($length >> 16) & 0xFF) .
chr(($length >> 8) & 0xFF) .
chr($length & 0xFF);
}

return $retval . $value;
}

// Ruta al fichero jpeg
$ruta = './phplogo.jpg';

// Establecer las etiquetas IPTC
$iptc = array(
'2#120' => 'Imagen de prueba',
'2#116' => 'Copyright 2008-2009, The PHP Group'
);

// Convertir las etiquetas IPTC en código binario
$data = '';

foreach($iptc as $etiqueta => $cadena)
{
$etiqueta = substr($etiqueta, 2);
$data .= iptc_make_tag(2, $etiqueta, $cadena);
}

// Incluir la información IPTC
$contenido = iptcembed($data, $ruta);

// Escribir la nueva información de la imagen en un fichero.
$fp = fopen($ruta, "wb");
fwrite($fp, $contenido);
fclose($fp);
?>

Notas

Nota:

Esta función no requiere la librería GD image.
