Gmagick::setimageindex

(PECL gmagick >= Unknown)

Gmagick::setimageindex — Establece la posición del iterador en la lista de imágenes especificada con el parámetro index

Descripción

public Gmagick::setimageindex(int $index): Gmagick

Establece la posición del iterador en la lista de imágenes especificada con el parámetro index.

Parámetros

index

El número de escena.

Valores devueltos

El objeto Gmagick si se tuvo éxito.

Errores/Excepciones

Arroja un GmagickException en caso de error.
