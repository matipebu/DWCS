Imagick::getGravity

(PECL imagick 2, PECL imagick 3)

Imagick::getGravity — Obtiene la gravedad

Descripción

Imagick::getGravity(): int

Obtiene la propiedad gravedad global para el objeto Imagick. Este método está disponible si Imagick ha sido compilado con la versión 6.4.0 o superior de ImageMagick.

Valores devueltos

Devuelve la propiedad gravedad. Consulte esta lista de constantes de gravedad.
