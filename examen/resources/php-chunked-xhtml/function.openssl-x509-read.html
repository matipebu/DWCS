openssl_x509_read

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

openssl_x509_read — Analiza un certificado X.509 y devuelve un identificador de recurso para él

Descripción

openssl_x509_read(mixed $x509certdata): resource

openssl_x509_read() analiza un certificado proporcionado por x509certdata y devuelve un identificador de recurso para él.

Parámetros

x509certdata

Valores devueltos

Devuelve un identificador de recurso si se tuvo éxito o false en caso de error.
