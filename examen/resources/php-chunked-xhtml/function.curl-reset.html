curl_reset

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_reset — Reinicializa todas las opciones de un gestor de sesión libcurl

Descripción

curl_reset(resource $ch): void

Esta función reinicializa a sus valores por defecto, todas las opciones de un gestor cURL dado.

Parámetros

ch

El recurso cURL devuelto por curl_init().

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo con curl_reset()

<?php
// Crear un gestor cURL
$ch = curl_init();

// Definir la opción CURLOPT_USERAGENT
curl_setopt($ch, CURLOPT_USERAGENT, "My test user-agent");

// Reinicializar todas las opciones ya configuradas
curl_reset($ch);

// Enviar la solicitud HTTP
curl_setopt($ch, CURLOPT_URL, 'http://example.com/');
curl_exec($ch); // el gestor de usuario definido previamente no será enviado, puesto que ha sido reseteado por curl_reset

// Cerrar el gestor
curl_close($ch);
?>

Notas

Nota:

curl_reset() también reinicia la URL dada como el parámetro de la función curl_init() .
