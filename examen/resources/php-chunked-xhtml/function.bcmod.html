bcmod

(PHP 4, PHP 5, PHP 7, PHP 8)

bcmod — Obtiene el módulo de un número de precisión arbitraria

Descripción

bcmod(string $left_operand, string $modulus): string

Obtiene el módulo del left_operand usando modulus.

Parámetros

left_operand

El left operand, como una cadena.

modulus

El modulus, como una cadena.

Valores devueltos

Devuelve el módulo como una cadena, o null si modulus es 0.

Ejemplos

Ejemplo #1 Ejemplo bcmod()

<?php
echo bcmod('4', '2'); // 0
echo bcmod('2', '4'); // 2
?>
