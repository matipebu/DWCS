end — Establece el puntero interno de un array a su último elemento

Descripción

end(array &$array): mixed

end() avanza el puntero interno del array hasta su último elemento y devuelve su valor.

Parámetros

array

El array. Este array es pasado por referencia porque es modificado por la función. Esto significa que se debe pasar una variable real y no una función que devuelve un array porque solamente variables puede ser pasadas por referencia.

Valores devueltos

Devuelve el valor del último elemento o false si el array está vacío.

Ejemplos

Ejemplo #1 Ejemplo de end()

<?php

$fruits = array('manzana', 'plátano', 'arándano');
echo end($fruits); // arándano

?>
