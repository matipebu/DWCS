ldap_next_attribute

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_next_attribute — Obtener el siguiente atributo en un resultado

Descripción

ldap_next_attribute(resource $link_identifier, resource $result_entry_identifier): string

Devuelve los atributos en una entrada. La primer llamada a la función ldap_next_attribute() es hecha con el parámetro result_entry_identifier que es devuelto por la función ldap_first_attribute().

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelto por ldap_connect().

result_entry_identifier

ber_identifier

El estado interno del puntero es mantenido por este parámetro.

Nota:

Este parámetro ya no se utiliza debido a que ahora es manipulado automáticamente por PHP. Por compatibilidades previas, PHP no lanzará un error si se pasa este parámetro.

Valores devueltos

Devuelve el siguiente atributo en una entrada en caso de éxito y false en caso de error.

Historial de cambios

Versión Descripción
5.2.4 El parámetro ber_identifier fue removido. Este se maneja ahora automáticamente por PHP.
