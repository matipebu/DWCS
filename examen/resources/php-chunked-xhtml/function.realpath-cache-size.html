realpath_cache_size

(PHP 5 >= 5.3.2, PHP 7, PHP 8)

realpath_cache_size — Obtiene el tamaño de la caché de la ruta real

Descripción

realpath_cache_size(): int

Obtiene la cantidad de memoria usada por la caché de la ruta real.

Valores devueltos

Devuelve la cantidad de memoria que la caché de la ruta real está usando.

Ejemplos

Ejemplo #1 Ejemplo de realpath_cache_size()

<?php
var_dump(realpath_cache_size());
?>

El resultado del ejemplo sería algo similar a:

int(412)
