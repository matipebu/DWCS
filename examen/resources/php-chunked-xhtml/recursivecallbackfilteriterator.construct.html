RecursiveCallbackFilterIterator::__construct

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

RecursiveCallbackFilterIterator::__construct — Crear un RecursiveCallbackFilterIterator desde un RecursiveIterator

Descripción

public RecursiveCallbackFilterIterator::__construct(RecursiveIterator $iterator, string $callback)

Crea un iterador filtrado desde un RecursiveIterator usando el parámetro callback para determinar si los elementos son aceptados o rechazados.

Parámetros

iterator

El iterador recursivo a ser filtrado.

callback

la llamada de retorno, debe retornar true para aceptar el elemento actual o en caso contrario false. Véase los Ejemplos.

Puede ser cualquier valor válido de callable.

Valores devueltos

No devuelve ningún valor.
