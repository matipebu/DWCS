PDOStatement::setAttribute

(PHP 5 >= 5.1.0, PHP 7, PHP 8, PECL pdo >= 0.2.0)

PDOStatement::setAttribute — Establece un atributo de sentencia

Descripción

public PDOStatement::setAttribute(int $attribute, mixed $value): bool

Establece un atributo de una sentencia. Actualmente, no se establecen atributos genéricos, sino solamente específicos del controlador:

  • PDO::ATTR_CURSOR_NAME (específico de Firebird y ODBC): Establece el nombre del cursor para UPDATE ... WHERE CURRENT OF.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
