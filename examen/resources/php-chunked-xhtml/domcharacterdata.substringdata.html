DOMCharacterData::substringData

(PHP 5, PHP 7, PHP 8)

DOMCharacterData::substringData — Extrae una serie de datos del nodo

Descripción

public DOMCharacterData::substringData(int $offset, int $count): string|false

Devuelve el substring especificado.

Parámetros

offset

Posición de inicio del substring a extraer.

count

El número de carácteres a extraer.

Valores devueltos

El substring especificado. Si la suma de offset y count excede la longitud, entonces todas las unidades de 16-bit hasta el final de la cadena son devueltas.

Errores/Excepciones

DOM_INDEX_SIZE_ERR

Lanzado si offset es negativo o mayor que el numero de unidades de 16-bit en la data, o si count es negativo.
