fann_get_activation_function

(PECL fann >= 1.0.0)

fann_get_activation_function — Devuelve la función de activación

Descripción

fann_get_activation_function(resource $ann, int $layer, int $neuron): int

Obtiene la función de activación para la neurona número the neuron en la capa número layer, contando la capa de entrada como capa 0.

No es posible obtener funciones de activación para neuronas de la capa de entrada.

El valor devuelto es una de las constantes de funciones de activación.

Parámetros

ann

Un resource de red neuronal.

layer

El número de capa.

neuron

El número de neurona.

Valores devueltos

Una constante de funciones de aprendizaje o -1 si la neurona no está definida en la red neuronal, o false en caso de error.
