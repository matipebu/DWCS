inotify_add_watch

(PECL inotify >= 0.1.2)

inotify_add_watch — Añade un seguimiento a una instancia inotify inicializada

Descripción

inotify_add_watch(resource $inotify_instance, string $pathname, int $mask): int

inotify_add_watch() añade un nuevo seguimiento o modifica un seguimiento existente para el fichero o directorio especificado en pathname.

Usando inotify_add_watch() sobre un objeto observado reemplaza el seguimiento existente. Utilizando la constante IN_MASK_ADD añade eventos al seguimiento existente.

Parámetros

inotify_instance

Recurso devuelto por inotify_init()

pathname

Fichero o directorio a observar

mask

Eventos para observar. Ver Constantes predefinidas.

Valores devueltos

The return value is a unique (inotify instance wide) watch descriptor. El valor devuelto es un descriptor de seguimiento único.
