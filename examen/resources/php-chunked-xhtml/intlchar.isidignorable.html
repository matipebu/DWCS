IntlChar::isIDIgnorable

(PHP 7, PHP 8)

IntlChar::isIDIgnorable — Comprobar si un punto de código es un carácter ignorable

Descripción

public static IntlChar::isIDIgnorable(mixed $codepoint): bool

Determina si el carácter especificado debería ser considerado como carácter ignorable en un identificador.

true para caracteres con categoría general "Cf" (controles de formato) así como con controles ISO que no pertenecen al espacio en blanco (U+0000..U+0008, U+000E..U+001B, U+007F..U+009F).

Nota:

Observe que Unicode recomienda simplemente ignorar Cf (controles de formato).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint es ignorable en identificadores, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isIDIgnorable("A"));
var_dump(IntlChar::isIDIgnorable(" "));
var_dump(IntlChar::isIDIgnorable("\u{007F}"));
?>

El resultado del ejemplo sería:

bool(false)
bool(false)
bool(true)
