curl_share_strerror

(PHP 7 >= 7.1.0, PHP 8)

curl_share_strerror — Devuelve string con la descripción del codigo de error dado

Descripción

curl_share_strerror(int $errornum): string

Devuelve un texto describiendo el mensaje de error del código de error dado.

Parámetros

errornum

Una de las constantes de los » códigos de error cURL.

Valores devueltos

Devuelve la descripción del error o null si el código de error es inválido.
