ReflectionClass::__toString

(PHP 5, PHP 7, PHP 8)

ReflectionClass::__toString — Devuelve la representación en forma de string del objeto ReflectionClass

Descripción

public ReflectionClass::__toString(): string

Devuelve la representación en forma de string del objeto ReflectionClass.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Representación en forma de string de esta instancia de ReflectionClass.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::__toString()

<?php
$reflectionClass = new ReflectionClass('Exception');
echo $reflectionClass->__toString();
?>
