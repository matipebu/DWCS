Constantes predefinidas

Estas constantes están definidas por esta extensión y estarán disponibles sólo cuando la extensión haya sido compilada con PHP, o bien sea cargada dinámicamente en ejecución.

SQLITE3_ASSOC (integer)
Especifica que el método Sqlite3Result::fetchArray() deberá devolver un array indexado por el nombre de columna como el devuelto en el correspondiente conjunto de resultados.
SQLITE3_NUM (integer)
Especifica que el método Sqlite3Result::fetchArray() deberá devolver un array indexado por el número de columna como el devuelto en el correspondiente conjunto de resultados, comenzando por la columna 0.
SQLITE3_BOTH (integer)
Especifica que el método Sqlite3Result::fetchArray() deberá devolver un array indexado tanto por el nombre como por el número de columna como el devuelto en el correspondiente conjunto de resultados, comenzando por la columna 0.
SQLITE3_INTEGER (integer)
Representa la clase de almacenamiento INTEGER de SQLite3.
SQLITE3_FLOAT (integer)
Representa la clase de almacenamiento REAL (FLOAT) de SQLite3.
SQLITE3_TEXT (integer)
Representa la clase de almacenamiento TEXT de SQLite3.
SQLITE3_BLOB (integer)
Representa la clase de almacenamiento BLOB de SQLite3.
SQLITE3_NULL (integer)
Representa la clase de almacenamiento NULL de SQLite3.
SQLITE3_OPEN_READONLY (integer)
Especifica que la base de datos SQLite3 será abierta para sólo lectura.
SQLITE3_OPEN_READWRITE (integer)
Especifica que la base de datos SQLite3 será abierta para lectura y escritura.
SQLITE3_OPEN_CREATE (integer)
Especifica que la base de datos SQLite3 será creada si aún no existe.
