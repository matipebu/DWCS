IntlChar::isupper

(PHP 7, PHP 8)

IntlChar::isupper — Comprobar si un punto de código tiene la categoría general "Lu" (letra mayúscula)

Descripción

public static IntlChar::isupper(mixed $codepoint): bool

Determina si el punto de código especificado tiene la categoría general "Lu" (letra mayúscula).

Nota:

Este método erra con algunos caracteres que tambíen son mayúscula pero tienen un valor diferente de categoría general. Para incluirlos, use IntlChar::isUUppercase().

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint es una letra mayúscula de Lu, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isupper("A"));
var_dump(IntlChar::isupper("a"));
var_dump(IntlChar::isupper("Φ"));
var_dump(IntlChar::isupper("φ"));
var_dump(IntlChar::isupper("1"));
?>

El resultado del ejemplo sería:

bool(true)
bool(false)
bool(true)
bool(false)
bool(false)
