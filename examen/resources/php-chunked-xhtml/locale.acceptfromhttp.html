Locale::acceptFromHttp

locale_accept_from_http

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Locale::acceptFromHttp -- locale_accept_from_http — Intentar encontrar la mejor configuración regional basada en la cabecera "Accept-Language" de HTTP

Descripción

Estilo orientado a objetos

public static Locale::acceptFromHttp(string $header): string

Estilo por procedimientos

locale_accept_from_http(string $header): string

Intenta encontrar la configuración regional que pueda satisfacer la lista de lenguajes que es pedida por la cabecera "Accept-Language" de HTTP.

Parámetros

header

La cadena que contiene la cabecera "Accept-Language" de acuerdo al formato de RFC 2616.

Valores devueltos

El identificador regional correspondiente.

Ejemplos

Ejemplo #1 Ejemplo de locale_accept_from_http()

<?php
$locale = locale_accept_from_http($_SERVER['HTTP_ACCEPT_LANGUAGE']);
echo $locale;
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$locale = Locale::acceptFromHttp($_SERVER['HTTP_ACCEPT_LANGUAGE']);
echo $locale;
?>

El resultado del ejemplo sería:

en_US
