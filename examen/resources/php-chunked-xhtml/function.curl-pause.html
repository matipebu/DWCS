curl_pause

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_pause — Pausa y reanuda una conexión

Descripción

curl_pause(resource $ch, int $bitmask): int

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

ch

El recurso cURL devuelto por curl_init().

bitmask

Una de las constantes CURLPAUSE_* .

Valores devueltos

Devuelve un código de error, (CURLE_OK si no hay error).
