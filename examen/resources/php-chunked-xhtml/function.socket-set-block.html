socket_set_block

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

socket_set_block — Establece el modo de bloqueo de un recurso socket

Descripción

socket_set_block(resource $socket): bool

La función socket_set_block() elimina la bandera O_NONBLOCK del socket especificado por el parámetro socket.

Cuando una operación (p.ej. recibir, enviar, conectar, aceptar, ...) se lleva a cabo sobre un socket de bloqueo, el script pausará su ejecución hasta que reciba una señal o pueda realizar la operación.

Parámetros

socket

Un recurso socket válido creado con socket_create() o socket_accept().

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Un ejemplo de socket_set_block()

<?php
$socket = socket_create_listen(1223);
socket_set_block($socket);

socket_accept($socket);
?>

Este ejemplo crea un socket de escucha en todas las interfaces en el puerto 1223 y establece el socket al modo O_BLOCK. socket_accept() se colgará hasta que haya una conexión que aceptar.
