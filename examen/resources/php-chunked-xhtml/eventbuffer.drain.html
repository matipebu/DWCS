EventBuffer::drain

(PECL event >= 1.2.6-beta)

EventBuffer::drain — Elimina un número específico de bytes del principio del buffer sin copiarlo a ningún lugar

Descripción

public EventBuffer::drain( int $len ): bool

Funciona del mismo modo que EventBuffer::read() , excepto que no copia los datos: únicamente los elimina del principio del buffer.

Parámetros

len

El número de bytes a eliminar del buffer.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

  • EventBuffer::appendFrom() - Mueve el número de bytes especificados desde un buffer fuente, al final del buffer actual
