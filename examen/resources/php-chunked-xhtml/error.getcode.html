Error::getCode

(PHP 7, PHP 8)

Error::getCode — Obtener el código de error

Descripción

final public Error::getCode(): int

Devuelve el código de error.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el código de error como un int

Ejemplos

Ejemplo #1 Ejemplo de Error::getCode()

<?php
try {
throw new Error("Un mensaje de error", 30);
} catch(Error $e) {
echo "El código del Error es: " . $e->getCode();
}
?>

El resultado del ejemplo sería algo similar a:

El código del Error es: 30
