Collator::getAttribute

collator_get_attribute

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Collator::getAttribute -- collator_get_attribute — Obtener el valor de un atributo de cotejo

Descripción

Estilo orientado a objetos

public Collator::getAttribute(int $attr): int

Estilo por procedimientos

collator_get_attribute(Collator $coll, int $attr): int

Obtiene el valor de un atributo de cotejo de tipo integer.

Parámetros

coll

Collator object.

attr

El atributo del que obtener el valor.

Valores devueltos

El valor del atributo, o false de tipo boolean en case de error.

Ejemplos

Ejemplo #1 Ejemplo de collator_get_attribute()

<?php
$coll = collator_create( 'en_CA' );
$val = collator_get_attribute( $coll, Collator::NUMERIC_COLLATION );
if( $val === false )
{
// Manejar el error.
}
?>
