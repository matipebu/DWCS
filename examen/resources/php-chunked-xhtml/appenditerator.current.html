AppendIterator::current — Obtiene el valor actual

Descripción

public AppendIterator::current(): mixed

Obtiene el valor actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Si el valor actual es válido o null en caso contrario.
