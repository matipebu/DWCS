cubrid_lob_size

(PECL CUBRID >= 8.3.1)

cubrid_lob_size — Obtener el tamaño de datos BLOB/CLOB

Descripción

cubrid_lob_size(resource $lob_identifier): string

cubrid_lob_size() se usa para obtener el tamaño de datos BLOB/CLOB.

Parámetros

lob_identifier

Identificador LOB.

Valores devueltos

Un string representando el tamaño de datos LOB, cuando el proceso tiene éxito.

false, cuando el proceso no tiene éxito.

Historial de cambios

Versión Descripción
8.4.0 Cambia el tipo de valor de retorno de int a string.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_lob_size()

<?php
$lobs = cubrid_lob_get($con, "SELECT doc_content FROM doc WHERE doc_id=5");
echo "Tamño del documento:".cubrid_lob_size($lobs[0]);
cubrid_lob_export($conn, $lobs[0], "doc_5.txt");
cubrid_lob_close($lobs);
?>
