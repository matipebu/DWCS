ldap_delete

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_delete — Eliminar una entrada de un directorio

Descripción

ldap_delete(resource $link_identifier, string $dn): bool

Elimina una entrada en particular en un directorio de LDAP.

Parámetros

link_identifier

Un enlace identificador de LDAP, devuelto por ldap_connect().

dn

El nombre distinguido de una entidad de LDAP.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
