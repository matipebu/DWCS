Constantes predefinidas

Desde PHP 4.3.0 es posible identificar en tiempo de ejecución cuál implementación de iconv es adoptada por esta extensión.

Constances de iconv
Nombre Tipo Descripción
ICONV_IMPL string Nombre de la implementación
ICONV_VERSION string Versión de la implementación

Nota:

Está completamente desaconsejado escribir scripts cuya implementación dependa de estas constantes.

Desde PHP 5.0.0 se dispone de las siguientes constantes:

Constantes de iconv disponibles desde PHP 5.0.0
Nombre Tipo Descripción
ICONV_MIME_DECODE_STRICT integer Máscara de bits usada para iconv_mime_decode()
ICONV_MIME_DECODE_CONTINUE_ON_ERROR integer Máscara de bits usada para iconv_mime_decode()
