curl_strerror

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_strerror — Devuelve un string que describe el código de error dado

Descripción

curl_strerror(int $errornum): string

Devuelve un mensaje de texto describiendo el código de error dado.

Parámetros

errornum

Una de las constantes de » código de error cURL .

Valores devueltos

Devuelve la descripción de error o null para códigos de error inválidos.

Ejemplos

Ejemplo #1 Ejemplo de curl_errno()

<?php
// Crea un gestor curl con un protocolo con nombre erróneo en una URL
$ch = curl_init("htp://example.com/");

// Envía solicitud
curl_exec($ch);

// Valida si se ha producido errores y muestra el mensaje de error
if($errno = curl_errno($ch)) {
$error_message = curl_strerror($errno);
echo "cURL error ({$errno}):\n {$error_message}";
}

// Cierra el gestor
curl_close($ch);
?>

El resultado del ejemplo sería:

cURL error (1):
 Unsupported protocol
