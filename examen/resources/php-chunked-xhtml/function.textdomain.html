textdomain

(PHP 4, PHP 5, PHP 7, PHP 8)

textdomain — Establece el dominio actual

Descripción

textdomain(string $text_domain): string

Esta función establece el dominio en el que se realizarán las búsquedas llamadas por gettext(), normalemente por el nombre de la aplicación.

Parámetros

text_domain

El nuevo mensaje del dominio, o null para obtener el valor actual y no cambiarlo

Valores devueltos

Si es correcto, esta función devuelve el dominio del mensaje actual, después de cambiarlo.
