chunk_split

(PHP 4, PHP 5, PHP 7, PHP 8)

chunk_split — Divide una cadena en trozos más pequeños

Descripción

chunk_split(string $body, int $chunklen = 76, string $end = "\r\n"): string

Puede ser utilizado para dividir una cadena en trozos más pequeños, lo que es útil, por ejemplo, para convertir la salida de la función base64_encode() y hacer que coincida con la semántica del RFC 2045. Inserta la cadena end cada chunklen caracteres.

Parámetros

body

La cadena que será fragmentada.

chunklen

La longitud del trozo.

end

La secuencia de final de línea.

Valores devueltos

Devuelve la cadena fragmentada.

Ejemplos

Ejemplo #1 Ejemplo de chunk_split()

<?php
// formatea $datos usando la semántica del RFC 2045
$nueva_cadena = chunk_split(base64_encode($datos));
?>
