RecursiveIteratorIterator::beginChildren

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

RecursiveIteratorIterator::beginChildren — Inicio del hijo

Descripción

public RecursiveIteratorIterator::beginChildren(): void

Se llama después de llamar a RecursiveIteratorIterator::getChildren(), y está asociada con RecursiveIteratorIterator::rewind().

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
