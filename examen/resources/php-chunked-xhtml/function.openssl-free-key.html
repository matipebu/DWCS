openssl_free_key

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

openssl_free_key — Liberar el recurso de clave

Descripción

openssl_free_key(resource $key_identifier): void

openssl_free_key() libera la clave asociada con el indetificador de clave key_identifier especificado de la memoria.

Parámetros

key_identifier

Valores devueltos

No devuelve ningún valor.
