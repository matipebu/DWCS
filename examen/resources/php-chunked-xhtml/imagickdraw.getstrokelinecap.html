ImagickDraw::getStrokeLineCap

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getStrokeLineCap — Devuelve la forma que se va a usar al final de los subtrazados abiertos cuando están contorneados

Descripción

ImagickDraw::getStrokeLineCap(): int
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la forma que se va a usar al final de los subtrazados abiertos cuando están contorneados.

Valores devueltos

Devuelve una de las constantes LINECAP_ o 0 si el cierre de línea de contorno no está establecido.
