gmp_testbit

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

gmp_testbit — Prueba si un bit es establecido

Descripción

gmp_testbit(GMP $a, int $index): bool

Prueba si un bit es establecido.

Parámetros

a

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

index

El bit a probar

Valores devueltos

Devuelve true si el bit se establece en el recurso $a, de otro modo devuelve false.

Errores/Excepciones

Se emite un error de nivel E_WARNING cuando index es menor que cero, y se devuelve false.

Ejemplos

Ejemplo #1 Ejemplo degmp_testbit()

<?php
$n = gmp_init("1000000");
var_dump(gmp_testbit($n, 1));
gmp_setbit($n, 1);
var_dump(gmp_testbit($n, 1));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
