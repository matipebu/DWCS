OCILob::getBuffering

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::getBuffering — Devuelve el estado actual del uso del búfer para un objeto grande

Descripción

public OCILob::getBuffering(): bool

Indica si el almacenamiento en búfer para objetos grandes está activado o no.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve false si el almacenamiento en búfer de objetos grandes está desactivado, y true si se utiliza dicho almacenamiento.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 La clase OCI-Lob ha sido renombrada a OCILob para ajustarse a los estándares de nomenclatura de PHP.
