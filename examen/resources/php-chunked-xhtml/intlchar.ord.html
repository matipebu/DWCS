IntlChar::ord

(PHP 7, PHP 8)

IntlChar::ord — Devolver el valor del punto de código de Unicode de un carácter

Descripción

public static IntlChar::ord(mixed $character): int

Devuelve el valor del punto de código de Unicode del carácter dado.

Esta función complementa a IntlChar::chr().

Parámetros

character

Un carácter Unicode.

Valores devueltos

Devuelve el valor del punto de código de Unicode, como integer.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::ord("A"));
var_dump(IntlChar::ord(" "));
var_dump(IntlChar::ord("\u{2603}"));
?>

El resultado del ejemplo sería:

int(65)
int(32)
int(9731)
