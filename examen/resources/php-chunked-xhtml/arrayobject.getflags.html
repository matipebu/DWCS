ArrayObject::getFlags

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

ArrayObject::getFlags — Obtiene las flags de comportamiento

Descripción

public ArrayObject::getFlags(): int

Obtener las flags de comportamiento de la clase ArrayObject. Véase el método ArrayObject::setFlags para obtener una lista de las flags disponibles.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve las flags de comportamiento de la clase ArrayObject.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::getFlags()

<?php
// Array de frutas
$fruits = array("limones" => 1, "naranjas" => 4, "bananas" => 5, "manzanas" => 10);

$fruitsArrayObject = new ArrayObject($fruits);

// Obtiene las flags actuales.
$flags = $fruitsArrayObject->getFlags();
var_dump($flags);

// Establece nuevas flags.
$fruitsArrayObject->setFlags(ArrayObject::ARRAY_AS_PROPS);

// Obtener las nuevas flags.
$flags = $fruitsArrayObject->getFlags();
var_dump($flags);
?>

El resultado del ejemplo sería:

int(0)
int(2)
