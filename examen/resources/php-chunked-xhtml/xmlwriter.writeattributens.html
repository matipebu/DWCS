XMLWriter::writeAttributeNs

xmlwriter_write_attribute_ns

(PHP 5 >= 5.1.2, PHP 7, PHP 8, PECL xmlwriter >= 0.1.0)

XMLWriter::writeAttributeNs -- xmlwriter_write_attribute_ns — Write full namespaced attribute

Descripción

Estilo orientado a objetos

public XMLWriter::writeAttributeNs(
    ?string $prefix,
    string $name,
    ?string $namespace,
    string $value
): bool

Estilo por procedimientos

xmlwriter_write_attribute_ns(
    XMLWriter $writer,
    ?string $prefix,
    string $name,
    ?string $namespace,
    string $value
): bool

Escribe un atributo de espacio de nombres completo.

Parámetros

xmlwriter

Sólo para llamadas por procedimientos. El resource XMLWriter que está siendo modificado. Este recurso proviene de una llamada a xmlwriter_open_uri() o xmlwriter_open_memory().

prefix

El prefijo del namespace. If prefix is null, the namespace will be omitted.

name

El nombre del atributo.

namespace

El namespace de URI. If namespace is null, the namespace declaration will be omitted.

value

El valor del atributo.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Historial de cambios

Versión Descripción
8.0.0 writer expects an XMLWriter instance now; previously, a resource was expected.
