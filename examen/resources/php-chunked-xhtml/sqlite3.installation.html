Instalación

La extensión SQLite3 está activada por defecto desde PHP 5.3.0. Se puede desactivar usando --without-sqlite3 en tiempo de compilación.

Los usuarios de Windows deben activar php_sqlite3.dll para poder usar esta extensión. Esta DLL se incluye con las distribuciones de PHP para Windows desde la versión 5.3.0.

Nota:

Durante un corto período de tiempo esta extensión era PECL pero esta versión solo es recomendada para uso experimental.
