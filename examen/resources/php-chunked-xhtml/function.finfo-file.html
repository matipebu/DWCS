finfo_file

finfo::file

(PHP >= 5.3.0, PHP 7, PHP 8, PECL fileinfo >= 0.1.0)

finfo_file -- finfo::file — Devuelve información sobre un fichero

Descripción

Estilo por procedimientos

finfo_file(
    resource $finfo,
    string $file_name = null,
    int $options = FILEINFO_NONE,
    resource $context = null
): string

Estilo orientado a objetos

public finfo::file(string $file_name = null, int $options = FILEINFO_NONE, resource $context = null): string

Esta función se usa para obtener información de un fichero.

Parámetros

finfo

Devuelve recurso Fileinfo por finfo_open().

file_name

Nombre del fichero a ser comprobado.

options

Una o más constantes Fileinfo .

context

Para una descripción de contexts, dirígase a Funciones de Flujos.

Valores devueltos

Devuelve una descripción de texto del contenido del argumento file_name, o false en caso de error.

Ejemplos

Ejemplo #1 Un ejemplo de finfo_file()

<?php
$finfo = finfo_open(FILEINFO_MIME_TYPE); // devuelve el tipo mime de su extensión
foreach (glob("*") as $filename) {
echo finfo_file($finfo, $filename) . "\n";
}
finfo_close($finfo);
?>

El resultado del ejemplo sería algo similar a:

text/html
image/gif
application/vnd.ms-excel
