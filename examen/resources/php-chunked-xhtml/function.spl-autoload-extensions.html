spl_autoload_extensions

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

spl_autoload_extensions — Registrar y devolver las extensiones de ficheros predeterminadas para spl_autoload

Descripción

spl_autoload_extensions(string $file_extensions = ?): string

Esta función puede modificar y verificar las extensiones de ficheros que estará usando spl_autoload() en respaldo de la función interna __autoload().

Parámetros

file_extensions

Cuando se llama sin ningún argumento, simplemente devuelve la lista actual de extensiones separadas por comas. Para modificar la lista de extensiones de fichero, llame a las funciones con la nueva lista de extensiones de fichero a usar en una simple cadena con cada extensión serparada por una coma.

Valores devueltos

Una lista delimitada por comas de las extensiones predeterminadas de ficheros de para spl_autoload().
