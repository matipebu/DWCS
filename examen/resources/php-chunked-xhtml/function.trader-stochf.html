trader_stochf

(PECL trader >= 0.2.0)

trader_stochf — Estocástico rápido

Descripción

trader_stochf(
    array $high,
    array $low,
    array $close,
    int $fastK_Period = ?,
    int $fastD_Period = ?,
    int $fastD_MAType = ?
): array

Parámetros

high

Precio alto, array de valores reales.

low

Precio bajo, array de valores reales.

close

Precio de cierre, array de valores reales.

fastK_Period

Periodo de tiempo para la construcción de la línea K-Rápida. Rango válido desde 1 hasta 100000.

fastD_Period

Suavizado para la realización de la línea D-Rápida. Rango válido desde 1 hasta 100000, normalmente establecido a 3.

fastD_MAType

Tipo de media móvil para D-rápido. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
