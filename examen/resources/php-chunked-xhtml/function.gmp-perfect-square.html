gmp_perfect_square

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_perfect_square — Comprueba el cuadrado perfecto

Descripción

gmp_perfect_square(GMP|int|string $num): bool

Revisa si un número es cuadrado perfecto.

Parámetros

num

El número a ser revisado como un cuadrado perfecto.

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

Devielve true si num es un cuadrado perfecto, sino false.

Ejemplos

Ejemplo #1 Ejemplo de gmp_perfect_square()

<?php
// 3 * 3, cuadrado perfecto
var_dump(gmp_perfect_square("9"));

// no es un cuadrado perfecto
var_dump(gmp_perfect_square("7"));

// 1234567890 * 1234567890, cuadrado perfecto
var_dump(gmp_perfect_square("1524157875019052100"));
?>

El resultado del ejemplo sería:

bool(true)
bool(false)
bool(true)
