GearmanClient::doLowBackground

(PECL gearman >= 0.5.0)

GearmanClient::doLowBackground — Ejecuta una tarea de prioridad baja en segundo plano

Descripción

public GearmanClient::doLowBackground(string $function_name, string $workload, string $unique = ?): string

Ejecuta una tarea de prioridad baja en segundo plano, retornando el manejador del trabajo que puede ser usado para obtener el estado de la tarea en ejecución. Las tareas de prioridad normal o alta tienen preferencia sobre las de baja prioridad en la cola de trabajos.

Parámetros

function_name

Función registrada que ejecutará el worker

workload

Datos serializados a ser procesados

unique

ID único para identificar una tarea en particular

Valores devueltos

El manejador de trabajo para la tarea enviada.
