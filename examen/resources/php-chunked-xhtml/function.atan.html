atan

(PHP 4, PHP 5, PHP 7, PHP 8)

atan — Arco tangente

Descripción

atan(float $num): float

Devuelve el arco tangente de num en radianes. atan() es la función complementaria de tan(), lo que quiere decir que a==tan(atan(a)) para cada valor de a en el rango de atan().

Parámetros

num

El argumento a procesar

Valores devueltos

El arco tangente de num en radianes.
