ImagickPixel::isPixelSimilarQuantum

(PECL imagick 3 >= 3.3.0)

ImagickPixel::isPixelSimilarQuantum — Description

Descripción

public ImagickPixel::isPixelSimilarQuantum(string $color, string $fuzz = ?): bool

Devuelve true si la distancia entre dos colores es menor que la distancia especificada. El valor de fuzz debería estar en el rango 0-QuantumRange. El valor máximo representa la mayor distancia posible en el espacio de color, p.ej., de RGB(0, 0, 0) a RGB(255, 255, 255) en el espacio de color RGB

Parámetros

color

fuzz

Valores devueltos
