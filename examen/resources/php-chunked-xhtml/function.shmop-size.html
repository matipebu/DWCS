shmop_size

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

shmop_size — Obtiene el tamaño de un segmento de memoria compartida

Descripción

shmop_size(resource $shmid): int

shmop_size() es utilizado ara obtener el tamaño en bytes, de un segmento de memoria compartida.

Parámetros

shmid

El identificador del segmento de memoria compartida creado por shmop_open()

Valores devueltos

Devuelve un entero, el cual representa el número de bytes que ocupa el segmento de memoria compartida.

Historial de cambios

Versión Descripción
7.0.0 El tipo de shmid ha sido cambiado de int a resource.

Ejemplos

Ejemplo #1 Obtención del tamaño de un segmento de memoria compartida

<?php
$shm_size = shmop_size($shm_id);
?>

Este ejemplo, almacenará el tamaño del segmento de memoria compartida identificado por $shm_id en $shm_size.
