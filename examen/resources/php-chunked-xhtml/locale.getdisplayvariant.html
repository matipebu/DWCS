Locale::getDisplayVariant

locale_get_display_variant

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Locale::getDisplayVariant -- locale_get_display_variant — Devolver un nombre manifiesto debidamente localizado de las variantes de la configuración regional de entrada

Descripción

Estilo orientado a objetos

public static Locale::getDisplayVariant(string $locale, string $in_locale = ?): string

Estilo por procedimientos

locale_get_display_variant(string $locale, string $in_locale = ?): string

Devuelve un nombre manifiesto debidamente localizado de las variantes de la configuración regional de entrada. Si es null se utiliza la configuración regional predeterminada.

Parámetros

locale

La configuración regional de la que devolver una variante manifiesta.

in_locale

El formato regional opcional a usar para mostrar el nombre de la variante

Valores devueltos

El nombre manifiesto de la variante de la configuración regional dada por $locale en el formato apropiado dado por $in_locale.

Ejemplos

Ejemplo #1 Ejemplo de locale_get_display_variant()

<?php
echo locale_get_display_variant('sl-Latn-IT-nedis', 'en');
echo ";\n";
echo locale_get_display_variant('sl-Latn-IT-nedis', 'fr');
echo ";\n";
echo locale_get_display_variant('sl-Latn-IT-nedis', 'de');
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
echo Locale::getDisplayVariant('sl-Latn-IT-nedis', 'en');
echo ";\n";
echo Locale::getDisplayVariant('sl-Latn-IT-nedis', 'fr');
echo ";\n";
echo Locale::getDisplayVariant('sl-Latn-IT-nedis', 'de');
?>

El resultado del ejemplo sería:

Natisone dialect;
dialecte de Natisone;
NEDIS
