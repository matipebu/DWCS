IntlCalendar::__construct

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::__construct — Constructor privado para no permitir la creación de instancias

Descripción

private IntlCalendar::__construct()

Constructor privado para no permitir la creación de instancias con el operador new.

En su lugar, llama a IntlCalendar::createInstance().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
