SQLite3Stmt::clear

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SQLite3Stmt::clear — Limpiar todos los parámetros vinculados actuales

Descripción

public SQLite3Stmt::clear(): bool

Limpia todos los parámetros vinculados actuales.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve true si la limpiza de parámetros vinculados tuvo éxito, false en caso de fallo.
