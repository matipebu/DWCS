curl_share_errno

(PHP 7 >= 7.1.0, PHP 8)

curl_share_errno — Devuelve el último número del error compartido cURL

Descripción

curl_share_errno(resource $sh): int

Devuelve un integer conteniendo el último número de error del gestor cURL compartido.

Parámetros

sh

Un gestor cURL compartido retornado por curl_share_init().

Valores devueltos

Devuelve un integer conteniendo el último número de error del gestor cURL compartido, o false en caso de error.
