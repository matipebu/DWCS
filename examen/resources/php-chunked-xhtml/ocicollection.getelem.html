OCICollection::getElem

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCICollection::getElem — Devuelve el valor de un elemento

Descripción

public OCICollection::getElem(int $index): string|float|null|false

Devuelve el valor del elemento cuyo índice es index (basado en 0).

Parámetros

index

El índice del elemento. El primer índice es 0.

Valores devueltos

Devuelve false si el elemento no existe; null si el elemento es null; un string si el elemento es una columna del tipo de datos string, o un número si es un campo numérico.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 La clase OCI-Collection ha sido renombrada a OCICollection para ajustarse a los estándares de nomenclatura de PHP.
