xhprof_disable

(PECL xhprof >= 0.9.0)

xhprof_disable — Detiene el perfilador xhprof

Descripción

xhprof_disable(): array

Detiene el perfilador, y devuelve los datos xhprof de la ejecución.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array de los datos xhprof, de la ejecución.

Ejemplos

Ejemplo #1 xhprof_disable() ejemplo

<?php
xhprof_enable();

$foo = strlen("foo bar");

$xhprof_data = xhprof_disable();

print_r($xhprof_data);
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [main()==>strlen] => Array
        (
            [ct] => 1
            [wt] => 279
        )

    [main()==>xhprof_disable] => Array
        (
            [ct] => 1
            [wt] => 9
        )

    [main()] => Array
        (
            [ct] => 1
            [wt] => 610
        )

)
