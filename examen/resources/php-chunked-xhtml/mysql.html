Información general de los controladores de MySQL para PHP

Introducción

Dependiendo de la versión de PHP, existen dos o tres API de PHP para acceder a las bases dedatos de MySQL. Los usuarios de PHP 5 pueden elegir entre la extensión mysql obsoleta, mysqli, o PDO_MySQL. PHP 7 elimina la extensión mysql, dejando solamente las últimas dos opciones.

Esta guía explica la terminología empleada para describir cada API, información sobre qué API elegir, y también información para ayudar a elegir qué biblioteca de MySQL usar con la API.
