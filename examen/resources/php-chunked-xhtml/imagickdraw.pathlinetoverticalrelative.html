ImagickDraw::pathLineToVerticalRelative

(PECL imagick 2, PECL imagick 3)

ImagickDraw::pathLineToVerticalRelative — Dibuja un trazado de línea vertical

Descripción

ImagickDraw::pathLineToVerticalRelative(float $y): bool
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Dibuja un trazado de línea vertical desde el punto actual al punto objetivo usando coordenadas relativas. El punto objetivo entonces se convierte en el nuevo punto actual.

Parámetros

y

coordenada y

Valores devueltos

No devuelve ningún valor.
