imagecolorclosesthwb

(PHP 4 >= 4.0.1, PHP 5, PHP 7, PHP 8)

imagecolorclosesthwb — Obtener el índice del color que tiene el tono, blancura y negrura

Descripción

imagecolorclosesthwb(
    resource $image,
    int $red,
    int $green,
    int $blue
): int

Obtiene el índice del color que tiene el tono, blancura y negrura más próximos al color dado.

Parámetros

image

Un recurso image, es devuelto por una de las funciones de creación de imágenes, como imagecreatetruecolor().

red

Valor del componente rojo.

green

Valor del componente verde.

blue

Valor del componente azul.

Valores devueltos

Devuelve un entero con el índice del color que tiene el tono, blancura y negrura más próximos al color dado.

Ejemplos

Ejemplo #1 Ejemplo usando imagecolorclosesthwb()

<?php
$im = imagecreatefromgif('php.gif');

echo 'TBN: ' . imagecolorclosesthwb($im, 116, 115, 152);

imagedestroy($im);
?>

El resultado del ejemplo sería algo similar a:

HWB: 33
