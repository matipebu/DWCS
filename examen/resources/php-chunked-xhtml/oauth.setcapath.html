OAuth::setCAPath

(PECL OAuth >= 0.99.8)

OAuth::setCAPath — Establece la ruta a CA y su información

Descripción

public OAuth::setCAPath(string $ca_path = ?, string $ca_info = ?): mixed

Establece el Certificado de Autoridad (CA), tanto la ruta como la información.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

ca_path

La ruta CA a ser establecida.

ca_info

La ruta de información CA a ser establecido.

Valores devueltos

Devuelve true al tener éxitos, o false si tanto ca_path o ca_info son considerados como inválidos.

Historial de cambios

Versión Descripción
1.0.0 Antes devolvía null en caso de falla, en lugar de false.
