trader_adosc

(PECL trader >= 0.2.0)

trader_adosc — Oscilador A/D Chaikin

Descripción

trader_adosc(
    array $high,
    array $low,
    array $close,
    array $volume,
    int $fastPeriod = ?,
    int $slowPeriod = ?
): array

Parámetros

high

Precio alto, array de valores reales.

low

Precio bajo, array de valores reales.

close

Precio de cierre, array de valores reales.

volume

Volumen comerciado, array de valores reales.

fastPeriod

Número del periodo para el MA rápido. Rango válido desde 2 hasta 100000.

slowPeriod

Número del periodo para el MA lento. Rango válido desde 2 hasta 100000.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de fallo.
