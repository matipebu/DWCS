ldap_free_result

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_free_result — Liberar la memoria de resultado

Descripción

ldap_free_result(resource $result_identifier): bool

Libera la memoria ubicada internamente para almacenar los resultados. Toda la memoria de resultado será liberada automáticamente cuando termine el script de ejecutarse.

Típicamente, toda la memoria ubicada para el resultado de LDAP se libera al final del script. En caso que el script esté haciendo búsquedas sucesivas lo cual devuelve conjuntos de resultados largos, ldap_free_result() podría ser llamado para mantener bajo el uso de la memoria en tiempo de ejecución por el script.

Parámetros

result_identifier

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
