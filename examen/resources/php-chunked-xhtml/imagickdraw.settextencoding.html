ImagickDraw::setTextEncoding

(PECL imagick 2, PECL imagick 3)

ImagickDraw::setTextEncoding — Especifica el conjunto de codificación del texto

Descripción

ImagickDraw::setTextEncoding(string $encoding): bool
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Especifica el conjunto de codificación del texto para usar en las anotaciones de texto. La única codificación de caracteres que se puede especificar por el momento es "UTF-8" para la representación Unicode como una secuencia de bytes. Especifique un string vacío para establecer la codificación de texto al predeterminado del sistema. El éxito en la anotación de texto usando Unicode podría requerir tipos de letra diseñados para admitir Unicode.

Parámetros

encoding

El nombre de la codificación.

Valores devueltos

No devuelve ningún valor.
