pg_options

(PHP 4, PHP 5, PHP 7, PHP 8)

pg_options — Obtener las opciones asociadas con la conexión

Descripción

pg_options(resource $connection = ?): string

pg_options() Devolverá una cadena que contiene las opciones especificadas en el recurso de connection de PostgreSQL.

Parámetros

connection

Recurso de conexión a la base de datos PostgreSQL. Cuando connection no esta presente, Se utiliza la conexión predeterminada. La conexión predeterminada es la última conexión hecha por pg_connect() o pg_pconnect().

Valores devueltos

Un string que contiene las opciones de connection, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de pg_options()

<?php
$pgsql_conn = pg_connect("dbname=mark host=localhost");
echo pg_options($pgsql_conn);
?>
