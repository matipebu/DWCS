IntlCalendar::getActualMinimum

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getActualMinimum — El valor mínimo de un campo, considerando el instante actual del objeto

Descripción

Estilo orientado a objetos

public IntlCalendar::getActualMinimum(int $field): int

Estilo por procedimientos

intlcal_get_actual_minimum(IntlCalendar $cal, int $field): int

Devuelve un valor mínimo relativo de un campo en torno al instante actual. La semántica exacta varía según cada campo, pero en general, será el valor que se obtendría si se estableciera el valor del campo al mínimo relativo más grande y se incrementara hasta alcanzar el mínimo global o sobrepasase, en donde el valor devuelto sería el mínimo global o el valor antes de sobrepasar, respectivamente.

Para el calendario gregoriano, este es siempre el mismo que IntlCalendar::getMinimum().

Parámetros

cal

El recurso IntlCalendar.

field

Una de las constantes del campo fecha/hora de IntlCalendar. Son valores de tipo integer entre 0 y IntlCalendar::FIELD_COUNT.

Valores devueltos

Un int que representa el valor mínimo en las del o false en caso de error.
