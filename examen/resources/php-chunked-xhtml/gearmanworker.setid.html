GearmanWorker::setId

(No version information available, might only be in Git)

GearmanWorker::setId — Asigna un identificador al trabajador de tal forma que pueda ser rastreado cuando se solicite a gearmand la lista de los trabajadores disponibles

Descripción

public GearmanWorker::setId(string $id): bool

Asigna un identificador al trabajador.

Parámetros

id

Un identificador de tipo string.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de GearmanWorker::setId()

Define un identificador para un trabajador simple.

<?php
$worker= new GearmanWorker();
$worker->setId('test');
?>

El resultado del ejemplo sería algo similar a:

Ejecutar el comando siguiente:
gearadmin --workers
