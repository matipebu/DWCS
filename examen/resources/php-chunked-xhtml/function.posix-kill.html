posix_kill

(PHP 4, PHP 5, PHP 7, PHP 8)

posix_kill — Enviar una señal a un proceso

Descripción

posix_kill(int $pid, int $sig): bool

Envía la señal sig al proceso con el identificador de proceso pid.

Parámetros

pid

El identificador de proceso.

sig

Una de las constantes de señales PCNTL.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
