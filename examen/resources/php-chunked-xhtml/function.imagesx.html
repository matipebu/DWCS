imagesx

(PHP 4, PHP 5, PHP 7, PHP 8)

imagesx — Obtener el ancho de una imagen

Descripción

imagesx(resource $image): int

Devuelve el ancho del recurso image dado.

Parámetros

image

Un recurso image, es devuelto por una de las funciones de creación de imágenes, como imagecreatetruecolor().

Valores devueltos

Devuelve el ancho de image o false si se produjeron errores.

Ejemplos

Ejemplo #1 Usar imagesx()

<?php

// crear una imagen de 300*200
$img = imagecreatetruecolor(300, 200);

echo imagesx($img); // 300

?>
