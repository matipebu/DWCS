openssl_pkey_get_public

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

openssl_pkey_get_public — Extrae la clave pública del certificado y la prepara para usarla

Descripción

openssl_pkey_get_public(mixed $certificate): resource

openssl_get_publickey() extrae la clave pública de certificate y la prepara para usarla con otras funciones.

Parámetros

certificate

certificate puede ser lo siguiente:

  1. un recurso de certificado X.509
  2. un string con el formato file://ruta/al/archivo.pem. El archivo nombrado debe contener un certificado/clave público PEM codificado (podría contener ambos).
  3. una clave pública con formato PEM.

Valores devueltos

Devuelve un identificador de clave positivo si tuvo éxito, o false si se produjo un error.
