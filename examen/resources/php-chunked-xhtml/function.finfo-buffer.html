finfo_buffer

finfo::buffer

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL fileinfo >= 0.1.0)

finfo_buffer -- finfo::buffer — Devuelve información sobre el búfer en formato string

Descripción

Estilo por procedimientos

finfo_buffer(
    resource $finfo,
    string $string = null,
    int $options = FILEINFO_NONE,
    resource $context = null
): string

Estilo orientado a objetos

public finfo::buffer(string $string = null, int $options = FILEINFO_NONE, resource $context = null): string

Esta función se usa para obtener información sobre datos binarios en formato string.

Parámetros

finfo

Recurso Fileinfo devuelto por finfo_open().

string

Contenido del fichero a ser comprobado.

options

Una o más constantes Fileinfo .

context

Valores devueltos

Devuelve una descripción contextual del argumento string o false si ocurrió un error.

Ejemplos

Ejemplo #1 Ejemplo de finfo_buffer()

<?php
$finfo = new finfo(FILEINFO_MIME);
echo $finfo->buffer($_POST["script"]) . "\n";
?>

El resultado del ejemplo sería algo similar a:

application/x-sh; charset=us-ascii
