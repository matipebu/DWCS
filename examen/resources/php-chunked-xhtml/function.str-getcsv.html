str_getcsv

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

str_getcsv — Convierte un string con formato CSV a un array

Descripción

str_getcsv(
    string $input,
    string $delimiter = ",",
    string $enclosure = '"',
    string $escape = "\\"
): array

Convierte el string dado por input con campos en formato CSV y devuelve un array que contiene los campos leídos.

Parámetros

input

El string a convertir.

delimiter

Establece el delimitador de campos (sólo un carácter).

enclosure

Establece el carácter circundante de cada campo (sólo un carácter).

escape

Establece el carácter de escape (sólo un carácter). Por defecto, es una barra invertida. (\)

Valores devueltos

Devuelve un array indexado que contiene los campos leídos.
