pg_port

(PHP 4, PHP 5, PHP 7, PHP 8)

pg_port — Devuelve el número de puerto asociado con la conexión

Descripción

pg_port(resource $connection = ?): int

pg_port() devuelve el número de puerto al que el recurso de connection PostgreSQL está conectado.

Parámetros

connection

Recurso de conexión de base de datos PostgreSQL. Cuando la connection no está presente, se utiliza la conexión por defecto. La conexión por defecto es la última conexión hecha por pg_connect() o pg_pconnect().

Valores devueltos

Un int que contiene el número de puerto del servidor de base de datos para la connection, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de pg_port()

<?php
$pgsql_conn = pg_connect("dbname=mark host=localhost");

if ($pgsql_conn) {
print "Conectado correctamente al puerto: " . pg_port($pgsql_conn) . "<br/>\n";
} else {
print pg_last_error($pgsql_conn);
exit;
}
?>
