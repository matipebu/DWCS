ps_shading

(PECL ps >= 1.3.0)

ps_shading — Crear un sombreado para su uso posterior

Descripción

ps_shading(
    resource $psdoc,
    string $type,
    float $x0,
    float $y0,
    float $x1,
    float $y1,
    float $c1,
    float $c2,
    float $c3,
    float $c4,
    string $optlist
): int

Crea un sombreado que puede ser utilizado por las funciones ps_shfill() o ps_shading_pattern().

El color del sombreado puede estar en cualquier espacio de color excepto para pattern.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

type

El tipo de sombreado puede ser radial o axial. Cada sombreado comienza con el color de relleno actual y termina con los valores de color pasados a los parámetros c1 a c4 (véase la función ps_setcolor() para sus significados).

x0, x1, y0, y1

Las coordenadas x0, y0, x1, y1 son el punto de inicio y el punto final del sombreado. Si el tipo de sombreado es radial los dos puntos son los puntos medios de un círculo inicial y final.

c1, c2, c3, c4

Véase la función ps_setcolor() para sus significados.

optlist

Si el sombreado es de tipo radial el parámetro optlist debe contener también los parámetros r0 y r1 con el radio del círculo inicial y final.

Valores devueltos

Devuelve el identificador del patrón o false en caso de error.
