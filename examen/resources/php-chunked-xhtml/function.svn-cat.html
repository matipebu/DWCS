svn_cat

(PECL svn >= 0.1.0)

svn_cat — Devuelve el contenido de un fichero en un repositorio

Descripción

svn_cat(string $repos_url, int $revision_no = ?): string

Devuelve el contenido del fichero de un repositorio indicado en repos_url, de forma opcional se puede indicar opcionalmente el número de revisión en revision_no.

Parámetros

repos_url

La ruta URL del ítem en el repositorio (cadena).

revision_no

Número de revisión del ítem a obtener, por defecto es la última revisión (entero).

Valores devueltos

Devuelve el contenido del ítem del repositorio en caso de éxito, y false en caso de error.

Notas

Advertencia

Esta función ha sido declarada EXPERIMENTAL. Su funcionamiento, nombre y la documentación que le acompaña puede cambiar sin previo aviso en futuras versiones de PHP. Utilícela bajo su propia responsabilidad.

Ejemplos

Ejemplo #1 Ejemplo básico

Este ejemplo devuelve el contenido de un fichero en su revisión 28:

<?php
$contents = svn_cat('http://www.example.com/svnroot/calc/gui.c', 28)
?>
