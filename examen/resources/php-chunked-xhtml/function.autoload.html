__autoload

(PHP 5, PHP 7)

__autoload — Intenta cargar una clase sin definir

Advertencia

Esta característica ha sido declarada OBSOLETA a partir de PHP 7.2.0. Su uso está totalmente desaconsejado.

Descripción

__autoload(string $class): void

Puede definir esta función para habilitar la carga de clases.

Parámetros

class

Nombre de la clase a cargar

Valores devueltos

No devuelve ningún valor.
