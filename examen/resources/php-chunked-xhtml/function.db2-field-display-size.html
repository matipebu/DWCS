db2_field_display_size

(PECL ibm_db2 >= 1.0.0)

db2_field_display_size — Devolver el número máximo de bytes requeridos para mostrar una columna

Descripción

db2_field_display_size(resource $stmt, mixed $column): int

Devuelve el número máximo de bytes requeridos para mostrar una columna de un bloque de resultados.

Parámetros

stmt

Especifica un recurso válido que contiene los resultados.

column

Especifica la columna del resultado. Puede ser tanto un número entero que represente la posición de la columna (indexada a partir de cero) como una cadena con el nombre de la columna.

Valores devueltos

Devuelve un valor entero con el número máximo de bytes requeridos para mostrar una columna específica. Si la columna no existe en el resultado, db2_field_display_size() devuelve false.
