ftp_set_option

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

ftp_set_option — Establecer varias opciones FTP de tiempo de ejecución

Descripción

ftp_set_option(resource $ftp_stream, int $option, mixed $value): bool

Esta función controla diversas opciones de tiempo de ejecución para la secuencia FTP especificada.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

option

Actualmente, se admiten las siguientes opciones:

Opciones FTP de tiempo de ejecución soportadas
FTP_TIMEOUT_SEC Cambia el tiempo de espera en segundos usado para todas las funciones de red. value debe ser un número entero mayor que 0. El tiempo de espera predeterminado es 90 segundos.
FTP_AUTOSEEK Cuando se activa, las peticiones GET o PUT con un parámetro resumepos o startpos buscarán primero la posición solicitada dentro del archivo. Este parámetro es activado por defecto.

value

Este parámetro depende de qué option se elige para ser alterado.

Valores devueltos

Devuelve true si la opción pudo ser establecida; false de lo contrario. Un mensaje de advertencia será lanzado si la option no es soportada o el value pasado no coincide con el valor esperado para la option especificada.

Ejemplos

Ejemplo #1 Ejemplo de ftp_set_option()

<?php
// Establecer el tiempo de espera de red a 10 segundos
ftp_set_option($conn_id, FTP_TIMEOUT_SEC, 10);
?>
