Collator::getErrorCode

collator_get_error_code

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Collator::getErrorCode -- collator_get_error_code — Obtener el último código de error del cotejeador

Descripción

Estilo orientado a objetos

public Collator::getErrorCode(): int|false

Estilo por procedimientos

collator_get_error_code(Collator $object): int|false

Parámetros

object

Un objeto de la clase Collator.

Valores devueltos

El código de error devuelto por la última llamada a una función de la API de Collator, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo collator_get_error_code()

<?php
$coll = collator_create( 'en_US' );
if( collator_get_attribute( $coll, Collator::FRENCH_COLLATION ) === false )
handle_error( collator_get_error_code() );
?>
