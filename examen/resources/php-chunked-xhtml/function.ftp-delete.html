ftp_delete

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_delete — Elimina un archivo en el servidor FTP

Descripción

ftp_delete(resource $ftp_stream, string $path): bool

ftp_delete() borra el archivo especificado por path desde el servidor FTP.

Parámetros

ftp_stream

El identificador de enlace de la conexión FTP.

path

El archivo a eliminar.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de ftp_delete()

<?php
$file = 'public_html/old.txt';

// establecer conexión básica
$conn_id = ftp_connect($ftp_server);

// iniciar sesión con nombre de usuario y contraseña
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// intentar eliminar el archivo $file
if (ftp_delete($conn_id, $file)) {
echo "$file se ha eliminado satisfactoriamente\n";
} else {
echo "No se pudo eliminar $file\n";
}

// cerrar la conexión ftp
ftp_close($conn_id);
?>
