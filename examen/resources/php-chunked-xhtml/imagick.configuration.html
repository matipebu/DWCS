Configuración en tiempo de ejecución

El comportamiento de estas funciones se ve afectado por la configuración de php.ini.

Opciones de configuración de Imagick
Nombre Por defecto Cambiable Historial de cambios
imagick.locale_fix false INI_ALL Disponible desde Imagick 2.1.0
imagick.progress_monitor false INI_SYSTEM Disponible desde Imagick 2.2.2
imagick.skip_version_check false INI_SYSTEM Disponible desde Imagick 3.3.0
Para más detalles y definiciones de los modos de INI_*, vea Dónde se puede realizar un ajuste de configuración.

He aquí una breve explicación de las directivas de configuración.

imagick.locale_fix boolean

Corrige un error de dibujo con localismos que emplean ',' como separadores decimales.

imagick.progress_monitor boolean

Usado para habilitar el monitor de progreso de imagen.

imagick.skip_version_check boolean

Cuando Imagick se carga, se comprueba el número de versión de ImageMagick con el que se compiló con el número de verisón que se está utilizando actualmente, produciendo una advertencia si no coinciden. Esta advertencia se puede omitir habilitando esta opción.

No se recomienda utilizar una versión de compilación de Imagick diferente a la versión de ImageMagick que se esté empleado. Aunque pueda parecer que funciona, puede conducir a fallos aleatorios u otro tipo de comportamiento no definido.
