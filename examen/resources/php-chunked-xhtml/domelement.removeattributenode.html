DOMElement::removeAttributeNode

(PHP 5, PHP 7, PHP 8)

DOMElement::removeAttributeNode — Elimina un atributo

Descripción

public DOMElement::removeAttributeNode(DOMAttr $attr): DOMAttr|false

Elimina el atributo attr del elemento.

Parámetros

attr

El nodo atributo.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Errores/Excepciones

DOM_NO_MODIFICATION_ALLOWED_ERR

Lanzado si el nodo es de sólo lectura.

DOM_NOT_FOUND_ERROR

Lanzado si attr no es un atributo del elemento.
