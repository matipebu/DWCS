Imagick::setFont

(PECL imagick 2 >= 2.1.0, PECL imagick 3)

Imagick::setFont — Establece el tipo de letra

Descripción

Imagick::setFont(string $font): bool

Establece la propiedad 'font' del objeto. Este método se puede usar para, por ejemplo, establecer el tipo de letra para una leyenda: pseudo-formato. El tipo de letra necesita estar configurado en la configuración de ImageMagick o ser un fichero cuyo nombre de font debe existir. Este método no debería confundirse con ImagickDraw::setFont() el cuál establece el tipo de letra para un objeto ImagickDraw específico. Este método está disponible si Imagick ha sido compilado con la versión 6.3.7 o superior de ImageMagick.

Parámetros

font

El nombre del tipo de letra o un nombre de fichero

Valores devueltos

Devuelve true en caso de éxito.

Ejemplos

Ejemplo #1 Un ejemplo de Imagick::setFont()

Ejemplo usando Imagick::setFont

<?php
/* Crear un nuevo objeto imagick */
$im = new Imagick();

/* Establecer el tipo de letra para el objeto */
$im->setFont("example.ttf");

/* Crear una leyenda nueva */
$im->newPseudoImage(100, 100, "caption:Hola");

/* Hacer algo con la imagen */
?>
