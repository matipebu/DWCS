odbc_result_all

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_result_all — Imprimir el resultado como una tabla HTML

Descripción

odbc_result_all(resource $result_id, string $format = ?): int

Imprime todas las filas de un identificador de resultado producido por odbc_exec(). El resultado se imprime con formato de tabla HTML.

Parámetros

result_id

El identificador de resultado.

format

Formateo de tabla global adicional.

Valores devueltos

Devuelve el número de filas del resultado o false si se produjo un error.
