mb_encoding_aliases

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

mb_encoding_aliases — Obtiene los alias de un tipo de codificación conocido

Descripción

mb_encoding_aliases(string $encoding): array

Devuelve un array con los alias de un tipo de codificación conocido dado por encoding type.

Parámetros

encoding

El tipo de codificación del cual se quieren obtener los alias.

Valores devueltos

Devuelve un array indexado numéricamente con los alias de la condificación en caso de éxito, o false en caso de error

Errores/Excepciones

Emite un error de nivel E_WARNING si encoding es desconocido.

Ejemplos

Ejemplo #1 Ejemplo de mb_encoding_aliases()

<?php
$encoding = 'ASCII';
$known_encodings = mb_list_encodings();

if (in_array($encoding, $known_encodings)) {

$aliases = mb_encoding_aliases($encoding);
print_r($aliases);

} else {

echo "Codificación ($encoding) desconocida.\n";

}
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [0] => ANSI_X3.4-1968
    [1] => iso-ir-6
    [2] => ANSI_X3.4-1986
    [3] => ISO_646.irv:1991
    [4] => US-ASCII
    [5] => ISO646-US
    [6] => us
    [7] => IBM367
    [8] => cp367
    [9] => csASCII
)
