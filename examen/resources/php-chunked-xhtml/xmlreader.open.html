XMLReader::open

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

XMLReader::open — Establece el URI conteniendo el XML a analizar

Descripción

public static XMLReader::open(string $uri, ?string $encoding = null, int $flags = 0): bool|XMLReader

Establece el URI que contiene el XML a ser analizado.

Parámetros

uri

El URI señalado para el documento.

encoding

El documento codificado o null.

flags

Una mascara de bit de las contantes de LIBXML_*.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error. Si es llamado de forma estática, devuelve un XMLReader o false en caso de error.

Errores/Excepciones

This method may be called statically, but prior to PHP 8.0.0, will issue an E_DEPRECATED error in this case.

Historial de cambios

Versión Descripción
8.0.0 XMLReader::open() is now declared as static method, but can still be called on an XMLReader instance.
