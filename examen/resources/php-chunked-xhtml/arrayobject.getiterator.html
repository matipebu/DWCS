ArrayObject::getIterator

(PHP 5, PHP 7, PHP 8)

ArrayObject::getIterator — Crear un nuevo iterador desde una instancia ArrayObject

Descripción

public ArrayObject::getIterator(): ArrayIterator

Crear un nuevo iterador desde una instancia ArrayObject.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un iterador de una clase ArrayObject.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::getIterator()

<?php
$array = array('1' => 'uno',
'2' => 'dos',
'3' => 'tres');

$arrayobject = new ArrayObject($array);

$iterator = $arrayobject->getIterator();

while($iterator->valid()) {
echo $iterator->key() . ' => ' . $iterator->current() . "\n";

$iterator->next();
}
?>

El resultado del ejemplo sería:

1 => uno
2 => dos
3 => tres
