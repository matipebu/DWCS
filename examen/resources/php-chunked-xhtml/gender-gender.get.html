Gender\Gender::get

(PECL gender >= 0.6.0)

Gender\Gender::get — Obtener el género de un nombre

Descripción

public Gender\Gender::get(string $name, integer $country = ?): integer

Obtener el género de un nombre en un país concreto.

Parámetros

name

Nombre a comprobar.

country

ID del país como constante de la clase Gender.

Valores devueltos

Devuelve el género de un nombre.
