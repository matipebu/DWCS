msg_get_queue

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

msg_get_queue — Crear o adjuntar a una cola de mensajes

Descripción

msg_get_queue(int $key, int $perms = 0666): resource

msg_get_queue() devuelve un id que se puede usar para acceder a la cola de mensajes de System V con la clave dada por key. La primera llamada crea la cola de mensajes con los permisos opcionales dados por perms. Una segunda llamada a msg_get_queue() por la misma clave key devolverá un identificador de cola de mensajes diferente, pero ambos identificadores acceden a la misma cola de mensajes subyacente.

Parámetros

key

ID numérico de la cola de mensajes

perms

Permisos de la cola. Por defecto es 0666. Si la cola de mensajes ya existe, perms será ignorado.

Valores devueltos

Devuelve un gestor de recurso que se puede usar para acceder a la cola de mensajes de System V.
