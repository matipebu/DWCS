MultipleIterator::current

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

MultipleIterator::current — Obtiene las instancias registradas del iterador

Descripción

public MultipleIterator::current(): array

Obtiene las instancias registradas del iterador del resultado current().

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array conteniendo los valores actuales de cada iterador adjuntado, o false si no hay ningún iterador adjuntado.

Errores/Excepciones

Lanza una RuntimeException si el modo MIT_NEED_ALL está establecido y almenos uno de los iteradores adjuntados no es válido. O lanza una IllegalValueException si la clave es null y se establece MIT_KEYS_ASSOC.
