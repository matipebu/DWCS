ps_close_image

(PECL ps >= 1.1.0)

ps_close_image — Cerrar una imagen y liberar memoria

Descripción

ps_close_image(resource $psdoc, int $imageid): void

Cierra una imagen y libera sus recursos. Una vez que una imagen ha sido cerrada ya no puede usarse más.

Parámetros

psdoc

El identificador de recursos del fichero postscript, como el devuelto por la función ps_new().

imageid

El identificador de recursos de la imagen, como el devuelto por las funciones ps_open_image() o ps_open_image_file().

Valores devueltos

Devuelve null en caso de éxito, o false en caso de error.
