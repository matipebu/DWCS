getservbyport

(PHP 4, PHP 5, PHP 7, PHP 8)

getservbyport — Obtener el servicio de Internet que corresponde con el puerto y protocolo

Descripción

getservbyport(int $port, string $protocol): string

getservbyport() devuelve el servicio de Internet asociado con port para el protocol especificado, de acuerdo a /etc/services.

Parámetros

port

El número de puerto.

protocol

protocol es un valor entre "tcp" o "udp" (en minúsculas).

Valores devueltos

Devuelve el nombre del servicio de Internet como una cadena.
