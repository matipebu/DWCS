XMLReader::expand

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

XMLReader::expand — Devuelve una copia del actual nodo como un objeto DOM

Descripción

public XMLReader::expand(?DOMNode $baseNode = null): DOMNode|false

Éste metodo copia el nodo actual y devuelve el apropiado objeto DOM.

Parámetros

baseNode

A DOMNode defining the target DOMDocument for the created DOM object.

Valores devueltos

El resultante DOMNode o false en caso de error.
