OAuth::setTimestamp

(PECL OAuth >= 1.0.0)

OAuth::setTimestamp — Establece el timestamp

Descripción

public OAuth::setTimestamp(string $timestamp): mixed

Establece el timestamp de OAuth para las subsecuentes peticiones.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

timestamp

El timestamp.

Valores devueltos

Devuelve true, a menos que timestamp sea inválido, en cuyo caso false será devuelto.

Historial de cambios

Versión Descripción
1.0.0 Antes devolvía null en caso de falla, en lugar de false.
