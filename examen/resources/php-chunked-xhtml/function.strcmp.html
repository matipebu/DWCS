strcmp

(PHP 4, PHP 5, PHP 7, PHP 8)

strcmp — Comparación de string segura a nivel binario

Descripción

strcmp(string $str1, string $str2): int

Tenga en cuenta que esta comparación considera mayúsculas y minúsculas.

Parámetros

str1

El primer string.

str2

El segundo string.

Valores devueltos

Devuelve < 0 si str1 es menor que str2; > 0 si str1 es mayor que str2 y 0 si son iguales.

Ejemplos

Ejemplo #1 Ejemplo de strcmp()

<?php
$var1 = "Hola";
$var2 = "hola";
if (strcmp($var1, $var2) !== 0) {
echo '$var1 no es igual a $var2 en una comparación que considera mayúsculas y minúsculas';
}
?>
