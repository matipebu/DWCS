streamWrapper::url_stat

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::url_stat — Recuperar información sobre un archivo

Descripción

public streamWrapper::url_stat(string $path, int $flags): array

Este método es llamado en respuesta a todas las funciones relacionadas con stat(), como:

  • copy()
  • fileperms()
  • fileinode()
  • filesize()
  • fileowner()
  • filegroup()
  • fileatime()
  • filemtime()
  • filectime()
  • filetype()
  • is_writable()
  • is_readable()
  • is_executable()
  • is_file()
  • is_dir()
  • is_link()
  • file_exists()
  • lstat()
  • stat()
  • SplFileInfo::getPerms()
  • SplFileInfo::getInode()
  • SplFileInfo::getSize()
  • SplFileInfo::getOwner()
  • SplFileInfo::getGroup()
  • SplFileInfo::getATime()
  • SplFileInfo::getMTime()
  • SplFileInfo::getCTime()
  • SplFileInfo::getType()
  • SplFileInfo::isWritable()
  • SplFileInfo::isReadable()
  • SplFileInfo::isExecutable()
  • SplFileInfo::isFile()
  • SplFileInfo::isDir()
  • SplFileInfo::isLink()
  • RecursiveDirectoryIterator::hasChildren()

Parámetros

path

La ruta del archivo o de la URL de donde obtener las estadísticas. Observe que en el caso que sea una URL, debe ser una URL delimitada con ://. No están soportadas formas de URL.

flags

Contiene banderas adicionales establecidas por la API de flujos. Puede contener uno o más de los siguientes valores usando OR entre ellos.

Bandera Descripción
STREAM_URL_STAT_LINK Para recursos con la capacidad de enlazar a otros recursos (como HTTP Location: forward, o un enlace simbólico de sistema de archivos). Esta bandera especifica que sólo debería ser devuelta la información acerca del enlace mismo, no del recurso apuntado por el enlace. Esta bandera está establecida en respuesta a las llamadas a lstat(), is_link(), o filetype().
STREAM_URL_STAT_QUIET Si esta bandera está establecida, la envoltura no debería lanzar ningún error. Si esta bandera no está establecida, uno mismo es responsable de notificar errores usando la función trigger_error() durante la obtención de estadísticas de la ruta.

Valores devueltos

Debería devolver tantos elementos como tiene stat(). Valores desconocidos o no disponibles deberían ser establecidos a un valor razonable (normalmente 0).

Errores/Excepciones

Emite un E_WARNING si la llamada a este método falla (por ej., por no estar implementado).

Notas

Nota:

La propiedad streamWrapper::$context es actualizada si se pasa un contexto válido en la función invocante.
