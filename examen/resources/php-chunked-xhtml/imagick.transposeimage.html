Imagick::transposeImage

(PECL imagick 2, PECL imagick 3)

Imagick::transposeImage — Crea una imagen espejo vertical

Descripción

Imagick::transposeImage(): bool

Crea una imagen espejo vertical reflejando los píxeles alrededor del eje x central mientras los rota 90 grados. Este método está disponible si Imagick ha sido compilado con la versión 6.2.9 o superior de ImageMagick.

Valores devueltos

Devuelve true en caso de éxito.

Ver también

  • Imagick::transverseImage() - Crea una imagen espejo horizontal

Ejemplos

Ejemplo #1 Imagick::transposeImage()

<?php
function transposeImage($imagePath) {
$imagick = new \Imagick(realpath($imagePath));
$imagick->transposeImage();
header("Content-Type: image/jpg");
echo $imagick->getImageBlob();
}

?>
