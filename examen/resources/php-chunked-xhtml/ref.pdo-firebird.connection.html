PDO_FIREBIRD DSN

(PECL PDO_FIREBIRD >= 0.1.0)

PDO_FIREBIRD DSN — Conectarse a bases de datos de Firebird

Descripción

El Nombre del Origen de Datos (DSN) de PDO_FIREBIRD está compuesto por los siguientes elementos:

DSN prefix

El prefijo DSN es firebird:.

dbname

El nombre de la base de datos.

charset

El juego de caracteres.

role

El nombre del rol de SQL.

Ejemplos

Ejemplo #1 Ejemplo de DSN de PDO_FIREBIRD DSN con ruta

El siguiente ejemplo muestra un DSN de PDO_FIREBIRD DSN para conectarse a bases de datos de Firebird: 

firebird:dbname=/ruta/a/BASEDATOS.FDB

Ejemplo #2 Ejemplo de DSN de PDO_FIREBIRD DSN con puerto y ruta

El siguiente ejemplo muestra un DSN de PDO_FIREBIRD DSN para conectarse a una base de datos de Firebird usando el puerto del nombre del host y la ruta: 

firebird:dbname=nombre_host/puerto:/ruta/A/BASEDATOS.FDB

Ejemplo #3 Ejemplo de DSN de PDO_FIREBIRD con localhost y ruta hacia empleado.fdb en un sistema Debian

El siguiente ejemplo muestra un DSN de PDO_FIREBIRD DSN para conectarse a la base de datos empleado.fdb de Firebird usando: 

firebird:dbname=localhost:/var/lib/firebird/2.5/data/empleado.fdb
