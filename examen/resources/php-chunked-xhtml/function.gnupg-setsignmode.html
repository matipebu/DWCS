gnupg_setsignmode

(PECL gnupg >= 0.1)

gnupg_setsignmode — Fija el modo para firmar

Descripción

gnupg_setsignmode(resource $identifier, int $signmode): bool

Fija el modo para firmar.

Parámetros

identifier

El identificador gnupg, desde una llamada a gnupg_init() o gnupg.

sigmode

El modo para firmar.

signmode toma el valor de una constante indicando qué tipo de firma debería producirse. Los posibles valores son: GNUPG_SIG_MODE_NORMAL, GNUPG_SIG_MODE_DETACH y GNUPG_SIG_MODE_CLEAR. Por defecto se usa GNUPG_SIG_MODE_CLEAR.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de gnupg_setsignmode() mediante funciones

<?php
$res = gnupg_init();
gnupg_setsignmode($res,GNUPG_SIG_MODE_DETACH); // produce una firma separada
?>

Ejemplo #2 Ejemplo de gnupg_setsignmode() mediante OO

<?php
$gpg = new gnupg();
$gpg -> setsignmode(gnupg::SIG_MODE_DETACH); // produce una firma separada
?>
