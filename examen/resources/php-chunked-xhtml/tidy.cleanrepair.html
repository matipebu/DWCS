tidy::cleanRepair

tidy_clean_repair

(PHP 5, PHP 7, PHP 8, PECL tidy >= 0.5.2)

tidy::cleanRepair -- tidy_clean_repair — Ejecuta una operación de limpieza y reparación de las etiquetas HTML

Descripción

Estilo orientado a objetos

public tidy::cleanRepair(): bool

Estilo por procedimientos

tidy_clean_repair(tidy $tidy): bool

Esta función limpia y repara el objeto object pasado como argumento.

Parámetros

tidy

El objeto Tidy.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de tidy::cleanrepair()

<?php
$html = '<p>test</I>';

$tidy = tidy_parse_string($html);
$tidy->cleanRepair();

echo $tidy;
?>

El resultado del ejemplo sería:

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 3.2//EN">
<html>
<head>
<title></title>
</head>
<body>
<p>test</p>
</body>
</html>
