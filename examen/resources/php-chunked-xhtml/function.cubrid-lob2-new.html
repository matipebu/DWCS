cubrid_lob2_new

(PECL CUBRID >= 8.4.1)

cubrid_lob2_new — Crea un objeto LOB

Descripción

cubrid_lob2_new(resource $conn_identifier = ?, string $type = "BLOB"): resource

La función cubrid_lob2_new() se usa para crear un objeto LOB (BLOB y CLOB). Se debería usar esta función antes de vincular un objeto LOB.

Parámetros

conn_identifier

Un identificador de conexión. Si no se especifica un identificador de conexión, se asumirá la última conexión abierte mediante cubrid_connect() o cubrid_connect_with_url().

type

Puede ser "BLOB" o "CLOB", y no distingue mayúsculas/minúsculas. El valor predeterminado es "BLOB".

Valores devueltos

Un identificador de LOB cuando tiene éxito.

false en caso de error.
