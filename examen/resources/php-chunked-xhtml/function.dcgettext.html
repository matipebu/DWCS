dcgettext

(PHP 4, PHP 5, PHP 7, PHP 8)

dcgettext — Sobrescribe el dominio de la búsqueda única del mensaje

Descripción

dcgettext(string $domain, string $message, int $category): string

Esta función le permite sobrescribir el dominio actual para una búsqueda unica del mensaje.

Parámetros

domain

El dominio

message

El mesanje

category

La categoría

Valores devueltos

Un string si es correcto.
