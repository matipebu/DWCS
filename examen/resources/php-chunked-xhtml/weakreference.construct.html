WeakReference::__construct

(PHP 7 >= 7.4.0, PHP 8)

WeakReference::__construct — Constructor que no permite la instanciación

Descripción

public WeakReference::__construct()

Este método existe sólo para no permitir la instanciación de la clase WeakReference. Las referencias débiles deben ser instanciadas con el método factory WeakReference::create().

Parámetros

Esta función no tiene parámetros.
