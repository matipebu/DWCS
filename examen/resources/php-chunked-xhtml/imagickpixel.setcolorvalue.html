ImagickPixel::setColorValue

(PECL imagick 2, PECL imagick 3)

ImagickPixel::setColorValue — Establece el valor normalizado de uno de los canales

Descripción

ImagickPixel::setColorValue(int $color, float $value): bool

Establece el valor del canal especificado de este objeto al valor proporcionado, que debería estar entre 0 y 1. Esta función se puede usar para proporcionar un canal de opacidad a un objeto ImagickPixel.

Parámetros

color

Una de las constantes de color de Imagick, p.ej. \Imagick::COLOR_GREEN o \Imagick::COLOR_ALPHA.

value

El valor a establecer para este canal, desde 0 a 1.

Valores devueltos

Devuelve true en caso de éxito.

Ejemplos

Ejemplo #1 Uso básico de Imagick::setColorValue()

<?php

$color = new \ImagickPixel('firebrick');

$color->setColorValue(Imagick::COLOR_ALPHA, 0.5);

print_r($color->getcolor(true));
?>

El resultado del ejemplo sería:

    [g] => 0.13333333333333
    [b] => 0.13333333333333
    [a] => 0.50000762951095
