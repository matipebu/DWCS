imagecreatefromxpm

(PHP 4 >= 4.0.1, PHP 5, PHP 7, PHP 8)

imagecreatefromxpm — Crea una nueva imagen a partir de un fichero o de una URL

Descripción

imagecreatefromxpm(string $filename): resource

imagecreatefromxpm() devuelve un identificador de imagen que representa la imagen obtenida desde el nombre de fichero dado.

Sugerencia

Se puede emplear un URL como nombre de fichero con esta función si las envolturas de fopen han sido activadas. Véase fopen() para más información de cómo especificar el nombre de fichero. Véanse las Protocolos y Envolturas soportados; continen enlaces con información sobre las diferentes capacidades que tienen las envolturas, notas sobre su empleo, e información de cualquier variable predefinida que podría proporcionarse.

Parámetros

filename

Ruta de la imagen XPM.

Valores devueltos

En caso de éxito, devuelve un identificador de recurso de imagen, y false en caso de error.

Historial de cambios

Versión Descripción
PHP 5.3.19 imagecreatefromxpm() está disponible en Windows.

Ejemplos

Ejemplo #1 Crear una instancia de imagen usando imagecreatefromxpm()

<?php
// Comprobar el soporte para XPM
if(!(imagetypes() & IMG_XPM))
{
die('¡No se encontró el soporte para xpm!');
}

// Crear una instancia de una imagen
$xpm = imagecreatefromxpm('./ejemplo.xpm');

// Hacer las operaciones con la imagen aquí

// PHP no tiene soporte para escribir imágenes xpm
// por lo que, en este caso, guardaremos la imagen como un
// fichero jpeg con el 100% de calidad
imagejpeg($xpm, './ejemplo.jpg', 100);
imagedestroy($xpm);
?>
