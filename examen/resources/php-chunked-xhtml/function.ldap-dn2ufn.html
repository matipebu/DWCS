ldap_dn2ufn

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_dn2ufn — Convertir un DN a un formato de nombramiento amigable al usuario

Descripción

ldap_dn2ufn(string $dn): string

Convierte un dn específico, a un formato más amigable al usuario, mostrando los tipos de nombre.

Parámetros

dn

El nombre distinguido de una entidad de LDAP.

Valores devueltos

Devuelve el nombre amigable al usuario.
