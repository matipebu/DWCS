ImagickDraw::getStrokeDashArray

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getStrokeDashArray — Devuelve un array que representa el patrón de rayas y huecos usado por trazados contorneados

Descripción

ImagickDraw::getStrokeDashArray(): array
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve un array que representa el patrón de rayas y huecos usado por trazados contorneados.

Valores devueltos

Devuelve un array si se tuvo éxito y un array vacío si no está establecido.
