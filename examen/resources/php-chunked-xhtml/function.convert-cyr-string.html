convert_cyr_string

(PHP 4, PHP 5, PHP 7)

convert_cyr_string — Convierte de un juego de caracteres cirílico a otro juego de caracteres cirílico

Descripción

convert_cyr_string(string $str, string $from, string $to): string

Convierte de un juego de caracteres cirílico a otro juego de caracteres cirílico

Parámetros

str

Cadena que será convertida

from

Fuente del juego de caracteres cirílicos, como un único caracter.

to

Objetivo o destino del juego de caracteres cirílicos, como un único caracter.

Los caracteres admitidos son:

  • k - koi8-r
  • w - windows-1251
  • i - iso8859-5
  • a - x-cp866
  • d - x-cp866
  • m - x-mac-cyrillic

Valores devueltos

Devuelve la cadena convertida.

Notas

Nota: Esta función es segura binariamente.
