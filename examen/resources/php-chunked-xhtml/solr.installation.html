Instalación

Se puede encontrar información para la instalación de esta extensión PECL en el capítulo del manual titulado Instalación de extensiones PECL. Se puede encontrar información adicional, tal como nuevos lanzamientos, descargas, ficheros fuente, información de mantenimiento, y un CHANGELOG, aquí: » https://pecl.php.net/package/solr.

Para obtener ayuda y soporte, visite la extensión en el grupo de Google. » Extensión PHP Apache Solr

Los binarios de Windows (ficheros DLL) para esta extensión de PECL están disponibles en el sitio web de PECL.

Nota:

El módulo solr se puede compilar en modo depuración pasando la bandera --enable-solr-debug a la configuración

Cuando se construya manualmente, asegúrese de incluir soporta para curl y libxml dentro de la construcción.
