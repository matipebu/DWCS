mysqli::$server_version

mysqli_get_server_version

(PHP 5, PHP 7, PHP 8)

mysqli::$server_version -- mysqli_get_server_version — Devuelve la versión del servidor MySQL como un valor entero

Descripción

Estilo orientado a objetos

int $mysqli->server_version;

Estilo por procedimientos

mysqli_get_server_version(mysqli $link): int

La función mysqli_get_server_version() devuelve la versión del servidor al que está conectado (representado por el parámetro link ) como un valor entero.

Parámetros

link

Sólo estilo por procediminetos: Un identificador de enlace devuelto por mysqli_connect() o mysqli_init()

Valores devueltos

Un entero que representa la versión del servidor.

El patrón de este número de versión es main_version * 10000 + minor_version * 100 + sub_version (p.ej. versión 4.1.0 es 40100).

Ejemplos

Ejemplo #1 Ejemplo de $mysqli->server_version

Estilo orientado a objetos

<?php
$mysqli = new mysqli("localhost", "my_user", "my_password");

/* verificar conexión */
if (mysqli_connect_errno()) {
printf("Connect failed: %s\n", mysqli_connect_error());
exit();
}

/* mostrar versión del servidor */
printf("Server version: %d\n", $mysqli->server_version);

/* verificar conexión */
$mysqli->close();
?>

Estilo por procedimientos

<?php
$link = mysqli_connect("localhost", "my_user", "my_password");

/* check connection */
if (mysqli_connect_errno()) {
printf("Connect failed: %s\n", mysqli_connect_error());
exit();
}

/* mostrar versión del servidor */
printf("Server version: %d\n", mysqli_get_server_version($link));

/* cerrar conexión */
mysqli_close($link);
?>

El resultado de los ejemplos sería:

Server version: 40102
