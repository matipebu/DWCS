getmxrr

(PHP 4, PHP 5, PHP 7, PHP 8)

getmxrr — Obtener los registros MX correspondientes a un nombre de host de Internet

Descripción

getmxrr(string $hostname, array &$mxhosts, array &$weight = ?): bool

Busca DNS para registros MX correspondientes a hostname.

Parámetros

hostname

El nombre de host de Internet.

mxhosts

Una lista de los registros MX encontrados se coloca en el array mxhosts.

weight

Si se proporciona el array weight, será rellenado con el peso de la información recopilada.

Valores devueltos

Devuelve true si se encuentra cualquier registro; false si no se encuentran registros o si ocurre un error.

Historial de cambios

Versión Descripción
5.3.0 Esta función ahora está disponible en plataformas Windows.

Notas

Nota:

Esto función no debería utilizarse con propósitos de verificación de direcciones. Solamente se devuelven los intercambiadores de correo en contrados en DNS, sin embargo, según el » RFC 2821 cuando no se enumera ningún intercambiador de correo, debería utlizarse el mismo hostname como el único intercambiador de correo con una prioridad de 0.

Nota:

Por compatibilidad con Windows antes de que fuera implementada, se puede intentar con las clase » Net_DNS de » PEAR.
