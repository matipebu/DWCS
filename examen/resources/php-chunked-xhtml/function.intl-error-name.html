intl_error_name

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

intl_error_name — Obtiene un nombre simbólico a partir de un código de error dado

Descripción

intl_error_name(int $error_code): string

Devuelve el nombre del código de error ICU.

Parámetros

error_code

Código de error ICU.

Valores devueltos

El string devuelto será el mismo que el nombre de la constante del código de error.

Ejemplos

Ejemplo #1 intl_error_name() ejemplo

<?php
$coll = collator_create( 'en_RU' );
$err_code = collator_get_error_code( $coll );

printf( "Symbolic name for %d is %s\n.", $err_code, intl_error_name( $err_code ) );
?>

El resultado del ejemplo sería algo similar a:

Symbolic name for -128 is U_USING_FALLBACK_WARNING.
