is_tainted

(PECL taint >=0.1.0)

is_tainted — Comprobar si un string está corrompido

Descripción

is_tainted(string $string): bool

Comprueba si un string está corrompido

Parámetros

string

Valores devueltos

Devuelve TRUE si el string está corrompido, FALSE en caso contrario.
