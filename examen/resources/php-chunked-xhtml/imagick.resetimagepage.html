Imagick::resetImagePage

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::resetImagePage — Reinicia una página de imagen

Descripción

public Imagick::resetImagePage(string $page): bool

La definición de página como un string. La string está en formato WxH+x+y (Ancho+Alto+x+y). Este método está disponible si Imagick ha sido compilado con la versión 6.3.6 o superior de ImageMagick.

Parámetros

page

La definición de página. Por ejemplo 7168x5147+0+0

Valores devueltos

Devuelve true en caso de éxito.
