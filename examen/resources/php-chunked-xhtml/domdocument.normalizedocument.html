DOMDocument::normalizeDocument

(PHP 5, PHP 7, PHP 8)

DOMDocument::normalizeDocument — Normaliza el documento

Descripción

public DOMDocument::normalizeDocument(): void

Este método actúa como si guardara y después cargara el documento, poniendo el el documento en una forma "normal".

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
