oci_lob_is_equal

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

oci_lob_is_equal — Compara dos localizadores LOB/FILE

Descripción

oci_lob_is_equal(OCI-Lob $lob1, OCI-Lob $lob2): bool

Compara dos localizadores LOB/FILE.

Parámetros

lob1

Un identificador LOB.

lob2

Un identificador LOB.

Valores devueltos

Devuelve true si los objetos son iguales, false si no.
