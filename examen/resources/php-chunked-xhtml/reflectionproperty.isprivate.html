ReflectionProperty::isPrivate

(PHP 5, PHP 7, PHP 8)

ReflectionProperty::isPrivate — Comprueba si una propiedad es privada

Descripción

public ReflectionProperty::isPrivate(): bool

Comprueba si una propiedad es privada.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

true si la propiedad es privada, o false en caso contrario.
