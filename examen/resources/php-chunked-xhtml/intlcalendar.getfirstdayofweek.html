IntlCalendar::getFirstDayOfWeek

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getFirstDayOfWeek — Obtener el primer día de la semana de la región del calendario

Descripción

Estilo orientado a objetos

public IntlCalendar::getFirstDayOfWeek(): int

Estilo por procedimientos

intlcal_get_first_day_of_week(IntlCalendar $cal): int

El día que se estima que empieza la semana, el valor predeterminado para esta región o el valor establecido con IntlCalendar::setFirstDayOfWeek().

Parámetros

cal

El recurso IntlCalendar.

Valores devueltos

Una de las constnates IntlCalendar::DOW_SUNDAY, IntlCalendar::DOW_MONDAY, …, IntlCalendar::DOW_SATURDAY o false en caso de error.

Ejemplos

Ejemplo #1 IntlCalendar::getFirstDayOfWeek()

<?php
ini_set('date.timezone', 'UTC');

$cal1 = IntlCalendar::createInstance(NULL, 'es_ES');
var_dump($cal1->getFirstDayOfWeek()); // Lunes
$cal1->set(2013, 1 /* February */, 3); // Un domingo
var_dump($cal1->get(IntlCalendar::FIELD_WEEK_OF_YEAR)); // 5

$cal2 = IntlCalendar::createInstance(NULL, 'en_US');
var_dump($cal2->getFirstDayOfWeek()); // Domingo
$cal2->set(2013, 1 /* February */, 3); // Un domingo
var_dump($cal2->get(IntlCalendar::FIELD_WEEK_OF_YEAR)); // 6

El resultado del ejemplo sería:

int(2)
int(5)
int(1)
int(6)
