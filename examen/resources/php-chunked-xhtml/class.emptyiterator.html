La clase EmptyIterator

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Introducción

La clase EmptyIterator para un iterador vacío.

Sinopsis de la Clase

class EmptyIterator implements Iterator {
/* Métodos */
public current(): never
public key(): never
public next(): void
public rewind(): void
public valid(): false
}
