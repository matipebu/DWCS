Imagick::compareImageLayers

(PECL imagick 2, PECL imagick 3)

Imagick::compareImageLayers — Devuelve la región circundante máxima entre imágenes

Descripción

public Imagick::compareImageLayers(int $method): Imagick

Compara cada imagen con la siguiente en una secuencia y devuelve la región circundante máxima de cualesquiera deferencias de píxeles que se descubra. Este método está disponible si Imagick ha sido compilado con la versión 6.2.9 o superior de ImageMagick.

Parámetros

method

Una de las constantes de método de capas.

Valores devueltos

Devuelve true en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.

Ejemplos

Ejemplo #1 Usar Imagick::compareImageLayers()

Comparar las capas de la imagen

<?php
/* crear un nuevo objeto imagick */
$im = new Imagick("test.gif");

/* optimizar las capas de la imagen */
$resultado = $im->compareImageLayers(imagick::LAYERMETHOD_COALESCE);

/* trabajar sobre el $resultado */
?>
