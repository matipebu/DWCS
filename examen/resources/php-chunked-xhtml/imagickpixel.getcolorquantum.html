ImagickPixel::getColorQuantum

(No version information available, might only be in Git)

ImagickPixel::getColorQuantum — Description

Descripción

public ImagickPixel::getColorQuantum(): array

Devuelve el color del píxel de un array como valores Quantum. Si ImageMagick se compiló como HDRI, estos serán valores de tipo float, de lo contrario serán enteros.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve una matriz con las claves "r", "g", "b", "a".
