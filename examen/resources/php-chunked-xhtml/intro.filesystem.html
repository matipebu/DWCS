Introducción

No es necesaria ninguna biblioteca externa para construir esta extensión, pero si quiere que PHP admita LFS (ficheros grandes) en Linux, necesita tener una versión reciente de glibc y compilar PHP con las siguientes banderas de compilación: -D_LARGEFILE_SOURCE -D_FILE_OFFSET_BITS=64.
