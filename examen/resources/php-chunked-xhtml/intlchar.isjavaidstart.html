IntlChar::isJavaIDStart

(PHP 7, PHP 8)

IntlChar::isJavaIDStart — Comprobar si un punto de código es permisible como el primer carácter de un identificador de Java

Descripción

public static IntlChar::isJavaIDStart(mixed $codepoint): bool

Determina si el carácter esepecficado es permisible como el incio de un identificador de Java.

Además de IntlChar::isIDStart(), true para caracteres con categorías generales "Sc" (símbolos de moneda) y "Pc" (puntuación de conexión).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint podría iniciar un identificador de Java, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::isJavaIDStart("A"));
var_dump(IntlChar::isJavaIDStart("$"));
var_dump(IntlChar::isJavaIDStart("\n"));
var_dump(IntlChar::isJavaIDStart("\u{2603}"));
?>

El resultado del ejemplo sería:

bool(true)
bool(true)
bool(false)
bool(false)
