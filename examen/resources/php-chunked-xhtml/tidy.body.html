tidy::body

tidy_get_body

(PHP 5, PHP 7, PHP 8, PECL tidy 0.5.2-1.0)

tidy::body -- tidy_get_body — Devuelve un objeto tidyNode empezando con la etiqueta <body>

Descripción

Estilo orientado a objetos

public tidy::body(): ?tidyNode

Estilo por procedimientos

tidy_get_body(tidy $tidy): ?tidyNode

Devuelve un objeto tidyNode empezando por la etiqueta <body>.

Parámetros

tidy

El objeto Tidy.

Valores devueltos

Devuelve un objeto tidyNode empezando por la etiqueta <body> del árbol analizado por tidy.

Ejemplos

Ejemplo #1 Ejemplo de tidy::getBody()

<?php
$html = '
<html>
<head>
<title>test</title>
</head>
<body>
<p>paragraph</p>
</body>
</html>';

$tidy = tidy_parse_string($html);

$body = $tidy->Body();
echo $body->value;
?>

El resultado del ejemplo sería:

<body>
<p>paragraph</p>
</body>
