MessageFormatter::getLocale

msgfmt_get_locale

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

MessageFormatter::getLocale -- msgfmt_get_locale — Obtener la configuración regional desde la que se creó el formateador

Descripción

Estilo orientado a objetos

public MessageFormatter::getLocale(): string

Estilo por procedimientos

msgfmt_get_locale(NumberFormatter $formatter): string

Obtiene la configuración regional desde la que se creó el formateador.

Parámetros

formatter

El recurso del formateador

Valores devueltos

El nombre de la configuración regional

Ejemplos

Ejemplo #1 Ejemplo de msgfmt_get_locale()

<?php
$fmt = msgfmt_create('en_US', "Number {0,number}");
echo msgfmt_get_locale($fmt);
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$fmt = new MessageFormatter('en_US', "Number {0,number}");
echo $fmt->getLocale();
?>

El resultado del ejemplo sería:

en_US
