strlen

(PHP 4, PHP 5, PHP 7, PHP 8)

strlen — Obtiene la longitud de un string

Descripción

strlen(string $string): int

Devuelve la longitud del string dado.

Parámetros

string

El string del cual medir su longitud.

Valores devueltos

La longitud del string en caso de éxito, y 0 si el string esta vacío.

Historial de cambios

Versión Descripción
5.3.0 Las versiones anteriores trataban los arrays como el string Array, devolviendo así una longitud de string de 5 y emitiendo un error de nivel E_NOTICE.

Ejemplos

Ejemplo #1 Un ejemplo destrlen()

<?php
$str = 'abcdef';
echo strlen($str); // 6

$str = ' ab cd ';
echo strlen($str); // 7
?>

Notas

Nota:

strlen() devuelve el número de bytes en lugar del número de caracteres de un string.

Nota:

strlen() devuelve null cuando se ejecuta sobre arrays, y emite un error de nivel E_WARNING.
