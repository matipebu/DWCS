pg_host

(PHP 4, PHP 5, PHP 7, PHP 8)

pg_host — Devuelve el nombre de host asociado a la conexión

Descripción

pg_host(resource $connection = ?): string

pg_host() devuelve el nombre de host del recurso de connection PostgreSQL al que está conectado.

Parámetros

connection

Recurso de conexión de base de datos PostgreSQL. cuando el parametro connection no está presente, se utiliza la conexión por defecto. La conexión por defecto es la última conexión realizada por pg_connect() o pg_pconnect().

Valores devueltos

Una string que contiene el nombre del host de la connection, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de pg_host()

<?php
$pgsql_conn = pg_connect("dbname=mark host=localhost");

if ($pgsql_conn) {
print "Conectado correctamente a: " . pg_host($pgsql_conn) . "<br/>\n";
} else {
print pg_last_error($pgsql_conn);
exit;
}
?>
