Imagick::getImageChannelDistortions

(PECL imagick 2 >= 2.3.0, PECL imagick 3)

Imagick::getImageChannelDistortions — Obtiene las distorsiones del canal

Descripción

public Imagick::getImageChannelDistortions(Imagick $reference, int $metric, int $channel = Imagick::CHANNEL_DEFAULT): float

Compara uno o más canales de imagen de una imagen con una imagen reconstruida y devuelve las métricas de la distorsión especificada Este método está disponible si Imagick ha sido compilado con la versión 6.4.4 o superior de ImageMagick.

Parámetros

reference

Objeto Imagick que contiene la imagen de referencia

metric

Consulte esta lista de constantes de tipo de métrica.

channel

Proporciona cualquier contante de canal que sea válida para su modo de canal. Para aplicarlo a más de un canal, combine constantes de canal using bitwise operadores bit a bit. El valor por defecto para Imagick::CHANNEL_DEFAULT. Referirse a esta lista de constantes de canal

Valores devueltos

Devuelve un valor de tipo double que describe la distorsión del canal.

Errores/Excepciones

Lanza ImagickException en caso de error.
