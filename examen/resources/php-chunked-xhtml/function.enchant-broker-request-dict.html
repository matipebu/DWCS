enchant_broker_request_dict

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_broker_request_dict — Crear un diccionario usanto una etiqueta

Descripción

enchant_broker_request_dict(resource $broker, string $tag): resource

Crea un diccionario usando una etiqueta no vacía que se desea para solicitar un diccionario ("en_US", "de_DE", ...)

Parámetros

broker

Recurso de agente

tag

Una etiqueta que describe la configuración regional, por ejemplo en_US, de_DE

Valores devueltos

Devuelve un recurso de diccionario si se tuvo éxito o false en caso de error.

Ejemplos

Ejemplo #1 Un ejemplo de enchant_broker_request_dict()

$agente = enchant_broker_init();
