atanh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

atanh — Arco tangente hiperbólica

Descripción

atanh(float $num): float

Devuelve el arco tangente hiperbólico de num, es decir, el valor cuya tangente hiperbólica es num.

Parámetros

num

El argumento a procesar

Valores devueltos

El arco tangente hiperbólico de num
