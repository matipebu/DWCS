trader_ma

(PECL trader >= 0.2.0)

trader_ma — Media móvil

Descripción

trader_ma(array $real, int $timePeriod = ?, int $mAType = ?): array

Parámetros

real

Array de valores reales.

timePeriod

Número del periodo. Rango válido desde 2 hasta 100000.

mAType

Tipo de media móvil. Se debería usar la serie de constantes TRADER_MA_TYPE_*.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
