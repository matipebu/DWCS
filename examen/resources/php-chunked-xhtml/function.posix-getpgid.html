posix_getpgid

(PHP 4, PHP 5, PHP 7, PHP 8)

posix_getpgid — Obtener el id del grupo de procesos para un control de trabajo

Descripción

posix_getpgid(int $pid): int

Devuelve el identificador del grupo de procesos del proceso pid o false en caso de error.

Parámetros

pid

El ide del proceso.

Valores devueltos

Devuelve el identificador, como valor de tipo integer.

Ejemplos

Ejemplo #1 Ejemplo de uso de posix_getpgid()

<?php
$pid = posix_getppid();
echo posix_getpgid($pid); //35
?>

Notas

Nota:

Esta no es una función POSIX, pero es usual en sistemas BSD y System V. Si el sistema no soporta esta función, no será incluidaen tiempo de compilación. Se puede comprobar con function_exists().
