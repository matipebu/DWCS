stream_socket_shutdown

(PHP 5 >= 5.2.1, PHP 7, PHP 8)

stream_socket_shutdown — Cerrar una conexión full-duplex

Descripción

stream_socket_shutdown(resource $stream, int $how): bool

Cierra (parcialmente o no) una conexión full-duplex.

Nota:

El búfer o búferes podrían o no estar vaciados.

Parámetros

stream

Un flujo abierto (con stream_socket_client(), por ejemplo)

how

Una de las siguientes constantes: STREAM_SHUT_RD (deshabilita recepciones), STREAM_SHUT_WR (deshabilita transmisiones) o STREAM_SHUT_RDWR (deshabilita recepciones y transmisiones).

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Un ejemplo de stream_socket_shutdown()

<?php

$servidor = stream_socket_server('tcp://127.0.0.1:1337');
$cliente = stream_socket_client('tcp://127.0.0.1:1337');

var_dump(fputs($cliente, "hola"));

stream_socket_shutdown($cliente, STREAM_SHUT_WR);
var_dump(fputs($cliente, "hola")); // no funciona ahora

?>

El resultado del ejemplo sería algo similar a:

int(5)

Notice: fputs(): send of 5 bytes failed with errno=32 Broken pipe in test.php on line 9
int(0)
