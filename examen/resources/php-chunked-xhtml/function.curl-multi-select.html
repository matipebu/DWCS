curl_multi_select

(PHP 5, PHP 7, PHP 8)

curl_multi_select — Espera actividad en cualquier conexión en curl_multi

Descripción

curl_multi_select(resource $mh, float $timeout = 1.0): int

Permanece bloqueado hasta que haya algún tipo de actividad en conexiones curl_multi.

Parámetros

mh

El multi recurso cURL multi devuelto por curl_multi_init().

timeout

Tiempo, en segundos, para esperar por una respuesta.

Valores devueltos

Si funciona, devuelve el número de descripciones que hay en los conjuntos de descripciones. Esto puede ser 0 si no hay actividad en cualquiera de los descriptores. Si falla, esta función devolverá -1 si la selección falla (desde la llamada al sistema subyacente).
