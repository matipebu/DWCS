El interfaz SplSubject

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Introducción

El interfaz SplSubject es usado en conjunto con SplObserver para implementar el patrón de diseño Observer.

Sinopsis de la Interfaz

interface SplSubject {
/* Métodos */
public attach(SplObserver $observer): void
public detach(SplObserver $observer): void
public notify(): void
}

Tabla de contenidos

  • SplSubject::attach — Adjuntar un SplObserver
  • SplSubject::detach — Separa un observador
  • SplSubject::notify — Notificar un observador
