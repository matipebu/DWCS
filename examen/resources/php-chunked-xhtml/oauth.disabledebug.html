OAuth::disableDebug

(PECL OAuth >= 0.99.3)

OAuth::disableDebug — Desactiva la depuración detallada

Descripción

public OAuth::disableDebug(): bool

Desactiva la solicitud de información detallada (desactivado por omisión). Alternativamente, la propiedad debug puede ser establecida al valor false para desactivar el depurador.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

true

Historial de cambios

Versión Descripción
0.99.8 La propiedad debug fue agregada.
