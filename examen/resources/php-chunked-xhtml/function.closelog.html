closelog

(PHP 4, PHP 5, PHP 7, PHP 8)

closelog — Cerrar conexión al log del sistema

Descripción

closelog(): bool

closelog() cierra el descriptor que se utiliza para escribir en el registro del sistema. El uso de closelog() es opcional.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
