svn_auth_get_parameter

(PECL svn >= 0.1.0)

svn_auth_get_parameter — Recupera un parámetros de autenticación

Descripción

svn_auth_get_parameter(string $key): string

Recupera un parámetro de autenticación donde la clave es key. Para obtener una lista de claves válidas y sus significados, consulte la lista de constantes de autenticación.

Parámetros

key

Nombre de la clave. Utilice una constante de autenticación definidas por esta extensión para especificar una clave.

Valores devueltos

Devuelve el valor de cadena del parámetro en key; devuelve null si el parámetro no existe.

Notas

Advertencia
