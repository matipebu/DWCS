cubrid_get_autocommit

(PECL CUBRID >= 8.4.0)

cubrid_get_autocommit — Devuelve el modo auto-commit de la conexión

Descripción

cubrid_get_autocommit(resource $conn_identifier): bool

La función cubrid_get_autocommit() se usa para obtener el estado del modo auto-commit de la conexión de dase de batos CUBIRD.

Para CUBRID 8.4.0, el modo auto-commit está deshabilitado de forma predeterminada para administración de transacciones.

Para CUBRID 8.4.1, el modo auto-commit está habilitado de forma predeterminada para administración de transacciones.

Parámetros

conn_identifier

Identificador de conexión.

Valores devueltos

true, cuando el auto-commit está activado.

false, cuando el auto-commit está desactivado.

null en caso de error.
