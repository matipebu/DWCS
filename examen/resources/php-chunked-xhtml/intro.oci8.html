Introducción

Estas funciones permiten acceder a Oracle Database 12c, 11g, 10g, 9i y 8i. Admiten sentencias SQL y PL/SQL. Sus características básicas incluyen el control de transacciones, la vinculación de variables de PHP a parámetros de sustitución de Oracle, y la admisión de tipos de objetos grandes (LOB) y colecciones. Las características de escalabilidad de Oracle, como el Agrupamiento de Conexiones Residentes de Bases de Datos (DRCP por sus siglás en inglés), también tienen soporte.
