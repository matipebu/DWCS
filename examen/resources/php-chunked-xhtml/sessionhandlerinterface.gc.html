SessionHandlerInterface::gc

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

SessionHandlerInterface::gc — Limpieza de sesiones antiguas

Descripción

abstract public SessionHandlerInterface::gc(int $maxlifetime): int

Limpia las sesiones expiradas. Llamado por session_start(), basado en las configuraciones session.gc_divisor, session.gc_probability y session.gc_maxlifetime.

Parámetros

maxlifetime

Las sesiones que no han sido actualizadas desde los últimos maxlifetime segundos serán eliminadas.

Valores devueltos

El valor devuelto (normalmente true en caso de éxito, false en caso de error). Observe que este valor es devuelto internamente a PHP para su procesamiento.
