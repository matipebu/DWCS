trader_t3

(PECL trader >= 0.2.0)

trader_t3 — Media móvil exponencial triple (T3)

Descripción

trader_t3(array $real, int $timePeriod = ?, float $vFactor = ?): array

Parámetros

real

Array de valores reales.

timePeriod

Número del periodo. Rango válido desde 2 hasta 100000.

vFactor

Factor de volumen. Rango válido desde 1 hasta 0.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
