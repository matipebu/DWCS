Introducción

La extensión HRTime implementa una clase StopWatch de alta resolución. Utiliza las mejores API posibles de plataformas diferentes que tienen una resolución de hasta nanosegundos. También hace posible implementar un cronómetro personalizado empleando pulsos de bajo nivel emitidos por las API subyacentes.
