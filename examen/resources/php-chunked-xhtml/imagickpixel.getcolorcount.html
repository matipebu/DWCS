ImagickPixel::getColorCount

(PECL imagick 2, PECL imagick 3)

ImagickPixel::getColorCount — Devuelve la cuenta de color asociada a este

Descripción

ImagickPixel::getColorCount(): int

Devuelve la cuenta de color asociada a este.

La cuenta de color es el número de píxeles de la imagen que tienen el mismo color que este ImagickPixel.

ImagickPixel::getColorCount solamente parece funcionar para objetos ImagickPixel creados a través de Imagick::getImageHistogram()

Valores devueltos

Devuelve la cuenta de color como un entero si se tuvo éxito, lanzando una excepción ImagickPixelException si falló.

Ejemplos

Ejemplo #1 getColorCount() de ImagickPixel

<?php
$imagick = new \Imagick();
$imagick->newPseudoImage(640, 480, "magick:logo");
$histogramElements = $imagick->getImageHistogram();
$lastColor = array_pop($histogramElements);
echo "La última cuenta de color de píxel es: ".$lastColor->getColorCount();
?>

La salida será similar a: 

La última cuenta de color de píxel es: 256244
