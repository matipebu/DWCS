ReflectionMethod::getModifiers

(PHP 5, PHP 7, PHP 8)

ReflectionMethod::getModifiers — Obtiene los modificadores del método

Descripción

public ReflectionMethod::getModifiers(): int

Devuelve un campo de bits de los modificadores de acceso para este método.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Representación numérica de los modificadores enumerados a continuación. El significado de estos modificadores se describe en las constantes predefinidas.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionMethod::getModifiers()

<?php
class Prueba
{
final public static function foo()
{
return;
}
public function bar()
{
return;
}
}

$foo = new ReflectionMethod('Prueba', 'foo');

echo "Modificadores del método foo():\n";
echo $foo->getModifiers() . "\n";
echo implode(' ', Reflection::getModifierNames($foo->getModifiers())) . "\n";

$bar = new ReflectionMethod('Prueba', 'bar');

echo "Modificadores del método bar():\n";
echo $bar->getModifiers() . "\n";
echo implode(' ', Reflection::getModifierNames($bar->getModifiers()));
?>

El resultado del ejemplo sería algo similar a:

Modificadores del método foo():
261
final public static
Modificadores del método bar():
65792
public
