IntlChar::istitle

(PHP 7, PHP 8)

IntlChar::istitle — Comprobar si un punto de código es una letra titular

Descripción

public static IntlChar::istitle(mixed $codepoint): bool

Determina si el punto de código especificado es una letra titular.

true para la categoría general "Lt" (letra titular).

Parámetros

codepoint

El valor de tipo integer del punto de código (p.ej. 0x2603 para U+2603 SNOWMAN), o el carácter codificado como un string UTF-8 (p.ej. "\u{2603}")

Valores devueltos

Devuelve true si codepoint es una letra titular, false si no.

Ejemplos

Ejemplo #1 Probar diferentes puntos de código

<?php
var_dump(IntlChar::istitle("A"));
var_dump(IntlChar::istitle("a"));
var_dump(IntlChar::istitle("Φ"));
var_dump(IntlChar::istitle("φ"));
var_dump(IntlChar::istitle("1"));
?>

El resultado del ejemplo sería:

bool(false)
bool(true)
bool(false)
bool(true)
bool(false)
