gmp_sign

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_sign — El símbolo del número

Descripción

gmp_sign(GMP $a): int

Revisa el símbolo del número.

Parámetros

a

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

Devuelve 1 si a es positivo, -1 si a es negativo, y 0 si a es cero.

Ejemplos

Ejemplo #1 Ejemplo de gmp_sign()

<?php
// positivo
echo gmp_sign("500") . "\n";

// negativo
echo gmp_sign("-500") . "\n";

// cero
echo gmp_sign("0") . "\n";
?>

El resultado del ejemplo sería:

1
-1
0
