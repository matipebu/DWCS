Collator::getErrorMessage

collator_get_error_message

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Collator::getErrorMessage -- collator_get_error_message — Obtener el texto del último código de error del cotejador

Descripción

Estilo orientado a objetos

public Collator::getErrorMessage(): string|false

Estilo por procedimientos

collator_get_error_message(Collator $object): string|false

Recupera el mensaje del último error.

Parámetros

object

Un objeto de la clase Collator.

Valores devueltos

La descripción de un error ocurrido en la última llamada a una función de la API de Collator, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de collator_get_error_message()

<?php
$coll = collator_create( 'lt' );
if( collator_compare( $coll, 'y', 'k' ) === false ) {
echo collator_get_error_message( $coll );
}
?>
