OAuth::setNonce

(PECL OAuth >= 0.99.1)

OAuth::setNonce — Configura el nonce para las solicitudes posteriores

Descripción

public OAuth::setNonce(string $nonce): mixed

Establece el nonce para todas las solicitudes posteriores.

Parámetros

nonce

El valor para oauth_nonce.

Valores devueltos

Devuelve true si tiene éxito, o false si nonce es considerado como inválido.

Historial de cambios

Versión Descripción
1.0.0 Antes devolvía null en caso de falla, en lugar de false.
