ImagickDraw::getTextEncoding

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getTextEncoding — Devuelve la codificación usada para anotaciones de texto

Descripción

ImagickDraw::getTextEncoding(): string
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve un string que especifica la codificación usada para anotaciones de texto.

Valores devueltos

Devuelve un string que especifica la codificación o false si la codificación de texto no está establecida.
