trader_log10

(PECL trader >= 0.2.0)

trader_log10 — Logaritmo en base 10 de vectores

Descripción

trader_log10(array $real): array

Calcula el logaritmo en base 10 para cada valor de real y devuelve el array resultante.

Parámetros

real

Array de valores reales.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
