gzdecode

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

gzdecode — Decodifica una cadena comprimida con gzip

Descripción

gzdecode(string $data, int $max_length = 0): string|false

Está función retorna una versión decodificada de la entrada data.

Parámetros

data

Los datos para decodificar, codificados con gzencode().

max_length

La longitud máxima de datos que decodificar.

Valores devueltos

La cadena decodificada, o o false en caso de error.

Errores/Excepciones

En caso de fallo, se emite un error de nivel E_WARNING.
