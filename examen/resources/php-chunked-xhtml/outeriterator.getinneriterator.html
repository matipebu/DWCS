OuterIterator::getInnerIterator

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

OuterIterator::getInnerIterator — Devuelve el iterador interno de la entrada actual

Descripción

public OuterIterator::getInnerIterator(): ?Iterator

Devuelve el iterador interno de la entrada actual.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Returns the inner iterator for the current entry if it exists, or null otherwise.
