GlobIterator::count

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

GlobIterator::count — Obtiene el número de directorios y ficheros

Descripción

public GlobIterator::count(): int

Obtener el número de directorios y ficheros encontrados por la expresión glob.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el número de directoeios y ficheros, como un integer.

Ejemplos

Ejemplo #1 Ejemplo de GlobIterator::count()

<?php
$iterator = new GlobIterator('*.xml');

printf("Elementos encontrado(s): %d \r\n", $iterator->count());
?>

El resultado del ejemplo sería algo similar a:

Elementos encontrado(s): 8
