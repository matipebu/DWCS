OCILob::size

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::size — Devuelve el tamaño de un objeto grande

Descripción

public OCILob::size(): int|false

Obtiene el tamaño de un objeto grande.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la longitud del valor del objeto grande o false en caso de error. Los objetos vacíos tienen una longitud de cero.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 La clase OCI-Lob ha sido renombrada a OCILob para ajustarse a los estándares de nomenclatura de PHP.
