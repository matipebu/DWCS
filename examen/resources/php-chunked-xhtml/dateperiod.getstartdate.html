DatePeriod::getStartDate

(PHP 5 >= 5.6.5, PHP 7, PHP 8)

DatePeriod::getStartDate — Obtener la fecha inicial

Descripción

Estilo orientado a objetos

public DatePeriod::getStartDate(): DateTimeInterface

Obtiene la fecha inicial del periodo.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un objeto DateTimeImmutable cuando el objeto DatePeriod se inicializó con un objeto DateTimeImmutable para el parámetro start.

De lo contrario, devuelve un objeto DateTime.

Ejemplos

Ejemplo #1 Ejemplo de DatePeriod::getStartDate()

<?php
$periodo = new DatePeriod('R7/2016-05-16T00:00:00Z/P1D');
$inicio = $periodo->getStartDate();
echo $inicio->format(DateTime::ISO8601);
?>

El resultado del ejemplo sería:

2016-05-16T00:00:00+0000
