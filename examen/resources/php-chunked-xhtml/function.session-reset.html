session_reset

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

session_reset — Reinicializar el array de sesión con los valores originales

Descripción

session_reset(): void

session_reset() reinicializa una sesión con los valores originales guardados en el almaceén de sesiones. Esta función requiere una sesión activa ya que descarta los cambios en $_SESSION.

Valores devueltos

No devuelve ningún valor.
