Imagick::clone

(PECL imagick 2, PECL imagick 3)

Imagick::clone — Hace una copia exacta de un objeto Imagick

Descripción

Imagick::clone(): Imagick

Hace una copia exacta de un objeto Imagick.

Advertencia

Esta función ha sido declarada OBSOLETA a partir de imagick 3.1.0 en favor del uso de la palabra clave clone.

Ejemplos

Ejemplo #1 Clonación de un objeto Imagick en diferentes versiones de imagick

<?php
// Clonar un objeto Imagick en imagick 2.x y 3.0:
$newImage = $image->clone();

// Clonar un objeto Imagick desde 3.1.0 en adelante:
$newImage = clone $image;
?>

Valores devueltos

Se devuelve una copia de del objeto Imagick.
