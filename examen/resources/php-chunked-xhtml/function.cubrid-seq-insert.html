cubrid_seq_insert

(PECL CUBRID >= 8.3.0)

cubrid_seq_insert — Insertar un elemento en una columna tipo secuencia usando OID

Descripción

cubrid_seq_insert(
    resource $conn_identifier,
    string $oid,
    string $attr_name,
    int $index,
    string $seq_element
): bool

La función cubrid_col_insert() se usa para insertar un elemento en un atributo de tipo secuencia en la posició solicitada.

Parámetros

conn_identifier

Identificador de conexión.

oid

OID de la instancia con la que se quiere trabajar.

attr_name

Nombre del atributo en donde insertar una instancia.

index

Posición del elemento donde se quiere insertar el elemento (basado en 1).

seq_element

Contenido del elemento que se quiere insertar.

Valores devueltos

true, cuando el proceso tiene éxito.

false, cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_seq_insert()

<?php
$conn = cubrid_connect("localhost", 33000, "demodb", "dba");

@cubrid_execute($conn, "DROP TABLE foo");
cubrid_execute($conn, "CREATE TABLE foo(a int AUTO_INCREMENT, b set(int), c sequence(int), d char(10))");
cubrid_execute($conn, "INSERT INTO foo(a, b, c, d) VALUES(1, {1,2,3}, {11,22,33,333}, 'a')");

$req = cubrid_execute($conn, "SELECT * FROM foo", CUBRID_INCLUDE_OID);

cubrid_move_cursor($req, 1, CUBRID_CURSOR_FIRST);
$oid = cubrid_current_oid($req);

$attr = cubrid_col_get($conn, $oid, "c");
var_dump($attr);

cubrid_seq_insert($conn, $oid, "c", 5, "44");

$attr = cubrid_col_get($conn, $oid, "c");
var_dump($attr);

cubrid_close_request($req);
cubrid_disconnect($conn);
?>

El resultado del ejemplo sería:
