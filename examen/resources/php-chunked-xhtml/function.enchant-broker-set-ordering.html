enchant_broker_set_ordering

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_broker_set_ordering — Declara una preferencia de diccionarios a usar para el lenguaje

Descripción

enchant_broker_set_ordering(resource $broker, string $tag, string $ordering): bool

Declara una preferencia de diccionarios a usar para el lenguaje descrito/referido por 'tag'. El orden es una lista delimitada por comas de nombres de proveedores. Como excepción especial, la etiqueta (tag) "*" se puede usar como una etiqueta de lenguaje para declarar un orden predeterminado para cualquier lenguaje que no declare explícitamente un orden.

Parámetros

broker

Recurso de agente

tag

Etiqueta de lenguaje. La etiqueta especial "*" se puede usar como una etiqueta de lenguaje para declarar un orden predeterminado para cualquier lenguaje que no declare explícitamente un orden.

ordering

Lista delimitada por comas de nombres de proveedores

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
