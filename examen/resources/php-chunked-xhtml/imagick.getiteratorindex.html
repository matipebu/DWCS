Imagick::getIteratorIndex

(PECL imagick 2, PECL imagick 3)

Imagick::getIteratorIndex — Obtiene el índice de la imagen activa actual

Descripción

Imagick::getIteratorIndex(): int

Devuelve el índice de la imagen activa actual dentro del objeto Imagick. Este método está disponible si Imagick ha sido compilado con la versión 6.2.9 o superior de ImageMagick.

Valores devueltos

Devuelve un integer que contiene el índice de la imagen en la pila.

Errores/Excepciones

Lanza ImagickException en caso de error.

Ejemplos

Ejemplo #1 Usar Imagick::getIteratorIndex():

Crear imágenes, establecer y obtener el índice del iterador

<?php
$im = new Imagick();
$im->newImage(100, 100, new ImagickPixel("red"));
$im->newImage(100, 100, new ImagickPixel("green"));
$im->newImage(100, 100, new ImagickPixel("blue"));

$im->setIteratorIndex(1);
echo $im->getIteratorIndex();
?>
