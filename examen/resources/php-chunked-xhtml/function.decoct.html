decoct

(PHP 4, PHP 5, PHP 7, PHP 8)

decoct — Decimal a octal

Descripción

decoct(int $number): string

Devuelve una cadena que contiene una representación octal del argumento number dado. El mayor número dque puede ser convertido depende de la plataforma empleada. Para plataformas de 32 bits, este es normalmente 4294967295 en decimal, resultando en 37777777777. Para plataformas de 64 bits, este es normalmente 9223372036854775807 en decimal, resultando en 777777777777777777777.

Parámetros

number

Valor decimal a convertir

Valores devueltos

Representación tipo cadena octal de numero

Ejemplos

Ejemplo #1 Ejemplo de decoct()

<?php
echo decoct(15) . "\n";
echo decoct(264);
?>

El resultado del ejemplo sería:

17
410
