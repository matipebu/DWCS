ldap_mod_add

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_mod_add — Agregar valores de atributo a los atributos actuales

Descripción

ldap_mod_add(resource $link_identifier, string $dn, array $entry): bool

Agrega uno o más atributos al dn específico. Se realiza la modificación a nivel de atributo en lugar de realizarce a nivel de Objeto. Las adiciones a nivel de Objeto son hechas por la función ldap_add().

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelta por ldap_connect().

dn

El nombre distinguido de una entidad de LDAP.

entry

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Notas

Nota: Esta función es segura binariamente.
