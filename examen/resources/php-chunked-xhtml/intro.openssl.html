Introducción

Este módulo utiliza las funciones de » OpenSSL para la generación y verificación de firmas y para sellar (cifrar) y abrir (descifrar) datos. OpenSSL ofrece muchas características que este módulo no soporta actualmente. Algunas de éstas podrían añadirse en el futuro.
