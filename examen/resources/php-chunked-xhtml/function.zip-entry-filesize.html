zip_entry_filesize

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_entry_filesize — Devuelve el tamaño del fichero actual de una entrada de directorio

Descripción

zip_entry_filesize(resource $zip_entry): int

Devuelve el tamaño actual de una entrada de directorio específica.

Parámetros

zip_entry

Una entrada de directorio devuelva por zip_read().

Valores devueltos

El tamaño de una entrada de directorio.
