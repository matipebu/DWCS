AppendIterator::getArrayIterator

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

AppendIterator::getArrayIterator — Obtiene el getArrayIterator

Descripción

public AppendIterator::getArrayIterator(): ArrayIterator

Este método obtiene el objeto ArrayIterator empleado para almacenar los iteradores añadidos con AppendIterator::append().

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve un ArrayIterator que contiene los iteradores añadidos.
