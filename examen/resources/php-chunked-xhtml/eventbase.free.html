EventBase::free

(PECL event >= 1.10.0)

EventBase::free — Recursos ddisponibles asignados para esta base de eventos

Descripción

public EventBase::free(): void

Desasigna los recursos asignados por libevent para el objeto EventBase.

Advertencia

El método EventBase::free() no destruye el objeto en sí. Para destruir el objeto completamente llame a unset() , o asigne null.

Este método no desasigna ni separa ninguno de los eventos que están actualmente asociados al objeto EventBase, ni EventBase, ni cierra ninguno de sus sockets - cuidado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
