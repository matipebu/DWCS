SolrDocument::__unset

(PECL solr >= 0.9.2)

SolrDocument::__unset — Elimina un campo del documento

Descripción

public SolrDocument::__unset(string $fieldName): bool

Elimina un campo del documento cuando al campo se accede como una propiedad del objeto.

Parámetros

fieldName

El nombre del campo.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
