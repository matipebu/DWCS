ldap_mod_del

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_mod_del — Borrar valores de atributo de los atributos actuales

Descripción

ldap_mod_del(resource $link_identifier, string $dn, array $entry): bool

Remueve uno o más atributos de un dn específico. Realiza la modificación a nivel de atributo en lugar de hacerlo a nivel de Objeto. Las eliminaciones a nivel de Objeto son hechas por la función ldap_delete().

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelto por ldap_connect().

dn

El nombre distinguido de una entidad de LDAP.

entry

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
