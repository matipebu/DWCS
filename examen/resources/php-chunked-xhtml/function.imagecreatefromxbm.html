imagecreatefromxbm

(PHP 4 >= 4.0.1, PHP 5, PHP 7, PHP 8)

imagecreatefromxbm — Crea una nueva imagen a partir de un fichero o de una URL

Descripción

imagecreatefromxbm(string $filename): resource

imagecreatefromxbm() devuelve un identificador de imagen que representa la imagen obtenida desde el nombre de fichero dado.

Sugerencia

Se puede emplear un URL como nombre de fichero con esta función si las envolturas de fopen han sido activadas. Véase fopen() para más información de cómo especificar el nombre de fichero. Véanse las Protocolos y Envolturas soportados; continen enlaces con información sobre las diferentes capacidades que tienen las envolturas, notas sobre su empleo, e información de cualquier variable predefinida que podría proporcionarse.

Parámetros

filename

Ruta de la imagen XBM.

Valores devueltos

En caso de éxito, devuelve un identificador de recurso de imagen, y false en caso de error.

Ejemplos

Ejemplo #1 Convertir una imagen XBM en una imagen png usando imagecreatefromxbm()

<?php
// Cargar el archivo xbm
$xbm = imagecreatefromxbm('./ejemplo.xbm');

// Convertirla al archivo png
imagepng($xbm, './ejemplo.png');
imagedestroy($xbm);
?>
