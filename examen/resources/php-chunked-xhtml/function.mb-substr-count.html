mb_substr_count

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

mb_substr_count — Cuenta el número de ocurrencias de un substring

Descripción

mb_substr_count(string $haystack, string $needle, string $encoding = mb_internal_encoding()): int

Cuenta el número de veces que el substring needle aparece en el string haystack.

Parámetros

haystack

El string a comprobar.

needle

El string a buscar.

encoding

El parámetro encoding es la codificación de caracteres. Si es omitido, será usado el valor de la codificación de caracteres interna.

Valores devueltos

El número de veces que el substring needle aparece en el string haystack.

Ejemplos

Ejemplo #1 Ejemplo de mb_substr_count()

<?php
echo mb_substr_count("This is a test", "is"); // imprime 2
?>
