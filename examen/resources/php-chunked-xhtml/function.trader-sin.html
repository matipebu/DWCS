trader_sin

(PECL trader >= 0.2.0)

trader_sin — Seno trigonométrico de vectores

Descripción

trader_sin(array $real): array

Calcula el seno para cada valor de real y devuelve el array resultante.

Parámetros

real

Array de valores reales.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
