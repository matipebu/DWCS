enchant_dict_add_to_session

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_dict_add_to_session — Añadir una palabra a esta sesión ortográfica

Descripción

enchant_dict_add_to_session(resource $dict, string $word): void

Añade una palabra al diccionario dado. Sólo será añadida para la sesión ortográfica activa.

Parámetros

dict

Recurso de diccionario

word

La palabra a añadir

Valores devueltos

No devuelve ningún valor.
