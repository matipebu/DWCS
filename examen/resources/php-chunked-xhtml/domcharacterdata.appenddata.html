DOMCharacterData::appendData

(PHP 5, PHP 7, PHP 8)

DOMCharacterData::appendData — Añade el string de caracteres al final de los datos de carácter del nodo.

Descripción

public DOMCharacterData::appendData(string $data): true

Añade el string data hasta el final de los datos de carácter del nodo.

Parámetros

data

El string de caracteres a añadir.

Valores devueltos

Siempre devuelve true.
