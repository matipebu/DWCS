rad2deg

(PHP 4, PHP 5, PHP 7, PHP 8)

rad2deg — Convierte el número en radianes a su equivalente en grados

Descripción

rad2deg(float $number): float

Esta función convierte a numero desde radianes a grados.

Parámetros

number

Un valor en radianes

Valores devueltos

El equivalente de number en grados

Ejemplos

Ejemplo #1 Ejemplo de rad2deg()

<?php

echo rad2deg(M_PI_4); // 45

?>
