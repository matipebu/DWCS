OCILob::export

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCILob::export — Exporta el contenido de un LOB a un fichero

Descripción

public OCILob::export(string $filename, ?int $offset = null, ?int $length = null): bool

Exporta el contenido de un LOB a un fichero.

Parámetros

filename

La ruta del fichero.

start

Indica desde dónde se empieza a exportar.

length

Indica la longitud de los datos a exportar.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 offset y length ahora son nullable.
8.0.0, PECL OCI8 3.0.0 La clase OCI-Lob ha sido renombrada a OCILob para ajustarse a los estándares de nomenclatura de PHP.
