Collator::sort

collator_sort

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Collator::sort -- collator_sort — Ordenar un array usando cotejador especificado

Descripción

Estilo orientado a objetos

public Collator::sort(array &$array, int $flags = Collator::SORT_REGULAR): bool

Estilo por procedimientos

collator_sort(Collator $object, array &$array, int $flags = Collator::SORT_REGULAR): bool

Esta función ordena un array según las normal regionales actuales.

Es equivalente al estándar de PHP sort().

Parámetros

object

Un objeto Collator.

array

El array de cadenas a ordenar.

flags

El tipo de ordenación opcional, uno de los siguientes:

  • Collator::SORT_REGULAR - comparar los elementos de forma normal (no cambia los tipos)

  • Collator::SORT_NUMERIC - comparar los elementos de forma numérica

  • Collator::SORT_STRING - comparar los elementos como cadenas

El valor predeterminado de $sort_flag value es Collator::SORT_REGULAR. También se utiliza si se ha especificado un valor no válido para flags.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de collator_sort()

<?php
$coll = collator_create( 'en_US' );
$arr = array( 'at', 'às', 'as' );

var_export( $arr );
collator_sort( $coll, $arr );
var_export( $arr );
?>

El resultado del ejemplo sería:

array (
  0 => 'at',
  1 => 'às',
  2 => 'as',
)array (
  0 => 'as',
  1 => 'às',
  2 => 'at',
)
