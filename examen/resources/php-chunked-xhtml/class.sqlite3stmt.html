La clase SQLite3Stmt

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

Introducción

La clase que maneja declaraciones preparadas para la extensión SQLite.

Sinopsis de la Clase

class SQLite3Stmt {
/* Métodos */
public bindParam(mixed $sql_param, mixed &$param, int $type = ?): bool
public bindValue(mixed $sql_param, mixed $value, int $type = ?): bool
public clear(): bool
public close(): bool
public execute(): SQLite3Result
public getSQL(bool $expand = false): string|false
public paramCount(): int
public readOnly(): bool
public reset(): bool
}
