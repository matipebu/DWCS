EventBuffer::addBuffer

(PECL event >= 1.2.6-beta)

EventBuffer::addBuffer — Mueve todos los datos de un buffer a la instancia actual de EventBuffer

Descripción

public EventBuffer::addBuffer( EventBuffer $buf ): bool

Mueve todos los datos del buffer indicado en el parámetro buf al final del actual EventBuffer . Es un movimiento de datos destructivo. Los datos de un buffer se mueven al otro buffer. Sin embargo, no se realizan copias innecesarias.

Parámetros

buf

El objeto EventBuffer.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
