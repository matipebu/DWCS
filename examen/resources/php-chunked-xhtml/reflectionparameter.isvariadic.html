ReflectionParameter::isVariadic

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

ReflectionParameter::isVariadic — Comprueba si el parámetro es varíadico

Descripción

public ReflectionParameter::isVariadic(): bool

Comprueba si el parámetro fue declarado como un parámetro variádico.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve true si el parámetro es varíadico, o false en caso contrario.
