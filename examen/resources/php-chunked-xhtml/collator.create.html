Collator::create

collator_create

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Collator::create -- collator_create — Crear un cotejador

Descripción

Estilo orientado a objetos

public static Collator::create(string $locale): Collator

Estilo por procedimientos

collator_create(string $locale): Collator

Las cadenas serán comparadas usando las opciones ya especificadas.

Parámetros

locale

La configuración regional que contiene las normas de cotejo requeridas. Se pueden pasar valores especiales de configuraciones regionales - si se le pasa null, se usarán las normas de cotejo regionales predeterminadas. Si se le pasa una cadena vacía ("") o "root", se usarán las normas del » UCA.

Valores devueltos

Devuelve una nueva instancia de un objeto Collator, o null en case de error.

Ejemplos

Ejemplo #1 Ejemplo de collator_create()

<?php
$coll = collator_create( 'en_US' );

if( !isset( $coll ) ) {
printf( "Falló la creación del cotejador: %s\n", intl_get_error_message() );
exit( 1 );
}
?>
