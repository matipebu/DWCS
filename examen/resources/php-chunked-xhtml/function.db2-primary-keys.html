db2_primary_keys

(PECL ibm_db2 >= 1.0.0)

db2_primary_keys — Devolver un resultado con las llaves primarias de una tabla

Descripción

db2_primary_keys(
    resource $connection,
    string $qualifier,
    string $schema,
    string $table-name
): resource

Devuelve un resultado con las llaves primarias de una tabla.

Parámetros

connection

Una conexión válida a la base de datos IBM DB2, Cloudscape, o Apache Derby.

qualifier

Un índice para bases de datos DB2 ejecutándose en servidores OS/390 o z/OS. Para otras bases de datos se debe pasar null o una cadena vacía.

schema

El esquema que contiene las tablas. Si schema es null, db2_primary_keys() revisa el esquema de la conexión actual.

table-name

El nombre de la tabla.

Valores devueltos

Devuelve un recurso de sentencia con el resultado que contiene las llaves primarias de la tabla especificada. El resultado se compone de las siguientes columnas:

Columna Descripción
TABLE_CAT Nombre del catálogo de la tabla que contiene la llave primaria. Este valor es nulo si la tabla no tiene catálogos.
TABLE_SCHEM Nombre del esquema de la tabla que contiene la llave primaria.
TABLE_NAME Nombre de la tabla que contiene la llave primaria.
COLUMN_NAME Nombre de la columna que contiene la llave primaria.
KEY_SEQ Posición (indexada a partir de uno) de la columna en la llave primaria.
PK_NAME Nombre de la llave primaria.
