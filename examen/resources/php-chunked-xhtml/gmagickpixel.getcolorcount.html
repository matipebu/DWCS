GmagickPixel::getcolorcount

(PECL gmagick >= Unknown)

GmagickPixel::getcolorcount — Devuelve la cuenta de color asociada a este color

Descripción

public GmagickPixel::getcolorcount(): int

Devuelve la cuenta de color asociada a este color

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la cuenta de color como un valor de tipo integer en caso de éxito, lanza una excepción de tipo GmagickPixelException en caso de error.
