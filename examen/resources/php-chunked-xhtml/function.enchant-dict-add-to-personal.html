enchant_dict_add_to_personal

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_dict_add_to_personal — Añadir una palabra a una lista personal de palabras

Descripción

enchant_dict_add_to_personal(resource $dict, string $word): void

Añade una palabra a una lista personal de palabras del diccionario dado.

Parámetros

dict

Recurso de diccionario

word

La palabra a añadir

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplo #1 Añadir una palabra a un PWL

<?php

$filename = './my_word_list.pwl';
$word = 'Supercalifragilisticexpialidocious';

$broker = enchant_broker_init();
$dict = enchant_broker_request_pwl_dict($broker, $filename);

enchant_dict_add_to_personal($dict, $word);

enchant_broker_free($broker);

?>
