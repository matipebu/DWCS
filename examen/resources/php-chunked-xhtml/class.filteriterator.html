La clase FilterIterator

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Introducción

Este iterador abstracto filtra valores no deseados. Esta clase debería ser ampliada para implementar filtros personalizados. FilterIterator::accept() debe ser implementado en la subclase.
