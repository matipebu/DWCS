AppendIterator::next

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

AppendIterator::next — Desplazarse al siguiente elemento

Descripción

public AppendIterator::next(): void

Se desplaza al siguiente elemento. Esto significa a otro iterador por lo que rebobina ese iterador.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
