ReflectionProperty::setAccessible

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

ReflectionProperty::setAccessible — Hace accesible una propiedad

Descripción

public ReflectionProperty::setAccessible(bool $accessible): void

Hace accesible una propiedad. Por ejemplo, podrá hacer que las propiedades protegidas o privadas sean accesibles.

Parámetros

accessible

true para permitir su acceso, o false.

Valores devueltos

No devuelve ningún valor.
