Introducción

Esta extensión proporciona acceso a la implementación de referencia de CommonMark, una versión racionalizada de la sintaxis de Markdown con una especificación.

Parsing:

La extensión CommonMark proporciona una sencilla API de análisis sintáctico:

CommonMark\Parse(string $content, int $options = ?): CommonMark\Node

Rendering:

La extensión CommonMark proporciona una sencilla API de renderizado que soporta múltiples formatos:

CommonMark\Render(CommonMark\Node $node, int $options = ?, int $width = ?): string
CommonMark\Render\HTML(CommonMark\Node $node, int $options = ?): string
CommonMark\Render\XML(CommonMark\Node $node, int $options = ?): string
CommonMark\Render\Man(CommonMark\Node $node, int $options = ?, int $width = ?): string
CommonMark\Render\Latex(CommonMark\Node $node, int $options = ?, int $width = ?): string

AST:

La extensión CommonMark implementa la visitación de los objetos CommonMark\NNode:

public CommonMark\Node::accept(CommonMark\Interfaces\IVisitor $visitor): void

CQL:

La extensión CommonMark proporciona una interfaz para CQL, CommonMark Query Language:

public CommonMark\CQL::__construct(string $query)
