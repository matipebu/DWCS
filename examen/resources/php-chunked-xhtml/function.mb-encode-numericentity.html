mb_encode_numericentity

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

mb_encode_numericentity — Convierte caracteres a referencias de string numéricas de HTML

Descripción

mb_encode_numericentity(
    string $str,
    array $convmap,
    string $encoding = mb_internal_encoding(),
    bool $is_hex = FALSE
): string

Convierte los códigos de caracteres especificados en el string str desde un código de caracter a un referencia de carácter numérico de HTML.

Parámetros

str

El string a ser codificado.

convmap

convmap es un array que especifica el área de código a convertir.

encoding

El parámetro encoding es la codificación de caracteres. Si es omitido, será usado el valor de la codificación de caracteres interna.

is_hex

Valores devueltos

El string convertido.

Historial de cambios

Versión Descripción
5.4.0 Añadido el parámetro is_hex.

Ejemplos

Ejemplo #1 Ejemplo de convmap

<?php
$convmap = array (
int start_code1, int end_code1, int offset1, int mask1,
int start_code2, int end_code2, int offset2, int mask2,
........
int start_codeN, int end_codeN, int offsetN, int maskN );
// Especificar el valor Unicode para start_codeN y end_codeN
// Añadir offsetN al valor y tomar el operador 'AND' con maskN,
// luego convertir el valor a una referencia de string numérica.
?>

Ejemplos

Ejemplo #2 Ejemplo de mb_encode_numericentity()

<?php
/* Convertir la parte izquierda de ISO-8859-1 a referncias de caracteres numéricas de HTML */
$convmap = array(0x80, 0xff, 0, 0xff);
$str = mb_encode_numericentity($str, $convmap, "ISO-8859-1");

/* Convertir código SJIS-win definido por el usuario del bloque 95-104 to a referencias
de string numéricas */
$convmap = array(
0xe000, 0xe03e, 0x1040, 0xffff,
0xe03f, 0xe0bb, 0x1041, 0xffff,
0xe0bc, 0xe0fa, 0x1084, 0xffff,
0xe0fb, 0xe177, 0x1085, 0xffff,
0xe178, 0xe1b6, 0x10c8, 0xffff,
0xe1b7, 0xe233, 0x10c9, 0xffff,
0xe234, 0xe272, 0x110c, 0xffff,
0xe273, 0xe2ef, 0x110d, 0xffff,
0xe2f0, 0xe32e, 0x1150, 0xffff,
0xe32f, 0xe3ab, 0x1151, 0xffff );
$str = mb_encode_numericentity($str, $convmap, "sjis-win");
?>
