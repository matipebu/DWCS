RecursiveTreeIterator::endIteration

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

RecursiveTreeIterator::endIteration — Fin de la iteración

Descripción

public RecursiveTreeIterator::endIteration(): void

Se llama cuando la iteración termina (o cuando RecursiveTreeIterator::valid() retorne false)

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.
