bindtextdomain

(PHP 4, PHP 5, PHP 7, PHP 8)

bindtextdomain — Establece u obtiene la ruta de un dominio

Descripción

bindtextdomain(string $domain, ?string $directory): string|false

La función bindtextdomain() establece u obtiene la ruta de acceso de un dominio.

Parámetros

domain

El dominio

directory

La ruta del directorio. Un string vacío significa el directorio actual. Si null, se devuelve el directorio actualmente establecido.

Valores devueltos

La ruta completa para el domain actual establecido, o false en caso de error.

Historial de cambios

Versión Descripción
8.0.3 directory es anulable ahora. Anteriormente, no era posible recuperar el directorio actual establecido.

Ejemplos

Ejemplo #1 Ejemplo de bindtextdomain()

<?php

$domain = 'myapp';
echo bindtextdomain($domain, '/usr/share/myapp/locale');

?>

El resultado del ejemplo sería:

/usr/share/myapp/locale

Notas

Nota:

La información de bindtextdomain() e mantiene por proceso, no por hilo.
