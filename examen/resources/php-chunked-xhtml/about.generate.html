Cómo generamos los formatos

Este manual está escrito en XML usando » DocBook XML DTD, y » PhD (El render [D]ocBook basado en[PH]P ) para el mantenimiento y formato.

Usamos XML como base, para generar diferentes formatos, centralizando un solo código fuente para todos los diferentes formatos del manual. Utilizamos la herramienta » PhD para crear las páginas del manual online. También usamos » Microsoft HTML Help Workshop para generar el formato de ayuda Windows HTML y por supuesto usamos PHP para crear algunas conversiones adicionales y formateado.

El manual de PHP es generado en varios idiomas y formatos, ver » https://www.php.net/docs.php para más información. El código fuente XML puede ser descargado desde Git y visualizado en » https://github.com/php/doc-en.
