IntlCalendar::getKeywordValuesForLocale

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getKeywordValuesForLocale — Obtener un conjunto de valores de palabras clave de zonas

Descripción

Estilo orientado a objetos

public static IntlCalendar::getKeywordValuesForLocale(string $key, string $locale, boolean $commonlyUsed): Iterator

Estilo por procedimientos

intlcal_get_keyword_values_for_locale(string $key, string $locale, boolean $commonlyUsed): Iterator

Para una clave de una configuración regional dada, obtiene el conjunto de valores que resultarían en un comoportamiento diferente. Por ahora, solamente está soportada la palabra clave 'calendar'.

Esta función requiere ICU 4.2 o posterior.

Parámetros

key

La palabra clave de la configuración regional de la que se quieren consultar valores relevantes. Solamente está soprtada 'calendar'.

locale

La configuración regional en la que adjuntar el par clave/valor.

commonlyUsed

Si mostrar solamente valores usados comúnmente para la configuración regional especificada.

Valores devueltos

Un iterador que genera string con los valores de la palabra clave de la configuración regional o false en caso de error.

Ejemplos

Ejemplo #1 IntlCalendar::getKeyworkValuesForLocale()

<?php
print_r(
iterator_to_array(
IntlCalendar::getKeywordValuesForLocale(
'calendar', 'fa_IR', true)));
print_r(
iterator_to_array(
IntlCalendar::getKeywordValuesForLocale(
'calendar', 'fa_IR', false)));
