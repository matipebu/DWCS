Yac::set

(PECL yac >= 1.0.0)

Yac::set — Guardar en el caché

Descripción

public Yac::set(string $keys, mixed $value, int $ttl = 0): bool
public Yac::add(array $key_vals): bool

Añade un elemento a la caché, si la clave ya existe, se sobreescribe.

Parámetros

keys

clave string

value

valor mixed, Todo tipo de valor php podría ser almacenado excepto recurso

ttl

tiempo de expiración

Valores devueltos

el valor de sí mismo
