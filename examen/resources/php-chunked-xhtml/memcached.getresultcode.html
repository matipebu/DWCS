Memcached::getResultCode

(PECL memcached >= 0.1.0)

Memcached::getResultCode — Devuelve el código resultante de la última operación

Descripción

public Memcached::getResultCode(): int

Memcached::getResultCode() devuelve una de las constantes Memcached::RES_*, la cual es el resultado del último método de Memcached ejecutado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El código resultante de la última operación de Memcached.

Ejemplos

Ejemplo #1 Ejemplo de Memcached::getResultCode()

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->add('foo', 'bar');
if ($m->getResultCode() == Memcached::RES_NOTSTORED) {
/* ... */
}
?>
