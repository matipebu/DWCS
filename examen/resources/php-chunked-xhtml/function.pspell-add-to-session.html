pspell_add_to_session

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

pspell_add_to_session — Añadir la palabra a la lista de palabras en la sesión actual

Descripción

pspell_add_to_session(int $dictionary_link, string $word): bool

pspell_add_to_session() añade una palabra a la lista de palabras asociadas con la sesión actual. Es muy parecida a pspell_add_to_personal()

Parámetros

dictionary_link

word

La palabra añadida.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
