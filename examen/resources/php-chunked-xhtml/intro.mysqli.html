Introducción

La extensión mysqli (mysql improved) permite acceder a la funcionalidad proporcionada por MySQL 4.1 y posterior. Se puede encontrar más información sobre el servidor de base de datos MySQL en » http://www.mysql.com/

Se puede encontrar información general del software disponible para usar MySQL desde PHP en la Información general

La documentación de MySQL se puede encontrar en » http://dev.mysql.com/doc/.

Esta documentación incluye partes del manual de MySQL con permiso de Oracle Corporation.

Los ejemplos emplean las bases de datos » world o » sakila, las cuales están a libre disposición.
