Memcached::getResultMessage

(PECL memcached >= 1.0.0)

Memcached::getResultMessage — Devolver un mensaje que describe el resultado de la última operación

Descripción

public Memcached::getResultMessage(): string

Memcached::getResultMessage() devuelve un string que describe el código resultante del último método de Memcached ejecutado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un mensaje que describe el resultado de la última operación Memcached.

Ejemplos

Ejemplo #1 Ejemplo de Memcached::getResultMessage()

<?php
$m = new Memcached();
$m->addServer('localhost', 11211);

$m->add('foo', 'bar'); // la primera vez debería funcionar
$m->add('foo', 'bar');
echo $m->getResultMessage(),"\n";
?>

El resultado del ejemplo sería:

NOT STORED
