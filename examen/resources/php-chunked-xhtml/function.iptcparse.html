iptcparse

(PHP 4, PHP 5, PHP 7, PHP 8)

iptcparse — Conviertir un bloque IPTC binario en simples etiquetas

Descripción

iptcparse(string $iptcblock): array

Convierte un bloque » IPTC en sus etiquetas.

Parámetros

iptcblock

Un bloque IPTC binario.

Valores devueltos

Devuelve una matriz usando el marcador de etiquetas como un índice, y el valor como el valor. Devuelve false si se produjo un error o si no se encontró información IPTC.

Ejemplos

Ejemplo #1 iptcparse() usado junto con getimagesize()

<?php
$tamaño = getimagesize('./prueba.jpg', $info);
if(isset($info['APP13']))
{
$iptc = iptcparse($info['APP13']);
var_dump($iptc);
}
?>

Notas

Nota:

Esta función no requiere la librería GD image.
