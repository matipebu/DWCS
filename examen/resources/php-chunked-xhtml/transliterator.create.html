Transliterator::create

transliterator_create

(PHP 5 >= 5.4.0, PHP 7, PHP 8, PECL intl >= 2.0.0)

Transliterator::create -- transliterator_create — Crear un transliterador

Descripción

Estilo orientado a objetos

public static Transliterator::create(string $id, int $direction = ?): Transliterator

Estilo por procedimientos

transliterator_create(string $id, int $direction = ?): Transliterator

Abrir un transliterador mediante su ID.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

id

El identificador.

direction

La dirección, por defecto es >Transliterator::FORWARD. También se puede establecer a Transliterator::REVERSE.

Valores devueltos

Devuelve un objeto de la clase Transliterator en caso de éxito, o null en caso de error.
