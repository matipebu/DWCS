pg_version

(PHP 5, PHP 7, PHP 8)

pg_version — Devuelve un array con el cliente, protocolo y versión del servidor (si está disponible)

Descripción

pg_version(resource $connection = ?): array

pg_version() Devuelve un array con el cliente, protocolo y la versión del servidor. Las versiones del protocolo y el servidor sólo están disponibles si PHP se compiló con PostgreSQL 7.4 o posterior.

Para obtener información más detallada del servidor, utilice pg_parameter_status().

Parámetros

connection

Recurso de conexión de base de datos PostgreSQL o PostgreSQL database connection resource. Cuando el parametro connection no esta presente, se utiliza la conexión por defecto. La conexión por defecto es la ultima conexión hecha por pg_connect() o pg_pconnect().

Valores devueltos

Devuelve un array con el client, protocol y server con sus claves y valores (si está disponibles). Devuelve false en caso de error o una conexión no válida.

Ejemplos

Ejemplo #1 Ejemplo de pg_version()

<?php
$dbconn = pg_connect("host=localhost port=5432 dbname=mary")
or die("No e ha podido conectar");

$v = pg_version($dbconn);

echo $v['client'];
?>

El resultado del ejemplo sería:

7.4
