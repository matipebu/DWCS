is_resource

(PHP 4, PHP 5, PHP 7, PHP 8)

is_resource — Comprueba si una variable es un recurso

Descripción

is_resource(mixed $var): bool

Comprueba si la variable es un recurso.

Parámetros

var

La variable que se evalúa.

Valores devueltos

Devuelve true si var es un resource, false de lo contrario.

Ejemplos

Ejemplo #1 Ejemplo de is_resource()

<?php

$db_link = @mysql_connect('localhost', 'mysql_user', 'mysql_pass');
if (!is_resource($db_link)) {
die('No pudo realizarse la conexión : ' . mysql_error());
}

?>
