RRDUpdater::__construct

(PECL rrd >= 0.9.0)

RRDUpdater::__construct — Crea una nueva instancia RRDUpdater

Descripción

public RRDUpdater::__construct(string $path)

Crea una nueva instancia RRDUpdater. Esta instancia es responsable de actualizar el archivo de base de datos RRD.

Parámetros

path

Ruta de archivo del sistema para el archivo de base de datos RRD, que se actualizará.

Valores devueltos

No devuelve ningún valor.
