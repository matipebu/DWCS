RecursiveTreeIterator::beginIteration

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

RecursiveTreeIterator::beginIteration — Inicio de la iteración

Descripción

public RecursiveTreeIterator::beginIteration(): RecursiveIterator

Se llama cuando comienza la iteración (después de llamar a RecursiveTreeIterator::rewind()).

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un RecursiveIterator.
