OCICollection::trim

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCICollection::trim — Elimina elementos del final de la colección

Descripción

public OCICollection::trim(int $num): bool

Elimina el número de elementos dado por num del final de la colección.

Parámetros

num

El número de elementos a eliminar.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 La clase OCI-Collection ha sido renombrada a OCICollection para ajustarse a los estándares de nomenclatura de PHP.
