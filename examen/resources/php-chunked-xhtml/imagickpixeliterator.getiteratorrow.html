ImagickPixelIterator::getIteratorRow

(PECL imagick 2, PECL imagick 3)

ImagickPixelIterator::getIteratorRow — Devuelve la fila actual del iterador de píxel

Descripción

ImagickPixelIterator::getIteratorRow(): int
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve la fila actual del iterador de píxel.

Valores devueltos

Devuelve el índice de tipo integer de la fila, lanzando una excepción ImagickPixelIteratorException si se produjo un error.
