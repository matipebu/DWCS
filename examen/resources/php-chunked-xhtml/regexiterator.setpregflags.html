RegexIterator::setPregFlags

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

RegexIterator::setPregFlags — Establece las flags de expresión regular

Descripción

public RegexIterator::setPregFlags(int $pregFlags): void

Establece las flags de expresión regular.

Parámetros

pregFlags

Las flags de expresión regular. Véase RegexIterator::__construct() para una lista de todas las flags disponibles.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de RegexIterator::setPregFlags()

Crea un nuevo RegexIterator que filtra todas las entradas donde la clave del array empiece con 'test'.

<?php
$test = array ('test 1', 'otro test', 'test 123');

$arrayIterator = new ArrayIterator($test);
$regexIterator = new RegexIterator($arrayIterator, '/^test/', RegexIterator::GET_MATCH);

$regexIterator->setPregFlags(PREG_OFFSET_CAPTURE);

foreach ($regexIterator as $clave => $valor) {
var_dump($valor);
}
?>

El resultado del ejemplo sería algo similar a:

array(1) {
  [0]=>
  array(2) {
    [0]=>
    string(4) "test"
    [1]=>
    int(0)
  }
}

    [0]=>

  }
}
