defined

(PHP 4, PHP 5, PHP 7, PHP 8)

defined — Comprueba si existe una constante con nombre dada

Descripción

defined(string $name): bool

Comprueba si existe la constante dada y está definida.

Nota:

Si quiere ver si existe una variable, use isset(), ya que defined() sólo se aplica a constantes. Si quiere ver si existe una función, use function_exists().

Parámetros

name

El nombre de la constante.

Valores devueltos

Devuelve true si la constante con nombre dada por name ha sido definida, si no devuelve false.

Ejemplos

Ejemplo #1 Comprobar constantes

<?php
/* Observe el uso de las comillas, esto es importante. Este ejemplo comprueba
* si e´string 'PRUEBA' es un nombre de una constante con nombre PRUEBA */
if (defined('PRUEBA')) {
echo PRUEBA;
}
?>
