trader_mult

(PECL trader >= 0.2.0)

trader_mult — Multiplicación aritmética de vectores

Descripción

trader_mult(array $real0, array $real1): array

Calcula el producto escalar de real0 con real1 y devuelve el vector resultante.

Parámetros

real0

Array de valores reales.

real1

Array de valores reales.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
