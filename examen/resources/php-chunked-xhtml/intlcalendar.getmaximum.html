IntlCalendar::getMaximum

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getMaximum — Obtener el valor máximo global de un campo

Descripción

Estilo orientado a objetos

public IntlCalendar::getMaximum(int $field): int

Estilo por procedimientos

intlcal_get_maximum(IntlCalendar $cal, int $field): int

Obtiene el máximo global de un campo, es este calendario específico. Este valor es mayor o igual que el devuelto por IntlCalendar::getActualMaximum(), el cual es a su vez mayor o igual que el devuelto por IntlCalendar::getLeastMaximum().

Parámetros

cal

El recurso IntlCalendar.

field

Una de las constantes del campo fecha/hora de IntlCalendar. Son valores de tipo integer entre 0 y IntlCalendar::FIELD_COUNT.

Valores devueltos

Un int que representa un valor de campo en la unidad del campo o false en caso de error.
