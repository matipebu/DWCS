IntlChar::getUnicodeVersion

(PHP 7, PHP 8)

IntlChar::getUnicodeVersion — Obtener la versión de Unicode

Descripción

public static IntlChar::getUnicodeVersion(): array

Obtiene información de la versión de Unicode.

El array de la versión contiene la información de la versión del estándar de Unicode que está utilizando ICU actualmente. Por ejemplo, la versión 3.1.1 de Unicode se representa como un array con los valores [3, 1, 1, 0].

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array que contiene el número de versión de Unicode.

Ejemplos

Ejemplo #1 Probar diferentes propiedades

<?php
var_dump(IntlChar::getUnicodeVersion());
?>

El resultado del ejemplo sería:

array(4) {
  [0]=>
  int(7)
  [1]=>
  int(0)
  [2]=>
  int(0)
  [3]=>
  int(0)
}
