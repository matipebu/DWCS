imagecolordeallocate

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecolordeallocate — Desasignar un color de una imagen

Descripción

imagecolordeallocate(resource $image, int $color): bool

Desasigna un color previamente asignado con imagecolorallocate() o imagecolorallocatealpha().

Parámetros

image

Un recurso image, es devuelto por una de las funciones de creación de imágenes, como imagecreatetruecolor().

color

El identificador de color.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Usar imagecolordeallocate()

<?php
$blanco = imagecolorallocate($im, 255, 255, 255);
imagecolordeallocate($im, $blanco);
?>
