ImagickDraw::getFont

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getFont — Devuelve la fuente

Descripción

ImagickDraw::getFont(): string
Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Devuelve un string especificando la fuente usada cuando se escribe texto.

Valores devueltos

Devuelve un string si se tuvo éxito o false si no está establecida la fuente.
