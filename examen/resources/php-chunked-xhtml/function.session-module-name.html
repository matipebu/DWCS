session_module_name

(PHP 4, PHP 5, PHP 7, PHP 8)

session_module_name — Obtiene y/o establece el módulo de sesión actual

Descripción

session_module_name(string $module = ?): string

session_module_name() obtiene el nombre del módulo de sesión actual.

Parámetros

module

Si se especifica module, ese módulo será usado en su lugar.

Valores devueltos

Devuelve el nombre del módulo de sesión actual.
