Imagick::setImageAlphaChannel

(PECL imagick 2 >= 2.1.0, PECL imagick 3)

Imagick::setImageAlphaChannel — Establece el canal alfa de la imagen

Descripción

public Imagick::setImageAlphaChannel(int $mode): bool

Activa o desactiva el canal alfa de la imagen. El parámetro mode es una de las constantes Imagick::ALPHACHANNEL_*. Este método está disponible si Imagick ha sido compilado con la versión 6.3.8 o superior de ImageMagick.

Parámetros

mode

Una de las constantes Imagick::ALPHACHANNEL_*

Valores devueltos

Devuelve true en caso de éxito.

Errores/Excepciones

Lanza ImagickException en caso de error.
