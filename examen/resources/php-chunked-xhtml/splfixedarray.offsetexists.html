SplFixedArray::offsetExists

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplFixedArray::offsetExists — Devuelve si el índice solicitado existe

Descripción

public SplFixedArray::offsetExists(int $index): bool

Comprueba si el índice index solicitado existe.

Parámetros

index

El índice a ser comprobado.

Valores devueltos

true si el indice index existe, en caso contrario false.
