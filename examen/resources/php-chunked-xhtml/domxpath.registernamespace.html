DOMXPath::registerNamespace

(PHP 5, PHP 7, PHP 8)

DOMXPath::registerNamespace — Registra el espacio de nombres con el objeto DOMXPath

Descripción

public DOMXPath::registerNamespace(string $prefix, string $namespace): bool

Registra namespace y prefix con el objeto DOMXPath.

Parámetros

prefix

El prefijo.

namespace

La URI del espacio de nombres.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
