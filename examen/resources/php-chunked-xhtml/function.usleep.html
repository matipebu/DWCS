usleep

(PHP 4, PHP 5, PHP 7, PHP 8)

usleep — Retrasar la ejecución en microsegundos

Descripción

usleep(int $micro_seconds): void

Retrasa la ejecución del programa durante el número de microsegundos dados.

Parámetros

micro_seconds

Tiempo de parada en microsegundos. Un microsegundo el la millonésima parte de un segundo.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de usleep()

<?php

// Hora actual
echo date('h:i:s') . "\n";

// Esperar 2 segundos
usleep(2000000);

// ¡De vuelta!
echo date('h:i:s') . "\n";

?>

El resultado del ejemplo sería:

11:13:28
11:13:30
