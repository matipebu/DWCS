fpm_get_status

(PHP 7 >= 7.3, PHP 8)

fpm_get_status — Devuelve el estado actual del pool de FPM

Descripción

fpm_get_status(): array

Esta función devuelve el estado actual del pool de FPM como una array asociativo. Siempre devuelve el estado completo, incluyendo la información de estado por proceso. Vea la guía de la página de estado de FPM para más detalles.

Tenga en cuenta que esta función sólo se definirá si se utiliza FPM para servir el script.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Array asociativo que contiene el estado completo del pool de FPM.
