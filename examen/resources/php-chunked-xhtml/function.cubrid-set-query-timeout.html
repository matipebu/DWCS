cubrid_set_query_timeout

(PECL CUBRID >= 8.4.1)

cubrid_set_query_timeout — Establecer el tiempo de espera de la ejecución de consultas

Descripción

cubrid_set_query_timeout(resource $req_identifier, int $timeout): bool

La función cubrid_set_query_timeout() se usa para establecer el tiempo de espera de la ejecución de consultas.

Parámetros

req_identifier

Identificador de petición.

timeout

El tiempo de espera, unidades de milisegundos

Valores devueltos

true, cuando el proceso tiene éxito.

false, cuando el proceso no tiene éxito.
