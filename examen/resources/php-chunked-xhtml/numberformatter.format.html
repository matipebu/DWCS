NumberFormatter::format

numfmt_format

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

NumberFormatter::format -- numfmt_format — Dar formato a un número

Descripción

Estilo orientado a objetos

public NumberFormatter::format(number $value, int $type = ?): string

Estilo por procedimientos

numfmt_format(NumberFormatter $fmt, number $value, int $type = ?): string

Dar formato a un valor numérico según las reglas del formateador.

Parámetros

fmt

Objeto NumberFormatter.

value

El valor de formato. Puede ser integer o float, otros valores se convierten en un valor numérico.

type

The tipo de formato a usar.

Valores devueltos

Devuelve un string con el valor formateado, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de numfmt_format()

<?php
$fmt = numfmt_create( 'de_DE', NumberFormatter::DECIMAL );
$data = numfmt_format($fmt, 1234567.891234567890000);
if(intl_is_failure(numfmt_format($fmt))) {
report_error("Formatter error");
}
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$fmt = new NumberFormatter( 'de_DE', NumberFormatter::DECIMAL );
$fmt->format(1234567.891234567890000);
if(intl_is_failure($fmt->getErrorCode())) {
report_error("Formatter error");
}
?>

El resultado del ejemplo sería:

1.234.567,891
