IntlCalendar::getAvailableLocales

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::getAvailableLocales — Obtener un array con las zonas para las que existen datos

Descripción

Estilo orientado a objetos

public static IntlCalendar::getAvailableLocales(): array

Estilo por procedimientos

intlcal_get_available_locales(): array

Proporciona la lista de zona para las que están instaladas los calendarios. A partir de ICU 51, esta es la lista de todas las zonas instaladas de ICU.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Un array de string, uno para cada zona.

Ejemplos

Ejemplo #1 IntlCalendar::getAvailableLocales()()

<?php
print_r(IntlCalendar::getAvailableLocales());
