IntlDateFormatter::localtime

datefmt_localtime

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

IntlDateFormatter::localtime -- datefmt_localtime — Convertir una cadena en un valor de marca de tiempo basado en un campo

Descripción

Estilo orientado a objetos

public IntlDateFormatter::localtime(string $value, int &$position = ?): array

Estilo por procedimientos

datefmt_localtime(IntlDateFormatter $fmt, string $value, int &$position = ?): array

Convierte la cadena $value a un valor de instante basado en un campo (un array de varios campos),, empezando en la posición dada por $parse_pos y consumiendo tanto del valor de entrada como sea posible.

Parámetros

fmt

El recurso del formateador.

value

La cadena que convertir a un instante

position

La posición a la cual empezar la conversión de $value (basado en cero). Si no ocurre ningún error antes de que $value sea consumido, $parse_pos contendrá -1, de otro modo contendrá la posición en la que finaliza la conversión (ocurrido el error). Si $parse_pos > strlen($value), la conversión falla automáticamente.

Valores devueltos

Un array Localtime compatible array of integers : contains 24 hour clock value in tm_hour field

Ejemplos

Ejemplo #1 Ejemplo de datefmt_localtime()

<?php

$fmt = datefmt_create(
'en_US',
IntlDateFormatter::FULL,
IntlDateFormatter::FULL,
'America/Los_Angeles',
IntlDateFormatter::GREGORIAN
);
$array = datefmt_localtime($fmt, 'Wednesday, December 31, 1969 4:00:00 PM PT', 0);
echo 'La primera salida convertida es ';
if ($array) {
foreach ($array as $clave => $valor) {
echo "$clave : $valor , ";
}
}

?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$fmt = new IntlDateFormatter(
'en_US',
IntlDateFormatter::FULL,
IntlDateFormatter::FULL,
'America/Los_Angeles',
IntlDateFormatter::GREGORIAN
);
$array = $fmt->localtime('Wednesday, December 31, 1969 4:00:00 PM PT', 0);
echo 'La primera salida convertida es ';
if ($array) {
foreach ($array as $clave => $valor) {
echo "$clave : $valor , ";
}
}

?>

El resultado del ejemplo sería:

La primera salida convertida es tm_sec : 0 , tm_min : 0 , tm_hour : 16 , tm_year : 1969 , 
tm_mday : 31 , tm_wday : 4 , tm_yday : 365 , tm_mon : 11 , tm_isdst : 0 ,
