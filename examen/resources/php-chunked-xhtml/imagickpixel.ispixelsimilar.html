ImagickPixel::isPixelSimilar

(PECL imagick 3 >= 3.3.0)

ImagickPixel::isPixelSimilar — Comprobar la distancia entre este color y otro

Descripción

ImagickPixel::isPixelSimilar(ImagickPixel $color, float $fuzz): bool

Comprueba la distancia entre el color descrito por este objeto ImagickPixel y el objeto proporcionado, trazando sus valores RGB en el cubo de color. Si la distancia entre dos puntos es menor que the valor del parámetro fuzz dado, los colores son similares. Este método reemplaza a ImagickPixel::isSimilar() y normaliza de forma correcta el valor de fuzz a QuantumRange de ImageMagick.

Parámetros

color

El objeto ImagickPixel a comparar con este objeto.

fuzz

La distancia máxima dentro de la que considerar que estos colores son iguales. El máximo teórico para este valor es la reíz cuadrada de tres (1.732).

Valores devueltos

Devuelve true en caso de éxito.
