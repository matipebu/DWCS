OCICollection::max

(PHP 5, PHP 7, PHP 8, PECL OCI8 >= 1.1.0)

OCICollection::max — Devuelve el número máximo de elementos de la colección

Descripción

public OCICollection::max(): int|false

Devuelve el número máximo de elementos de la colección.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el número máximo como un entero, o false en caso de error.

Si el valor devuelto es 0, entonces el número de elementos no está limitado.

Historial de cambios

Versión Descripción
8.0.0, PECL OCI8 3.0.0 La clase OCI-Collection ha sido renombrada a OCICollection para ajustarse a los estándares de nomenclatura de PHP.
