ReflectionProperty::isStatic

(PHP 5, PHP 7, PHP 8)

ReflectionProperty::isStatic — Comprueba si la propiedad es estática

Descripción

public ReflectionProperty::isStatic(): bool

Comprueba si la propiedad es estática.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

true si la propiedad es estática, o false en caso contrario.
