mysqli::dump_debug_info

mysqli_dump_debug_info

(PHP 5, PHP 7, PHP 8)

mysqli::dump_debug_info -- mysqli_dump_debug_info — Volcado de información de depuración en el registro

Descripción

Estilo orientado a objetos

mysqli::dump_debug_info(): bool

Estilo por procedimientos

mysqli_dump_debug_info(mysqli $link): bool

Esta función está diseñada para ser ejecutada por un usuario con el privilegio SUPER y se utiliza para volcar la información de depuración en el registro para el servidor MySQL relacionado con la conexión.

Parámetros

link

Sólo estilo por procediminetos: Un identificador de enlace devuelto por mysqli_connect() o mysqli_init()

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
