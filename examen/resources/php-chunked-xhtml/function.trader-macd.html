trader_macd

(PECL trader >= 0.2.0)

trader_macd — Convergencia/divergencia de la media móvil

Descripción

trader_macd(
    array $real,
    int $fastPeriod = ?,
    int $slowPeriod = ?,
    int $signalPeriod = ?
): array

Parámetros

real

Array de valores reales.

fastPeriod

Número del periodo para el MA rápido. Rango válido desde 2 hasta 100000.

slowPeriod

Número del periodo para el MA lento. Rango válido desde 2 hasta 100000.

signalPeriod

Suavizado para la línea de señal (nb de periodo). Rango válido desde 1 hasta 100000.

Valores devueltos

Devuelve un array con los datos calculados o false en caso de error.
