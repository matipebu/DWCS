pg_untrace

(PHP 4 >= 4.0.1, PHP 5, PHP 7, PHP 8)

pg_untrace — Desactivar el rastreo de una conexión de PostgreSQL

Descripción

pg_untrace(resource $connection = ?): bool

Detener el seguimiento iniciado porpg_trace().

Parámetros

connection

Recurso de conexión a la base de datos PostgreSQL. Cuando no hay connection, se utiliza la conexión predeterminada. La conexión predeterminada es la última conexión realizada por pg_connect() o pg_pconnect().

Valores devueltos

Siempre retorna true.

Ejemplos

Ejemplo #1 Ejemplo de pg_untrace()

<?php
$pgsql_conn = pg_connect("dbname=mark host=localhost");

if ($pgsql_conn) {
pg_trace('/tmp/trace.log', 'w', $pgsql_conn);
pg_query("SELECT 1");
pg_untrace($pgsql_conn);
// Ahora el seguimiento de la comunicación de back-end está deshabilitado
} else {
print pg_last_error($pgsql_conn);
exit;
}
?>
