dio_close

(PHP 4 >= 4.2.0, PHP 5 < 5.1.0)

dio_close — Cierra el descriptor de fichero fd

Descripción

dio_close(resource $fd): void

La función dio_close() cierra el descriptor de fichero fd.

Parámetros

fd

Descriptor de fichero devuelto por dio_open().

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Cerrando un descriptor de fichero

<?php
$fd = dio_open('/dev/ttyS0', O_RDWR);

dio_close($fd);
?>
