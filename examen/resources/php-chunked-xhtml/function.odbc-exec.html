odbc_exec

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_exec — Preparar y ejecutar una sentencia SQL

Descripción

odbc_exec(resource $connection_id, string $query_string, int $flags = ?): resource

Envía una sentencia SQL al servidor de bases de datos.

Parámetros

connection_id

El conector identificador ODBC, ver odbc_connect() para más información.

query_string

La sentencia SQL.

flags

Este parámetro no se usa actualmente.

Valores devueltos

Devuelve un identificador de resultado de ODBC si el comando SQL se ejecutó con éxito, o false si se produjo un error.
