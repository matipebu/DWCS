Locale::getDefault

locale_get_default

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Locale::getDefault -- locale_get_default — Obtener el valor de la configuración regional predeterminada del identificador global 'default_locale' de INTL

Descripción

Estilo orientado a objetos

public static Locale::getDefault(): string

Estilo por procedimientos

locale_get_default(): string

Obtiene el valor de la configuración regional predeterminada. Cuando se inicializa PHP, este valor se establece al valor 'intl.default_locale' en php.ini si tal valor existe, o desde la función uloc_getDefault() de ICU.

Parámetros

Valores devueltos

La configuración regional actual en tiempo de ejecución

Ejemplos

Ejemplo #1 Ejemplo de locale_get_default()

<?php
ini_set('intl.default_locale', 'de-DE');
echo locale_get_default();
echo '; ';
locale_set_default('fr');
echo locale_get_default();
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
ini_set('intl.default_locale', 'de-DE');
echo Locale::getDefault();
echo '; ';
Locale::setDefault('fr');
echo Locale::getDefault();
?>

El resultado del ejemplo sería:

de-DE; fr
