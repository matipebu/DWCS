Memcache::getVersion

(PECL memcache >= 0.2.0)

Memcache::getVersion — Devuelve la versión del servidor

Descripción

Memcache::getVersion(): string

Memcache::getVersion() devuelve una cadena con el número de la versión del servidor. También puede usar la función memcache_get_version().

Valores devueltos

Devuelve una cadena con el número de la versión del servidor o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de Memcache::getVersion()

<?php

/* API OO */
$memcache = new Memcache;
$memcache->connect('memcache_host', 11211);
echo $memcache->getVersion();

/* API por procedimientos */
$memcache = memcache_connect('memcache_host', 11211);
echo memcache_get_version($memcache);

?>
