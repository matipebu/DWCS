ArrayObject::count

(PHP 5, PHP 7, PHP 8)

ArrayObject::count — Obtener el número de propiedades públicas en la clase ArrayObject

Descripción

public ArrayObject::count(): int

Obtiene el número de propiedades públicas en la clase ArrayObject.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El número de propiedades públicas en la clase ArrayObject.

Nota:

Cuando la clase ArrayObject se construye a partir de un array todas las propiedades son públicas.

Ejemplos

Ejemplo #1 Ejemplo de ArrayObject::count()

<?php
class Ejemplo {
public $public = 'prop:public';
private $prv = 'prop:private';
protected $prt = 'prop:protected';
}

$arrayobj = new ArrayObject(new Ejemplo());
var_dump($arrayobj->count());

$arrayobj = new ArrayObject(array('primero','segundo','tercero'));
var_dump($arrayobj->count());
?>

El resultado del ejemplo sería:

int(1)
int(3)
