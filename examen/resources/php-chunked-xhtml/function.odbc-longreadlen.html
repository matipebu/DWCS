odbc_longreadlen

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_longreadlen — Manejo de columnas LONG

Descripción

odbc_longreadlen(resource $result_id, int $length): bool

Habilita el manejo de columnas de tipo LONG y LONGVARBINARY.

Parámetros

result_id

El identificador de resultado.

length

El número de bytes devueltos a PHP está controlado por el parámetro length. Si está establecido a 0, la información de la columna Long se pasa a través del cliente.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Notas

Nota:

El manejo de columnas LONGVARBINARY también está afectado por odbc_binmode().
