gnupg_seterrormode

(PECL gnupg >= 0.6)

gnupg_seterrormode — Fija el modo para error_reporting

Descripción

gnupg_seterrormode(resource $identifier, int $errormode): void

Fija el modo para error_reporting.

Parámetros

identifier

El identificador gnupg, desde una llamada a gnupg_init() o gnupg.

errormode

El modo de error.

errormode toma el valor de una constante indicando qué tipo de error_reporting debe ser usado. Los posibles valores son: GNUPG_ERROR_WARNING, GNUPG_ERROR_EXCEPTION y GNUPG_ERROR_SILENT. Por defecto se usa GNUPG_ERROR_SILENT.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Pejemplo de gnupg_seterrormode() mediante funciones

<?php
$res = gnupg_init();
gnupg_seterrormode($res,GNUPG_ERROR_WARNING); // lanza un PHP-Warning en caso de error
?>

Ejemplo #2 Ejemplo de gnupg_seterrormode() mediante OO

<?php
$gpg = new gnupg();
$gpg -> seterrormode(gnupg::ERROR_EXCEPTION); // lanza una excepción en caso de error
?>
