Imagick::setPointSize

(PECL imagick 2 >= 2.1.0, PECL imagick 3)

Imagick::setPointSize — Establece el tamaño del punto

Descripción

Imagick::setPointSize(float $point_size): bool

Establece la propiedad tamaño del punto de objeto. Este método se puede usar para, por ejemplo, establecer el tamaño de la fuente para las leyendas: pseudo-formato. Este método está disponible si Imagick ha sido compilado con la versión 6.3.7 o superior de ImageMagick.

Parámetros

point_size

El tamaño del punto

Valores devueltos

Devuelve true en caso de éxito.

Ver también

  • Imagick::getPointSize() - Obtiene el tamaño de punto

Ejemplos

Ejemplo #1 Un ejemplo de Imagick::setPointSize()

Ejemplo usando Imagick::setPointSize

<?php
/* Crear el nuevo objeto imagick */
$im = new Imagick();

/* Establecer la fuente para el objeto */
$im->setFont("ejemplo.ttf");

/* Establecer el tamaño del punto */
$im->setPointSize(12);

/* Crear una leyenda nueva */
$im->newPseudoImage(100, 100, "caption:Hola");

/* Hacer algo con la imagen */
?>
