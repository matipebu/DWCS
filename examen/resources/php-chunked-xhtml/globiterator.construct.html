GlobIterator::__construct

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

GlobIterator::__construct — Construye un directorio usando glob

Descripción

public GlobIterator::__construct(string $pattern, int $flags = FilesystemIterator::KEY_AS_PATHNAME | FilesystemIterator::CURRENT_AS_FILEINFO)

Construye un nuevo iterador de directorio de una expresión glob.

Parámetros

pattern

Un patrón glob().

flags

flags de opción, las flags pueden ser máscaras de las constantes FilesystemIterator.

Ejemplos

Ejemplo #1 Ejemplo de GlobIterator

<?php
$iterator = new GlobIterator('*.dll', FilesystemIterator::KEY_AS_FILENAME);

if (!$iterator->count()) {
echo 'No matches';
} else {
$n = 0;

printf("Matched %d item(s)\r\n", $iterator->count());

foreach ($iterator as $item) {
printf("[%d] %s\r\n", ++$n, $iterator->key());
}
}
?>

El resultado del ejemplo sería algo similar a:

Matched 2 item(s)
[1] php5ts.dll
[2] php_gd2.dll
