cubrid_fetch_assoc

(PECL CUBRID >= 8.3.0)

cubrid_fetch_assoc — Devuelve un array asociativo que se corresponde con la fila obtenida

Descripción

cubrid_fetch_assoc(resource $result, int $type = ?): array

Esta función devuelve un array asociativo que se corresponde con la fila obtenida, y después mueve el puntero interno de datos hacia adelante, o devuelve FALSE cuando se ha alcanzado el final.

Parámetros

result

result proviene de una llamada a la función cubrid_execute()

type

El tipo solamente puede ser CUBRID_LOB. Este parámetro se usará solamente cuando sea necesario operar con objetos lob.

Valores devueltos

Un array asociativo, cuando el proceso tuvo éxito.

false, cuando no existen más filas; NULL, cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_fetch_assoc()

<?php
$conexión = cubrid_connect("localhost", 33000, "demodb");
$solicitud = cubrid_execute($conexión, "SELECT name,area,seats,address FROM stadium WHERE nation_code='GRE' AND seats > 10000");

printf("%-40s %-10s %-6s %-20s\n", "name", "area", "seats", "address");
while ($fila = cubrid_fetch_assoc($solicitud)) {
printf("%-40s %-10s %-6s %-20s\n",
$fila["name"], $fila["area"], $fila["seats"], $fila["address"]);
}

// si se operan con objetos lob, se puede usar cubrid_fetch_assoc($solicitud, CUBRID_LOB)

cubrid_close_request($solicitud);

cubrid_disconnect($conexión);
?>
