Memcached::__construct

(PECL memcached >= 0.1.0)

Memcached::__construct — Crear una instancia de Memcached

Descripción

public Memcached::__construct(string $persistent_id = ?)

Crea una instancia de Memcached que representa la conexión a los servidores de memcache.

Parámetros

persistent_id

Por defecto, las instancias de Memcached son destruidas al final de la petición. Para crear una instancia que persista entre peticiones, use persistent_id para especificar un ID único para la instancia. Todas las instancias creadas con el mismo persistent_id compartirán la misma conexión.

Valores devueltos

Un objeto Memcached.

Ejemplos

Ejemplo #1 Crear un objeto Memcached

<?php
/* Crear una instancia normal */
$m = new Memcached();
echo get_class($m);

/* Crear una instancia persistente */
$m2 = new Memcached('story_pool');
$m3 = new Memcached('story_pool');

/* Ahora $m2 y $m3 comparten la misma conexión */
?>
