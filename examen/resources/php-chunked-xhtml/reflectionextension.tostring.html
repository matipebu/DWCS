ReflectionExtension::__toString

(PHP 5, PHP 7, PHP 8)

ReflectionExtension::__toString — Convertir a string

Descripción

public ReflectionExtension::__toString(): string

Exporta una extensión reflexionada devolvíendola en forma de string. Es equivalente al método ReflectionExtension::export() estableciendo en el parámetro return el valor true.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve la extensión en forma de string, de la misma forma que haría ReflectionExtension::export().
