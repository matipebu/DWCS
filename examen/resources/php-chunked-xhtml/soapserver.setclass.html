SoapServer::setClass

(PHP 5, PHP 7, PHP 8)

SoapServer::setClass — Define la clase que controla las peticiones SOAP

Descripción

public SoapServer::setClass(string $class_name, mixed $... = ?): void

Exporta todos los métodos de una clase específica.

El objecto puede ser persistente durante un determinado período de sesiones PHP con el método SoapServer::setPersistence().

Parámetros

class_name

Nombre de la clase exportada.

...

Estos parámetros opcionales serán pasados al constructor de la clase por defecto durante la creación del objeto.

Valores devueltos

No devuelve ningún valor.
