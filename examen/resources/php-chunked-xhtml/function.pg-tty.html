pg_tty

(PHP 4, PHP 5, PHP 7, PHP 8)

pg_tty — Devolver el nombre TTY asociado con la conexión

Descripción

pg_tty(resource $connection = ?): string

pg_tty() Devuelve el nombre TTY al que se envía la salida de depuración del servidor en el recurso de connection de PostgreSQL.

Nota:

pg_tty() Es obsoleto, ya que el servidor ya no presta atención a la configuración de TTY, pero la función aún se mantiene para la compatibilidad hacia atrás.

Parámetros

connection

Recurso de conexión a la base de datos PostgreSQL. Cuando no hay connection, se utiliza la conexión predeterminada. La conexión predeterminada es la última conexión realizada por pg_connect() o pg_pconnect().

Valores devueltos

Un string que contiene el TTY de depuración de la connection, o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de pg_tty()

<?php
$pgsql_conn = pg_connect("dbname=mark host=localhost");

if ($pgsql_conn) {
print "Server debug TTY is: " . pg_tty($pgsql_conn) . "<br/>\n";
} else {
print pg_last_error($pgsql_conn);
exit;
}
?>
