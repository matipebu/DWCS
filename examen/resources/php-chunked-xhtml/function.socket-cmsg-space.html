socket_cmsg_space

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

socket_cmsg_space — Calcular el tamaño del búfer de mensajes

Descripción

socket_cmsg_space(int $level, int $type, int $n = 0): int

Calcula el tamaño del búfer que debería ser asignado para recibir datos subordinados.
