mysqli::stmt_init

mysqli_stmt_init

(PHP 5, PHP 7, PHP 8)

mysqli::stmt_init -- mysqli_stmt_init — Inicializa una sentencia y devuelve un objeto para usarlo con mysqli_stmt_prepare

Descripción

Estilo orientado a objetos

mysqli::stmt_init(): mysqli_stmt

Estilo por procedimientos

mysqli_stmt_init(mysqli $link): mysqli_stmt

Asigna e inicializa un objeto de sentencia apropiado para mysqli_stmt_prepare().

Nota:

Cualquier llamada subsiguiente a cualquier función de mysqli_stmt fallará a menos que se llame antes a mysqli_stmt_prepare().

Parámetros

link

Sólo estilo por procediminetos: Un identificador de enlace devuelto por mysqli_connect() o mysqli_init()

Valores devueltos

Returns an object.
