curl_multi_remove_handle

(PHP 5, PHP 7, PHP 8)

curl_multi_remove_handle — Elimina un multi recurso de un conjunto de recursos cURL

Descripción

curl_multi_remove_handle(resource $mh, resource $ch): int

Eliminia un recurso ch dado de un recurso mh dado. Cuando el recurso ch es eliminado, se puede volver a ejecutar curl_exec() en el mismo recurso. Eliminar el recurso cuando está en uso, cortará todas las transferencias en curso.

Parámetros

mh

El multi recurso cURL multi devuelto por curl_multi_init().

ch

El recurso cURL devuelto por curl_init().

Valores devueltos

Devuelve 0 en caso de éxito, o uno de los códigos de error CURLM_XXX.
