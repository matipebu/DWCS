pg_socket

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

pg_socket — Obtener un identificador de sólo lectura en el socket subyacente a una conexión de PostgreSQL

Descripción

pg_socket(resource $connection): resource

pg_socket() Devuelve un resource de sólo lectura que corresponde al socket subyacente a la conexión dada de PostgreSQL.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

connection

Recurso de conexión a la base de datos PostgreSQL.

Valores devueltos

Un recurso socket en caso de exito o o false en caso de error.
