odbc_close_all

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_close_all — Cerrar todas las conexiones ODBC

Descripción

odbc_close_all(): void

odbc_close_all() cerrará todas las conexión del/de los servidor/es de bases de datos.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

No devuelve ningún valor.

Notas

Nota:

Esta función fallará si hay transaccines abiertas sobre una conexión. En este caso la conexión permanecerá abierta.
