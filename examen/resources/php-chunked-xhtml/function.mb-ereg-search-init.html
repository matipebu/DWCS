mb_ereg_search_init

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

mb_ereg_search_init — Configura el string y la expresión regular para una comparación de expresiones regulares multibyte

Descripción

mb_ereg_search_init(string $string, string $pattern = ?, string $option = "msr"): bool

mb_ereg_search_init() establece string y pattern para una expresión regular multibyte. Estos valores se usan para mb_ereg_search(), mb_ereg_search_pos(), y mb_ereg_search_regs().

Parámetros

string

El estring de búsquda.

pattern

El patrón de búsqueda.

option

La opción de búsqueda.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Notas

Nota:

La codificación interna o la codificación especificada por mb_regex_encoding() será usada en esta función.
