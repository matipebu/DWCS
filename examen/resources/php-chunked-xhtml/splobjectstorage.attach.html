SplObjectStorage::attach

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

SplObjectStorage::attach — Agrega un objeto en el almacenamiento

Descripción

public SplObjectStorage::attach(object $object, mixed $info = null): void

Añade un object dentro del almacenamiento, y opcionalmente se asocian a algunos datos.

Parámetros

object

El object a ser añadido.

info

Los datos asociados con el object.

Valores devueltos

No devuelve ningún valor.

Ejemplos

Ejemplo #1 Ejemplo de SplObjectStorage::attach()

<?php
$o1 = new stdClass;
$o2 = new stdClass;
$s = new SplObjectStorage();
$s->attach($o1); // similar a $s[$o1] = NULL;
$s->attach($o2, "hola"); // similar a $s[$o2] = "hola";

var_dump($s[$o1]);
var_dump($s[$o2]);

?>

El resultado del ejemplo sería algo similar a:

NULL
string(4) "hola"
