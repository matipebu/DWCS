Generator::getReturn

(PHP 7, PHP 8)

Generator::getReturn — Obtener el valor devuelto de un generador

Descripción

public Generator::getReturn(): mixed

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el valor devuelto del generador una vez ha finalizado su ejecución.

Ejemplos

Ejemplo #1 Ejemplo de Generator::getReturn()

<?php

$gen = (function() {
yield 1;
yield 2;

return 3;
})();

foreach ($gen as $val) {
echo $val, PHP_EOL;
}

echo $gen->getReturn(), PHP_EOL;

El resultado del ejemplo sería:

1
2
3
