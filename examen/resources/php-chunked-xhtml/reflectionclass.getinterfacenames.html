ReflectionClass::getInterfaceNames

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

ReflectionClass::getInterfaceNames — Obtiene nombres de las interfaces

Descripción

public ReflectionClass::getInterfaceNames(): array

Devuelve los nombres de las interfaces.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Array numérico con el nombre de las interfaces en el valor.

Ejemplos

Ejemplo #1 Ejemplo de ReflectionClass::getInterfaceNames()

<?php
interface Foo { }

interface Bar { }

class Baz implements Foo, Bar { }

$rc1 = new ReflectionClass("Baz");

print_r($rc1->getInterfaceNames());
?>

El resultado del ejemplo sería algo similar a:

Array
(
    [0] => Foo
    [1] => Bar
)
