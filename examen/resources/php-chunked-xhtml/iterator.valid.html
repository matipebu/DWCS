Iterator::valid

(PHP 5, PHP 7, PHP 8)

Iterator::valid — Comprueba si la posición actual es válido

Descripción

public Iterator::valid(): bool

Este método se llama después de Iterator::rewind() y Iterator::next() para comprobar si la posición actual es válido.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

El valor de retorno se debe fundir a bool y luego evaluar. Devuelve true en caso de éxito o false en caso de error.
