gmp_perfect_power

(PHP 7 >= 7.3.0, PHP 8)

gmp_perfect_power — Perfect power check

Descripción

gmp_perfect_power(GMP|int|string $num): bool

Checks whether num is a perfect power.

Parámetros

num

resource de número de GMP en PHP 5.5 y anterior, o un objeto de GMP en PHP 5.6 y posterior, o un string numérico siempre que sea posible convertirlo a un número.

Valores devueltos

Returns true if num is a perfect power, false otherwise.
