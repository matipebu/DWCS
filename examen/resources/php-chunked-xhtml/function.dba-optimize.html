dba_optimize

(PHP 4, PHP 5, PHP 7, PHP 8)

dba_optimize — Optimizar base de datos

Descripción

dba_optimize(resource $handle): bool

dba_optimize() optimiza la base de datos interna.

Parámetros

handle

El gestor de base de datos, devuelto por dba_open() o dba_popen().

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
