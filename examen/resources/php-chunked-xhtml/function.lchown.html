lchown

(PHP 5 >= 5.1.3, PHP 7, PHP 8)

lchown — Cambia el propietario de un enlace simbólico

Descripción

lchown(string $filename, mixed $user): bool

Intenta cambiar el propietario del enlace simbólico filename al usuario user.

Sólo el superusuario puede cambiar el propietario de un enlace simbólico.

Parámetros

filename

Ruta al archivo.

user

Nómbre o número de usuario.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Cambiar el propietario de un enlace simbólico

<?php
$objetivo = 'salida.php';
$enlace = 'salida.html';
symlink($objetivo, $enlace);

lchown($enlace, 8);
?>

Notas

Nota: Esta función no funcionará en ficheros remotos ya que el fichero debe ser accesible vía el sistema de ficheros del servidor para poder ser examinado.

Nota: Esta función no está implementada en plataformas Windows.
