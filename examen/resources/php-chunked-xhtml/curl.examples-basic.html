Ejemplo de curl básico

Una vez haya compilado PHP con soporte para cURL, puede empezar a usar las funciones cURL. Básicamente debe inicializar una sesión cURL usando curl_init(), establecer todas las opciones para la transferencia usando curl_setopt(), ejecutar la sesión con la función curl_exec() y finalizar la sesión con curl_close()- Aquí tiene un ejemplo donde usamos funciones cURL para capturar el contenido de la página principal de example.com en un fichero:

Ejemplo #1 Uso del módulo cURL de PHP para capturar página principal de example.com

<?php

$ch = curl_init("http://www.example.com/");
$fp = fopen("example_homepage.txt", "w");

curl_setopt($ch, CURLOPT_FILE, $fp);
curl_setopt($ch, CURLOPT_HEADER, 0);

curl_exec($ch);
curl_close($ch);
fclose($fp);
?>
