iconv_get_encoding

(PHP 4 >= 4.0.5, PHP 5, PHP 7, PHP 8)

iconv_get_encoding — Recupera variables de configuración interna de la extensión iconv

Descripción

iconv_get_encoding(string $type = "all"): mixed

Recupera variables de configuración interna de la extensión iconv

Parámetros

type

El valor del parámetro opcional type puede ser:

  • all
  • input_encoding
  • output_encoding
  • internal_encoding

Valores devueltos

Retorna el valor actual de la variable de configuración interna si tiene éxito o o false en caso de error.

Si se omite type o se fija a "all", iconv_get_encoding() retorna un array que almacena todas las variables.

Ejemplos

Ejemplo #1 Ejemplo de iconv_get_encoding()

<pre>
<?php
iconv_set_encoding("internal_encoding", "UTF-8");
iconv_set_encoding("output_encoding", "ISO-8859-1");
var_dump(iconv_get_encoding('all'));
?>
</pre>

El resultado del ejemplo sería:

Array
(
    [input_encoding] => ISO-8859-1
    [output_encoding] => ISO-8859-1
    [internal_encoding] => UTF-8
)
