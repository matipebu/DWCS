ReflectionFunctionAbstract::isGenerator

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

ReflectionFunctionAbstract::isGenerator — Devuelve si la función es un generador

Descripción

public ReflectionFunctionAbstract::isGenerator(): bool

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve true si la función es un generador, false si no lo es, o null en caso de error.
