cubrid_lob2_close

(PECL CUBRID >= 8.4.1)

cubrid_lob2_close — Cierra un objeto LOB

Descripción

cubrid_lob2_close(resource $lob_identifier): bool

La función cubrid_lob2_close() se usa para cerra un objeto LOB devuelveto por cubrid_lob2_new() u obtenido de un conjunto de resultados.

Parámetros

lob_identifier

Un identificador de LOB, como un resultado de cubrid_lob2_new() u obtenido de un conjunto de resultados.

Valores devueltos

true, en caso de éxito.

false, en caso de error.
