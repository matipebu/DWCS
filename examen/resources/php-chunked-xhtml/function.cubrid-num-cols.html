cubrid_num_cols

(PECL CUBRID >= 8.3.0)

cubrid_num_cols — Devolver el número de columnas del conjunto de resultados

Descripción

cubrid_num_cols(resource $result): int

La función cubrid_num_cols() se usa para obtener el número de columnas del resultado de la consulta. Se puede usar sólo cuando la consulta ejecutada es una sentencia SELECT.

Parámetros

result

El resultado.

Valores devueltos

El número de columnas, cuando el proceso tiene éxito.

false, si la sentencia SQL no es SELECT.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_num_cols()

<?php
$conn = cubrid_connect("localhost", 33000, "demodb", "dba");

$req = cubrid_execute($conn, "SELECT * FROM code");

$row_num = cubrid_num_rows($req);
$col_num = cubrid_num_cols($req);

printf("Row Num: %d\nColumn Num: %d\n", $row_num, $col_num);

cubrid_disconnect($conn);
?>

El resultado del ejemplo sería:

Row Num: 6
Column Num: 2
