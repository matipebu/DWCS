intl_is_failure

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

intl_is_failure — Comprueba si el código de error dado indica un fallo

Descripción

intl_is_failure(int $error_code): bool

Parámetros

error_code

es un valor devuelto por las funciones: intl_get_error_code(), collator_get_error_code() .

Valores devueltos

true si el código indica algún fallo, y false en caso de éxito o de advertencia.

Ejemplos

Ejemplo #1 intl_is_failure() ejemplo

<?php
function check( $err_code )
{
var_export( intl_is_failure( $err_code ) );
echo "\n";
}

check( U_ZERO_ERROR );
check( U_USING_FALLBACK_WARNING );
check( U_ILLEGAL_ARGUMENT_ERROR );
?>

El resultado del ejemplo sería algo similar a:

false
false
true
