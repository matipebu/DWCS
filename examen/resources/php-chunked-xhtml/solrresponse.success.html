SolrResponse::success

(PECL solr >= 0.9.2)

SolrResponse::success — ¿Tuvo éxito la petición?

Descripción

public SolrResponse::success(): bool

Se usa para verificar si la petición al servidor tuvo éxito.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve true si se tuvo éxito y false si no lo tuvo.
