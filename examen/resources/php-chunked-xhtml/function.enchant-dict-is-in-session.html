enchant_dict_is_in_session

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL enchant >= 0.1.0 )

enchant_dict_is_in_session — Si existe o no una palabra en esta sesión ortográfica

Descripción

enchant_dict_is_in_session(resource $dict, string $word): bool

Indica si una palabra ya existe en la sesión actual.

Parámetros

dict

Recurso de diccionario

word

La palabra a buscar

Valores devueltos

Devuelve true si la palabra exsite o false
