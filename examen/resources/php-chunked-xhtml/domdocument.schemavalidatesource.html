DOMDocument::schemaValidateSource

(PHP 5, PHP 7, PHP 8)

DOMDocument::schemaValidateSource — Valida un documento basado en un esquema

Descripción

public DOMDocument::schemaValidateSource(string $source, int $flags = 0): bool

Valida un documento basado en un esquema definido en la cadena dada.

Parámetros

source

Una cadena que contiene el esquema.

flags

Una máscara de bits de banderas de validación de esquemas de Libxml. Actualmente el único valor admitido es LIBXML_SCHEMA_CREATE. Disponible desde Libxml 2.6.14.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
