OAuthProvider::reportProblem

(PECL OAuth >= 1.0.0)

OAuthProvider::reportProblem — Reportar un problema

Descripción

final public static OAuthProvider::reportProblem(string $oauthexception, bool $send_headers = true): string

Sucede en un problema como OAuthException, con posibles problemas listados en la sección OAuth constants.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

oauthexception

La OAuthException.

Valores devueltos

No devuelve ningún valor.
