cubrid_lob_get

(PECL CUBRID >= 8.3.1)

cubrid_lob_get — Obtener información BLOB/CLOB

Descripción

cubrid_lob_get(resource $conn_identifier, string $sql): array

cubrid_lob_get() se usa para obtener meta-información BLOB/CLOB desde la base de datos de CUBRID, CUBRID obtiene BLOB/CLOB ejecutando la sentencia SQL, y devuelve todos los LOBs como un array de recursos. Asegúrese de que SQL solo recupera una columna y su tipo de información es BLOB o CLOB.

Recuerde usar cubrid_lob_close() para liberar los LOBs si ya no los necesita más.

Parámetros

conn_identifier

Identificador de conexión.

sql

La sentencia SQL a ser ejecutada.

Valores devueltos

Devuelve un array de recursos LOB, cuando el proceso tiene éxito.

false, cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_lob_get()

<?php
$conn = cubrid_connect ("localhost", 33000, "demodb", "dba");

cubrid_execute($conn,"DROP TABLE if exists doc");
cubrid_execute($conn,"CREATE TABLE doc (id INT, doc_content CLOB)");
cubrid_execute($conn,"INSERT INTO doc VALUES (5,'hello,cubrid')");

$lobs = cubrid_lob_get($conn, "SELECT doc_content FROM doc WHERE id=5");
echo "Doc size: ".cubrid_lob_size($lobs[0])." bytes";
cubrid_lob_export($conn, $lobs[0], "doc_5.txt");
cubrid_lob_close($lobs);
cubrid_disconnect($conn);
?>
