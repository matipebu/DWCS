IntlCalendar::setLenient

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::setLenient — Establecer si la interpretación de la fecha/hora es tolerante

Descripción

Estilo orientado a objetos

public IntlCalendar::setLenient(string $isLenient): ReturnType

Estilo por procedimientos

intlcal_set_lenient(IntlCalendar $cal, string $isLenient): ReturnType

Define si el calendario está en ‘modo tolerante’. En dicho modo, se aceptan algunos de valores fuera de rango para algunos campos, siendo el comportamiento similar al de IntlCalendar::add() (es decir, el valor sobrepasa, acarreando más campos significativos cada vez). Si el modo tolerante está desactivado, tales valores generarán un error.

Parámetros

cal

El recurso IntlCalendar.

isLenient

Use true para activar el modo tolerante; false de lo contrario.

Valores devueltos

Devuelve true en caso de éxito. Los errores solamente pueden ocurrir debido a parámetros no válidos.
