ResourceBundle::create

resourcebundle_create

ResourceBundle::__construct

(PHP 5 >= 5.3.2, PHP 7, PHP 8, PECL intl >= 2.0.0)

ResourceBundle::create -- resourcebundle_create -- ResourceBundle::__construct — Crear un paquete de recursos

Descripción

Estilo orientado a objetos (método)

public static ResourceBundle::create(string $locale, string $bundlename, bool $fallback = ?): ResourceBundle

Estilo por procedimientos

resourcebundle_create(string $locale, string $bundlename, bool $fallback = ?): ResourceBundle

Estilo orientado a objetos (constructor)

public ResourceBundle::__construct(string $locale, string $bundlename, bool $fallback = ?)

Crea un paquete de recursos.

Parámetros

locale

La configuración regional para la cual los recursos deberían ser cargados (nombre de la configuración regional, p.ej. en_CA).

bundlename

El directorio donde es almacenada la información o el nombre del fichero .dat.

fallback

Si la configuración debería coincidir exactamente o si está permitido volver a la configuración regional madre.

Valores devueltos

Devuelve un objeto de la clase ResourceBundle o null en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de resourcebundle_create()

<?php
$r = resourcebundle_create( 'es', "/usr/share/data/myapp");
echo $r['teststring'];
?>

Ejemplo #2 Ejemplo de ResourceBundle::create()

<?php
$r = ResourceBundle::create( 'es', "/usr/share/data/myapp");
echo $r['teststring'];
?>

El resultado del ejemplo sería:

¡Hola, mundo!
