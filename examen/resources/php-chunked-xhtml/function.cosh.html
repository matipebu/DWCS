cosh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

cosh — Coseno hiperbólico

Descripción

cosh(float $arg): float

Devuelve el coseno hiperbólico de arg, definido como (exp(arg) + exp(-arg))/2.

Parámetros

arg

El argumento a procesar

Valores devueltos

El coseno hiperbólico de arg
