function_exists

(PHP 4, PHP 5, PHP 7, PHP 8)

function_exists — Devuelve true si la función dada ha sido definida

Descripción

function_exists(string $function_name): bool

Comprueba la lista de funciones definidas, las incluidas (internas) y las definidas por el usuario, para function_name.

Parámetros

function_name

El nombre de la función, como cadena.

Valores devueltos

Devuelve true si function_name existe y es una función, si no, false.

Nota:

Esta función devolverá false para construcciones, tales como include_once y echo.

Ejemplos

Ejemplo #1 Ejemplo de function_exists()

<?php
if (function_exists('imap_open')) {
echo "Las funciones de IMAP están disponibles.<br />\n";
} else {
echo "Las funciones de IMAP no están disponibles.<br />\n";
}
?>

Notas

Nota:

Un nombre de función puede existir incluso si la función misma no es utilizabla a causa de las opciones de configuración o de compilación (siendo un ejemplo las funciones de image).
