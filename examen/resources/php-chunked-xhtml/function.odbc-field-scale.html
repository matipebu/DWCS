odbc_field_scale

(PHP 4, PHP 5, PHP 7, PHP 8)

odbc_field_scale — Obtener la escala de un campo

Descripción

odbc_field_scale(resource $result_id, int $field_number): int

Obtiene la escala de un campo referenciado mediante número del identificador de resultado dado.

Parámetros

result_id

El identificador de resultado.

field_number

El número de campo. La numeración de campos comienza en 1.

Valores devueltos

Devuelve la escala del campo como un valor de tipo integer, o false si se produjo un error.
