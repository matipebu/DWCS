ReflectionClass::isSubclassOf

(PHP 5, PHP 7, PHP 8)

ReflectionClass::isSubclassOf — Comprueba si es una subclase

Descripción

public ReflectionClass::isSubclassOf(string $class): bool

Comprueba si la clase es una subclase de la clase especificada o implementa una iterfaz específica.

Parámetros

class

Nombre de la clase con la que hacer la comprobación.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.
