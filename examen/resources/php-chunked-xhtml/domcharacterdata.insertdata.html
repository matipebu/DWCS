DOMCharacterData::insertData

(PHP 5, PHP 7, PHP 8)

DOMCharacterData::insertData — Inserta un string en la posición de 16-bit especificada

Descripción

public DOMCharacterData::insertData(int $offset, string $data): bool

Inserta el string data en la posición offset.

Parámetros

offset

La posición en la cual se insertará.

data

El string a insertar.

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Errores/Excepciones

DOM_INDEX_SIZE_ERR

Lanzado si offset es negativo o mayor que el número de unidades de 16-bit en la data.
