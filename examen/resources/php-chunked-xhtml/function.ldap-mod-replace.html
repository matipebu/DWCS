ldap_mod_replace

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_mod_replace — Reemplazar valores de atributos con valores nuevos

Descripción

ldap_mod_replace(resource $link_identifier, string $dn, array $entry): bool

Reemplaza uno o más atributos de un dn específico. Realiza la modificación a nivel de atributo en lugar de hacerlo a nivel de Objeto. Las modificaciones a nivel de Objeto son hechas por la función ldap_modify().

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelto por ldap_connect().

dn

El nombre distinguido de una entidad de LDAP.

entry

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Notas

Nota: Esta función es segura binariamente.
