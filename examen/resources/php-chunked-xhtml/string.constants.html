nl_langinfo() Constantes de la Categoría LC_TIME
Constantes Descripción
ABDAY_(1-7) (int) Nombre abreviado del día n-ésimo de la semana.
DAY_(1-7) (int) Nombre del día n-ésimo de la semana (DAY_1 = domingo).
ABMON_(1-12) (int) Nombre abreviado del mes n-ésimo del año.
MON_(1-12) (int) Nombre del mes n-ésimo del año.
AM_STR (int) String para ante meridiano.
PM_STR (int) String para post meridiano.
D_T_FMT (int) String que puede ser usado como el string de formato de strftime() para representar la hora y fecha.
D_FMT (int) String que puede ser usado como el string de formato de strftime() para representar la fecha.
T_FMT (int) String que puede ser usado como el string de formato de strftime() para representar la hora.
T_FMT_AMPM (int) String que puede ser usado como el string de formato de strftime() para representar la hora en formato de 12 horas con el ante/post meridiano.
ERA (int) Era alterna.
ERA_YEAR (int) Año en formato de era alterna.
ERA_D_T_FMT (int) Fecha y hora en formato de era alterna (el string se puede utilizar en strftime()).
ERA_D_FMT (int) Fecha en formato de era alterna (el string se puede utilizar en strftime()).
ERA_T_FMT (int) Hora en formato de era alterna (el string se puede utilizar en strftime()).
nl_langinfo() Constantes de la Categoría LC_MONETARY
Constantes Descripción
INT_CURR_SYMBOL (int) Símbolo internacional de moneda.
CURRENCY_SYMBOL (int) Símbolo local de moneda.
CRNCYSTR (int) El mismo valor que CURRENCY_SYMBOL.
MON_DECIMAL_POINT (int) Caracter de punto decimal.
MON_THOUSANDS_SEP (int) Separador de miles (grupos de tres dígitos).
MON_GROUPING (int) Como el elemento "grouping".
POSITIVE_SIGN (int) Signo para valores positivos.
NEGATIVE_SIGN (int) Signo para valores negativos.
INT_FRAC_DIGITS (int) Dígitos fraccionarios internacionales.
FRAC_DIGITS (int) Dígitos fraccionarios locales.
P_CS_PRECEDES (int) Devuelve 1 si CURRENCY_SYMBOL precede a un valor positivo.
P_SEP_BY_SPACE (int) Devuelve 1 si un espacio separa CURRENCY_SYMBOL de un valor positivo.
N_CS_PRECEDES (int) Devuelve 1 si CURRENCY_SYMBOL precede a un valor negativo.
N_SEP_BY_SPACE (int) Devuelve 1 si un espacio separa CURRENCY_SYMBOL de un valor negativo.
P_SIGN_POSN (int)

  • Devuelve 0 si hay paréntesis alrededor de la cantidad y CURRENCY_SYMBOL.
  • Devuelve 1 si el string de signo precede la cantidad y CURRENCY_SYMBOL.
  • Devuelve 2 si el string de signo sigue a la cantidad y CURRENCY_SYMBOL.
  • Devuelve 3 si el string de signo precede inmediatamente al CURRENCY_SYMBOL.
  • Devuelve 4 si el string de signo sigue inmediatamente al CURRENCY_SYMBOL.

N_SIGN_POSN (int)
