Locale::getKeywords

locale_get_keywords

(PHP 5 >= 5.3.0, PHP 7, PHP 8, PECL intl >= 1.0.0)

Locale::getKeywords -- locale_get_keywords — Obtener las palabras clave de la configuración regional de entrada

Descripción

Estilo orientado a objetos

public static Locale::getKeywords(string $locale): array

Estilo por procedimientos

locale_get_keywords(string $locale): array

Obtiene las palabras clave de la configuración regional de entrada.

Parámetros

locale

La configuración regional de la que extraer las palabras clave.

Valores devueltos

Un array asociativo que contiene los pares palabras-clave/valor de esta configuración regional

Ejemplos

Ejemplo #1 Ejemplo de locale_get_keywords()

<?php
$array_palabras_clave = locale_get_keywords( 'de_DE@currency=EUR;collation=PHONEBOOK' );
if ($array_palabras_clave) {
foreach ($array_palabras_clave as $clave => $valor) {
echo "$clave = $valor\n";
}
}
?>

Ejemplo #2 Ejemplo orientado a objetos

<?php
$array_palabras_clave = Locale::getKeywords( 'de_DE@currency=EUR;collation=PHONEBOOK' );
if ($array_palabras_clave) {
foreach( $array_palabras_clave as $clave => $valor){
echo "$clave = $valor\n";
}
}
?>

El resultado del ejemplo sería:

collation = PHONEBOOK
currency = EUR
