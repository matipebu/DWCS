RecursiveIteratorIterator::setMaxDepth

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

RecursiveIteratorIterator::setMaxDepth — Establecer el máximo de profundidad

Descripción

public RecursiveIteratorIterator::setMaxDepth(int $max_depth = -1): void

Establece el máximo de profundidad permitida.

Advertencia

Esta función no está documentada actualmente, solamente se encuentra disponible la lista de parámetros.

Parámetros

max_depth

El máximo de profundidad permitida. Se usa -1 para cualquier profundidad.

Valores devueltos

No devuelve ningún valor.

Errores/Excepciones

Lanza una Exception si max_depth es menor que -1.
