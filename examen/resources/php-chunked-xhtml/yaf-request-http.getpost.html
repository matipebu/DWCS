Yaf_Request_Http::getPost

(Yaf >=1.0.0)

Yaf_Request_Http::getPost — Recupera una variable de POST

Descripción

public Yaf_Request_Http::getPost(string $name, string $default = ?): mixed

Recupera una variable de POST.

Parámetros

name

El nombre de la variable.

default

Si se proporciona este parámetro, será devuelto si no se pudo encontrar la variable.
