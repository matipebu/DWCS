HRTime\StopWatch::getElapsedTime

(PECL hrtime >= 0.4.3)

HRTime\StopWatch::getElapsedTime — Obtener el tiempo transcurrido para todos los intervalos

Descripción

public HRTime\StopWatch::getElapsedTime(integer $unit = ?): float

Obtener el tiempo transcurrido para todos los intervalos anteriormente cerrados.

Parámetros

unit

La unidad de tiempo representada por una constante HRTime\Unit. La predeterminada es HRTime\Unit::SECOND.

Valores devueltos

Devuelve un valor de tipo float indicando el tiempo transcurrido.
