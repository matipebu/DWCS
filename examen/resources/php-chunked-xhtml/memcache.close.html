Memcache::close

(PECL memcache >= 0.4.0)

Memcache::close — Cierra la conexión al servidor memcached

Descripción

Memcache::close(): bool

Memcache::close() cierra la conexión al servidor memcached. Esta función no cierra conexiones persistentes, que solo son cerradas durante apagado/reinicio del servidor web. También se puede usar la función memcache_close().

Valores devueltos

Devuelve true en caso de éxito o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo Memcache::close()

<?php

/* procedural API */
$memcache_obj = memcache_connect('memcache_host', 11211);
/*
Algo aquí ....
*/
memcache_close($memcache_obj);

/* OO API */
$memcache_obj = new Memcache;
$memcache_obj->connect('memcache_host', 11211);
/*
Algo aquí..
*/
$memcache_obj->close();

?>
