Introducción

Esta es una extensión para planificar eficientemente los eventos basados en las Entradas/Salidas, el tiempo y las señales; usando el mejor mecanismo de notificación Entrada/Salida disponible para una plataforma especifica. Es un puerto de “libevent” para la infraestructura de PHP.

Nota:

Note, Windows support introduced in event-1.9.0 .

Version 1.0.0 Introduce una nueva API orientada a Objetos (rompiendo la compatibilidad con versiones anteriores) y soporte de libevent 2+ incluyendo HTTP, DNS, OpenSSL y el event listener

Nota:

Nota, event-1.0.0 y superiores no son compatibles con las versiones previas
