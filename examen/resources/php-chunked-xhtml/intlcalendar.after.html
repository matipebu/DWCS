IntlCalendar::after

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlCalendar::after — Indica si el instante de este objeto es posterior al del proporcionado

Descripción

Estilo orientado a objetos

public IntlCalendar::after(IntlCalendar $other): bool

Estilo por procedimientos

intlcal_after(IntlCalendar $cal, IntlCalendar $other): bool

Devuelve si el instante de este objeto es posterior al instante del argumento.

Parámetros

cal

El recurso IntlCalendar.

other

El calendario cuyo instante será comprobado con el instante del objeto primario.

Valores devueltos

Devuelve true si el instante de este objeto es posterior al instante del argumento calendar. Devuelve false en caso contrario. También devuelve false en caso de error. Se puede utilizar exceptions o intl_get_error_code() para detectar condiciones de error.

Ejemplos

Ejemplo #1 IntlCalendar::after()

<?php
$cal1 = IntlCalendar::createInstance();
$cal2 = clone $cal1;

var_dump($cal1->after($cal2), //false
$cal2->after($cal1)); //false

$cal1->roll(IntlCalendar::FIELD_MILLISECOND, true);

var_dump($cal1->after($cal2), //true
$cal2->after($cal1)); //false
