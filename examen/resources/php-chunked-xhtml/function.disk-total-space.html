disk_total_space

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

disk_total_space — Devuelve el tamaño total de un sistema de archivos o partición de disco

Descripción

disk_total_space(string $directory): float

Dada una cadena que contiene un directorio, esta función devolverá el número total de bytes del sistema de archivos o partición de disco correspondiente.

Parámetros

directory

Un directorio del sistema de archivos o partición de disco.

Valores devueltos

Devuelve el número total de bytes como un float o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de disk_total_space()

<?php
// $ds contiene el número total de bytes disponibles en "/"
$ds = disk_total_space("/");

// En Windows:
$ds = disk_total_space("C:");
$ds = disk_total_space("D:");
?>

Notas

Nota: Esta función no funcionará en ficheros remotos ya que el fichero debe ser accesible vía el sistema de ficheros del servidor para poder ser examinado.
