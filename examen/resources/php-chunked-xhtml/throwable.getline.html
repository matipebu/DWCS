Throwable::getLine

(PHP 7, PHP 8)

Throwable::getLine — Obtiene la línea en la que el objeto fue instanciado

Descripción

public Throwable::getLine(): int

Devuelve el número de línea en la que el objeto fue instanciado.

Parámetros

Esta función no tiene parámetros.

Valores devueltos

Devuelve el número de línea en la que el objeto fue instanciado.
