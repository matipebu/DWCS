ldap_get_values_len

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_get_values_len — Obtener todos los valores binarios desde una entrada de resultados

Descripción

ldap_get_values_len(resource $link_identifier, resource $result_entry_identifier, string $attribute): array

Lee todos los valores del atributo en la entrada en el resultado.

Esta función es utilizada exactamente como ldap_get_values() excepto que esta maneja datos binarios y no datos de cadena.

Parámetros

link_identifier

Un identificador de enlace de LDAP, devuelto por ldap_connect().

result_entry_identifier

attribute

Valores devueltos

Devuelve un arreglo de valores para el atributo en caso de éxito y false en caso de error. Los valores individuales son accedidos por un índice entero en el arreglo. El índice primero es 0. El número de valores puede ser encontrado indexando "count" en el arreglo resultante.
