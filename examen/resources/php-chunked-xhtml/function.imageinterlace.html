imageinterlace

(PHP 4, PHP 5, PHP 7, PHP 8)

imageinterlace — Habilitar o deshabilitar en entrelazamiento

Descripción

imageinterlace(resource $image, int $interlace = 0): int

imageinterlace() activa o desactiva el bit de entrelazamiento.

Si el bit de entrelazamiento está establecido y la imagen es usada como una imagen JPEG, la imagen es creada como un JPEG progresivo.

Parámetros

image

Un recurso image, es devuelto por una de las funciones de creación de imágenes, como imagecreatetruecolor().

interlace

Si no es cero, la imagen será entrelazada, si no, el bit de entrelazamiento será desactivado.

Valores devueltos

Devuelve 1 si el bit de entrelazamiento está establecido para la imagen, 0 si no.

Ejemplos

Ejemplo #1 Activar el entrelazamiento usando imageinterlace()

<?php
// Crear una instancia de imagen
$im = imagecreatefromgif('php.gif');

// Habilitar el entrelazamiento
imageinterlace($im, true);

// Guardar la imagen entrelazada
imagegif($im, './php_entrelazada.gif');
imagedestroy($im);
?>
