cubrid_save_to_glo

(PECL CUBRID >= 8.3.0)

cubrid_save_to_glo — Guardar el fichero solicitado en una instancia GLO

Descripción

cubrid_save_to_glo(resource $conn_identifier, string $oid, string $file_name): int

La función cubrid_save_to_glo() se usa para guardar el fichero solicitado en una instancia glo.

Parámetros

conn_identifier

Identificador de conexión.

oid

OID de la instancia glo en donde se quiere guardar un fichero.

file_name

El nombre del archivo que quierer guardar.

Valores devueltos

true, cuando el proceso tiene éxito.

false, cuando el proceso no tiene éxito.

Ejemplos

Ejemplo #1 Ejemplo de cubrid_save_to_glo()

<?php
$req = cubrid_execute ($con, "select image from person where id=1");
if ($req) {
list ($oid) = cubrid_fetch($req);
cubrid_close_request($req);
$res = cubrid_save_to_glo ($con, $oid, "input.jpg");
if ($res) {
echo "imagen cambiada con éxito";
}
}
?>

Notas

Nota:

Por razones de compatibilidad con versiones anteriores, los siguientes alias obsoletos podrían usarse: cubrid_save_to_glo()

Nota:

Esta función está eliminada desde CUBRID 3.1.
