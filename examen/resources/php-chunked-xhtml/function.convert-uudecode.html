convert_uudecode

(PHP 5, PHP 7, PHP 8)

convert_uudecode — Descodifica una cadena codificada mediante uuencode

Descripción

convert_uudecode(string $data): string

convert_uudecode() descodifica una cadena codificada mediante uuencode.

Parámetros

data

Los datos codificados mediante uuencode.

Valores devueltos

Devuelve los datos descodificados como una cadena o false en caso de error.

Ejemplos

Ejemplo #1 Ejemplo de convert_uudecode()

<?php
/* Can you imagine what this will print? :) */
echo convert_uudecode("+22!L;W9E(%!(4\"$`\n`");
?>
